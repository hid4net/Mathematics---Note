# 1. 初等分析

## 1.1. 数
$复数 \cnums \begin{cases}
  实数 \R \begin{cases}
    有理数 ℚ \begin{cases}
      整数 \Z \begin{cases}
          自然数 \N \begin{cases}
            正整数 \begin{cases}
              质数 / 素数 \\
              合数 \end{cases}\\
            零: 0 \end{cases}\\
            负整数 \end{cases}\\
        分数 \end{cases}\\
      无理数 \end{cases}\\
  虚数
\end{cases}$

* 正数, 负数
* 质数 / 素数, 合数
* 因数, 倍数, 公约数 (最大公约数), 公倍数 (最小公倍数)
* 科学计数法, 进制
* 有效数字, 舍入
* 千分位
* ...

### 1.1.1. 复数
> 更多内容参见 [11. 复变函数](#11-复变函数)
* **虚数**: $\mathrm{i} = \sqrt {-1}$
* 用**复平面** (横坐标是实轴, 纵坐标是虚轴) 表示复数
    * **复数**: $z = x + \mathrm{i} y$, 其中 $x$ 是**实部**, $\mathrm{i} y$ 是**虚部**
    * **模**: $|z| = \sqrt {x^2+y^2}$
    * **幅角**: $\mathrm{Arg} z = \arctan \dfrac{y}{x} + 2k\pi \quad (k = 0, \pm 1, \pm 2, \cdots)$
    * **主幅角**: $\arg z = \arctan \dfrac{y}{x} \in (-\pi, \pi]$
    * 运算法则:
        * 满足交换律, 结合律, 分配律
        * $z_1 \pm z_2 = (x_1 \pm x_2) + \mathrm{i}(y_1 \pm y_2)$
        </br>
        * $z_1 \cdot z_2 = (x_1 x_2 - y_1 y_2) + \mathrm{i} (x_2 y_1 + x_1 y_2)$
        </br>
        * $\dfrac {z_1}{z_2} = (\dfrac {x_1 x_2 + y_1 y_2}{x_2^2 + y_2^2}) + \mathrm{i}(\dfrac {x_2 y_1 - x_1 y_2}{x_2^2 + y_2^2}), \quad z_2 \ne 0$
        </br>
    * **共轭**: $\overline {x + \mathrm{i} y} := x - \mathrm{i} y$
        * $\overline {z_1 \pm z_2} = \overline {z_1} + \overline {z_2}, \quad \overline {z_1 \cdot z_2} = \overline {z_1} \cdot \overline {z_2}, \quad \overline {(\dfrac {z_1} {z_2})} = \dfrac {\ \overline {z_1}\ } {\overline {z_2}}$
        * $\overline {(\overline {z})} = z$
        * $z + \overline z = 2x, \quad z - \overline z = \mathrm{i} 2y, \quad z \cdot \overline z = |z|^2 = x^2 + y^2$
        </br>
    * **欧拉 (Euler) 公式**: $\mathrm{e}^{x+\mathrm{i} y}=\mathrm{e}^x(\cos y + \mathrm{i} \cdot \sin y)$
        * $\mathrm{e}^{\mathrm{i} \mathrm{\pi}} + 1 = 0$
* 用极坐标表示复数
    * **复数**: $z = r \mathrm{e}^{\mathrm{i} \theta}$,
        * 对应复平面下的复数 $z = r(\cos \theta + \mathrm{i} \sin \theta)$, 其中 $\theta = \arctan \dfrac{y}{x}$
    * 运算法则:
        * $\large r \mathrm{e}^{\mathrm{i} \theta} \cdot \rho \mathrm{e}^{\mathrm{i} \varphi} = r \rho \ \mathrm{e}^{\mathrm{i}(\theta + \varphi)}$
        </br>
        * $\large (r \mathrm{e}^{\mathrm{i} \theta})^n = r^n \mathrm{e}^{\mathrm{i} n \theta}$
        </br>
        * $\large \dfrac {r \mathrm{e}^{\mathrm{i} \theta}} {\rho \mathrm{e}^{\mathrm{i} \varphi}} = \dfrac {r}{\rho} \ \mathrm{e}^{\mathrm{i}(\theta - \varphi)}$
        </br>
        * $\large \sqrt[n]{r \mathrm{e}^{\mathrm{i} \theta}} = \Large {r^{\frac 1 n} \mathrm{e}^{\mathrm{i} \frac {\theta + 2n\pi}{n}}}$

## 1.2. 运算

### 1.2.1. 基本运算
* 加 `+` , 减 `-` , 乘 `*` , 除 `/` , 相反数
    * 对于任意复数 $a,b$, $\\ \begin{aligned}
            (-a)(-b) &= ab, \\
            (-a)b &= -ab, \\
            a(-b) &=-ab \\
            -(-a) &= a, \\
            (-1)a &= -a
        \end{aligned}$
    * 分数:
        * $\dfrac{a}{b} \pm \dfrac{c}{d} = \dfrac{ad \pm bc}{bd}$
            </br>
        * $\dfrac{a}{b} \cdot \dfrac{c}{d} = \dfrac{ac}{bd}$
            </br>
        * $\dfrac {\ \dfrac{a}{b} \ } {\dfrac{c}{d}} = \dfrac{a}{b} \cdot \dfrac{d}{c} = \dfrac{ad}{bc}$
* 交换律, 结合律, 分配律
    * 交换律: $a + b = b + a; \ ab = ba$
    * 结合律: $(a + b) + c = a + (b + c); \ (ab)c = a(bc)$
    * 分配律: $a(b + c) = ab + ac$
* 正比, 反比
    * 若 $y$ 与 $x$ 成正比, 则 $y=kx$ ($k$ 为比例系数)
    * 若 $y$ 与 $x$ 成反比, 则 $y= \dfrac k x$ ($k$ 为比例系数)
* 绝对值($|x|$), 近似 (四舍五入, 取整)

### 1.2.2. 乘方, 开方, 指数
概念: 次, 根, 幂
* $a^m \cdot a^n=a^{m+n}, \quad \dfrac {a^m} {a^n} = a^{m-n}$
    </br>
* $(a^m)^n = a^{mn},\quad (ab)^m={a^m}{b^m},\quad (\dfrac a b)^m=\dfrac {a^m}{b^m}$
    </br>
* $a^{-m} = \dfrac {1}{a^m}$
    </br>
* $a^{m/n} = \sqrt [n] {a^m}$

### 1.2.3. 对数
对数 $\log_a N (a > 0,a \ne 1, N > 0)$, 其中: $a$为底
* 对数恒等式: $N = a^{\log_a N} = \mathrm{e}^{\ln N}$
    </br>
* $\log_a({MN}) = \log_a M + \log_a N,\quad \log_a({\dfrac M N}) = \log_a M - \log_a N$
    </br>
* $\log_a({M^n}) = n\log_a M,\quad \log_a \sqrt[n]M = \dfrac{1}{n} \log_a M$
    </br>
* 换底公式: $\log_a M = \dfrac {\log_b M}{\log_b a}$
    </br>
* $\log_a 1 = 0,\quad \log_a a = 1$

### 1.2.4. 等式运算
* $a = b \Rarr a \pm c = b \pm c$
* $a = b \Rarr ac = bc$
* $a = b, c \ne 0 \Rarr \dfrac{a}{c} = \dfrac{b}{c}$
    </br>
* $a = b \ne 0 \Rarr \dfrac{1}{a} = \dfrac{1}{b}$
    </br>
* $a = b, c = d \Rarr a \pm c = b \pm d$
* $a = b, c = d \Rarr a c = b d$
* $a = b, c = d \ne 0 \Rarr \dfrac{a}{c} = \dfrac{b}{d}$
    </br>
* 合比定理: $\dfrac {a}{b} = \dfrac {c}{d} \Rarr \dfrac {a+b}{b} = \dfrac {c+d}{d}$
* 分比定理: $\dfrac {a}{b} = \dfrac {c}{d} \Rarr \dfrac {a-b}{b} = \dfrac {c-d}{d}$
* 合分比定理: $\dfrac {a}{b} = \dfrac {c}{d} \Rarr \dfrac {a+b}{a-b} = \dfrac {c+d}{c-d}$
* $\dfrac {a}{b} = \dfrac {c}{d} = \dfrac {e}{f} \Rarr \dfrac {a}{b} = \dfrac {c}{d} = \dfrac {e}{f} = \dfrac {a+c+e}{b+d+f}$

### 1.2.5. 不等式运算
* $a \leqslant b \Rarr a \pm c \leqslant b \pm c$
* $a \leqslant b, c \geqslant 0 \Rarr a c \leqslant b c$
* $a \leqslant b, c \leqslant d \Rarr a \pm c \leqslant b \pm d$
* $a \leqslant b, 0 \leqslant c \leqslant d \Rarr a c \leqslant b d$
    </br>
* $a \leqslant b, c < 0 \Rarr a c \geqslant b c$
    </br>
* $a \leqslant b, c > 0 \Rarr \dfrac a c \leqslant \dfrac b c$
    </br>
* $a \leqslant b, c < 0 \Rarr \dfrac a c \geqslant \dfrac b c$
    </br>
* $0 < a \leqslant b \Rarr \dfrac 1 a \geqslant \dfrac 1 b$
    </br>
* $a > b > 0, n > 0 \Rarr a^n > b^n$
* $a > b > 0, n 为正整数 \Rarr \sqrt[n]a > \sqrt[n]b$
    </br>
* $\dfrac a b < \dfrac c d \Rarr \dfrac a b < \dfrac {a + c}{b + d} < \dfrac c d$
* 非负数的算术平均值不小于其几何平均值, 即
    * $\dfrac {a + b}2 > \sqrt {ab}$
        </br>
    * $\dfrac {a + b + c}3 > \sqrt[3] {abc}$
        </br>
    * $\dfrac {a_1 + a_2 + a_3 + \cdots + a_n}n > \sqrt[n] {a_1 a_2 a_3 \cdots a_n}$
* 绝对值不等式
    * $|a \pm b| \leqslant |a| + |b|$
    * $|a - b| \geqslant |a| - |b|$
    * $-|a| \leqslant a \leqslant |a|$

## 1.3. 集合
* **集合 (简称集)**, 表示方法
    * 列举所有元素
        > 如 $A = \{a_1, a_2, \cdots, a_n\}$
    * $M = \{x | x 具有性质\}$
        > 如 $B=\{x|x^2-1=0\} \ \ (等效于 \{0, 1\})$
* **元素 (简称元)**, 与集合的关系:
    * **属于** ($a \in A$)
    * **不属于** ($a \notin A$)
* 基本概念:
    * 有限集, 无限集
    * **子集**, 与集合的关系:
        * **包含于** ($A \subset B$)
        * **包含** ($A \supset B$)
    * **真子集**, 与集合的关系: 包含于且不相等 ($A \subsetneqq B$)
        * $\N \subsetneqq \Z \subsetneqq ℚ \subsetneqq \R$
    * **空集** ($\varnothing$)
    * **全集** ($I$)
* 集合运算:
    * **并集**: 记作 $A \cup B$, 即 $A \cup B = \{x|x \in A, 或 x \in B\}$
    * **交集**: 记作 $A \cap B$, 即 $A \cap B = \{x|x \in A, 且 x \in B\}$
    * **差集**: 记作 $A \setminus B$, 即 $A \setminus B = \{x|x \in A, 且 x \notin B\}$
    * **补集（余集）**: 记作$A^C$, 即 $A^C = I \setminus A$
    * **笛卡尔 (Descartes) 乘积** 或 **直积**: 记作: $A \times B$, 即 $A \times B = \{(x,y) | x \in A, 且 y \in B \}$
        * $\R \times \R =  \{(x,y) | x \in \R, y \in \R \}$ 即为 $xOy$ 面上全体点的集合, $\R \times \R$ 常记作 $\R^2$
    * 满足交换律、结合律、分配律、对偶律
* **区间**:
    * **开区间**, 记作 $(a, b)$, 即 $(a, b) = \{x| a < x < b\}$, $a, b$ 称为**端点**
    * **闭区间**, 记作 $[a, b]$, 即 $[a, b] = \{x| a \leqslant x \leqslant b\}$
    * **左开右闭区间**, 记作 $[a, b)$, 即 $[a, b) = \{x|a \leqslant x < b\}$
    * **左闭右开区间**, 记作 $(a, b]$, 即 $(a, b] = \{x|a < x \leqslant b\}$
    * **有限区间**
    * **无限区间**
        > 如 $[a, +\infin)$, $(-\infin, a]$
* **邻域**
    * 以 $a$ 为中心的开区间为**邻域**, 记作 $U(a)$
    * $(a-\delta, a+\delta)$ 记作 $U(a, \delta)$
        * $a$ 是 **邻域中心**, $\delta$ 是 **邻域半径**
        * 左 $\delta$ 邻域, 右 $\delta$ 邻域
    * **去心邻域**: $(a-\delta, a+\delta)$ 中不包含 $a$, 记作 $\mathring U (a, \delta)$

## 1.4. 数列 $\{x_n\}$
* **通项/一般项**: $x_n$
* 一些特殊的数列
    * 等差数列: $x_n = x_{n-1} + d = x_1 + (n-1)d$, 其中: $x_1$ 为首项, $d$ 为公差
        * $\sum x_n = \dfrac {x_1+x_n}{2} n = na + \dfrac {n(n-1)}{2} d$
        * 设 $a, b, c$ 成等差数列, 则等差数列中项 $b=\dfrac 1 2 (a+c)$
    * 等比数列: $x_n = x_{n-1} \cdot q = x_1q^{n-1}$, 其中: $x_1$ 为首项, $q$ 为公比
        * $\sum x_n = \dfrac {x_1(1-q^n)}{1-q} = \dfrac {x_1 - x_n q}{1-q}$
* 常用的几种数列的和
    * 若 $x_n = n$, 则 $\sum x_n = 1+2+3+ \cdots= \dfrac 1 2 n(n+1)$
    * 若 $x_n = n^2$, 则 $\sum x_n = 1^2 + 2^2 + 3^2 + \cdots = \dfrac 1 6 n(n+1)(2n+1)$
    * 若 $x_n = n^3$, 则 $\sum x_n = 1^3 + 2^3 + 3^3 + \cdots = \Big[\dfrac 1 2 n(n+1) \Big]^2$
    * 若 $x_n = n*(n+1)$, 则 $\sum x_n = 1 \cdot 2 + 2 \cdot 3 + 3 \cdot 4 + \cdots= \dfrac 1 3 n(n+1)(n+2)$
    * 若 $x_n = n(n+1)(n+2) $, 则 $\sum x_n = 1 \cdot 2 \cdot 3 + 2 \cdot 3 \cdot 4 + \cdots= \dfrac 1 4 n(n+1)(n+2)(n+3)$

## 1.5. 函数

### 1.5.1. 基本概念

# 2. 极限

## 2.1. 数列的极限
* **定义**: 设 $\{x_n\}$ 为一个数列, 如果存在常数 $a$, 对于任意给定的正数 $\varepsilon$ (无论多小), 总存在正整数 $N$, 使得当 $n > N$ 时, 不等式 $|x_n-a| < \varepsilon$ 都成立, 那么就称 $a$ 是数列 $\{x_n\}$ 的极限, 或者数列 $\{x_n\}$ 收敛于 $a$, 记作 $$\lim _{x\to \infin} x_n = a \  或 \ x_n \to a \ (n \to \infin)$$
    * 也可表述为: $\displaystyle \lim _{n \to \infin} x_n = a \Harr \forall \varepsilon > 0, \exists 正整数, 当 n > N 时, 有 |x_n - a| < \varepsilon$
* **定理**:
    1. **极限的唯一性**: 如果数列 $\{x_n\}$ 收敛, 那么它的极限唯一
    1. **收敛数列的有界性**: 如果数列 $\{x_n\}$ 收敛, 那么它一定有界
    1. **收敛数列的保号性**: 如果 $\displaystyle \lim _{x\to \infin} x_n = a$, 且 $a>0 (或 a < 0)$, 那么存在正整数 $N>0$, 当 $n>N$ 时, 都有 $x_n >0 (或 x_n < 0)$
        * 如果数列 $\{x_n\}$ 从某项起有 $x_n \geqslant 0$ (或 $x_n \leqslant 0$), 且 $\displaystyle \lim _{x\to \infin} x_n = a$, 那么 $a \geqslant 0$ (或 $a \leqslant 0$)
    1. **收敛数列与其子数列的关系**: 如果数列 $\{x_n\}$ 收敛 $a$, 那么它的任一子数列也收敛, 且极限也是 $a$

## 2.2. 函数的极限
* **定义**:
    * 设函数 $f(x)$ 在点 $x_0$ 的某一去心邻域内有定义, 如果存在常数 $A$, 对于任意给定的正数 $\varepsilon$（不论多小）, 总存在正数 $\delta$, 使得当 $x$ 满足不等式 $0<|x-x_n|< \delta$ 时, 对应的函数值 $f(x)$ 都满足不等式 $|f(x)-A| < \varepsilon$, 那么常数 $A$ 就叫做函数 $f(x)$ 当 $x\to x_n$ 时的极限, 记作 $$\lim _{x \to x_0} f(x) = A \ 或 \ f(x) \to A \ (当 x \to x_0)$$
        * 也可表述为 $\displaystyle \lim _{x \to x_0}{f(x)=A} \Harr \forall \varepsilon > 0, \exists \delta >0, 当 0<|x-x_0| <\delta 时, 有 |f(x)-A|< \varepsilon$
    * 设函数 $f(x)$ 当 $|x|$ 大于某一正数时有定义. 如果存在常数 $A$, 对于任意给定的正数$\varepsilon$（不论多小）, 总存在着正数 $X$, 使得当 $x$ 满足不等式 $|x|>X$ 时, 对应的函数值 $f(x)$ 都满足不等式  $|f(x)-A| < \varepsilon$, 那么常数 $A$ 就叫做函数 $f(x)$ 当$x\to \infin$ 时的极限, 记作 $$\lim _{x \to \infin} f(x) = A \ 或 \ f(x) \to A \ (当 x \to \infin)$$
        * 也可表述为 $\displaystyle \lim _{x \to \infin}{f(x)=A} \Harr \forall \varepsilon > 0, \exists x > 0, 当|x| > X 时, 有 |f(x)-A|< \varepsilon$
* **单侧极限**
    * 左极限: $\displaystyle \lim _{x \to x_0^-} f(x)$
    * 右极限: $\displaystyle \lim _{x \to x_0^+} f(x)$
* **定理**:
    1. **极限的唯一性**: 如果 $\displaystyle \lim _{x \to x_0} f(x)$ 存在, 那么它的极限唯一
    1. **函数极限的局部有界性**: 如果 $\displaystyle \lim _{x \to x_0} f(x)=A$, 那么存在常数 $M>0$ 和 $\delta>0$, 使得当 $0<|x-x_0| < \delta$ 时, 有 $|f(x)|\leqslant M$
    1. **函数极限的局部保号性**: 如果 $\displaystyle \lim _{x \to x_0} f(x)=A$, 且 $A>0 (或 A < 0)$, 那么存在常数 $\delta>0$, 使得当 $0<|x-x_0| < \delta$ 时, 有 $f(x) > 0 (或 f(x) < 0)$
        * 如果 $\displaystyle \lim _{x \to x_0} f(x)=A (A \ne 0)$, 那么就存在着 $x_0$ 的某一去心邻域$\mathring U(x_0)$, 当 $x \in \mathring U(x_0)$ 时, 就有 $|f(x)| > \dfrac{|A|}{2}$
        * 如果在 $x_0$ 的某去心邻域内 $f(x) \geqslant 0$ (或 $f(x) \leqslant 0$), 而且 $\displaystyle \lim _{x \to x_0} f(x)=A$, 那么 $A \geqslant 0$ (或 $A \leqslant 0$)
    1. **函数极限与数列极限的关系**: 如果极限 $\displaystyle \lim _{x \to x_0} f(x)$ 存在, $\{x_n\}$ 为函数 $f(x)$ 的定义域内任一收敛于 $x_0$ 的数列, 且满足: $x_n \ne x_0 (n \in \N^+)$, 那么相应的函数值数列 $\{f(x_n)\}$ 必收敛, 且 $\displaystyle \lim _{x \to \infin} f(x_n) = \lim _{x \to x_0} f(x)$

## 2.3. 无穷大与无穷小
* **定义**: 设函数 $f(x)$ 在 $x_0$ 的某一去心邻域内有定义 (或 $|x|$ 大于某一正整数时有定义),
    * 如果对于任意给定的整数 $M$ (无论多大), 总存在整数 $\delta$ (或正数 $X$), 只要 $x$ 适合不等式 $0 <|x - x_0| < \delta (或 |x| > X)$ 对应的函数值总满足不等式 $|f(x)| > M$, 则称函数 $f(x)$ 为当 $x \to x_0 (或 x \to \infin)$ 时的**无穷大**
    * 如果函数 $f(x)$ 当 $x \to x_0 /\infin$ 时的极限为零, 那么称函数 $f(x)$ 为当 $x \to x_0/\infin$ 时的**无穷小**
    * 如果 $\lim \dfrac {\beta}{\alpha} = 0$, 则 $\beta$ 是比 $\alpha$ **高阶的无穷小**, 记作 $\beta = o(a)$
        </br>
    * 如果 $\lim \dfrac {\beta}{\alpha} = \infin$, 则 $\beta$ 是比 $\alpha$ **低阶的无穷小**
        </br>
    * 如果 $\lim \dfrac {\beta}{\alpha} = c \ne 0$, 则 $\beta$ 是 $\alpha$ 的**同阶的无穷小**
        </br>
    * 如果 $\lim \dfrac {\beta}{\alpha^k} = c \ne 0$, 则 $\beta$ 是关于 $\alpha$ 的 **$k$ 阶的无穷小**
        </br>
    * 如果 $\lim \dfrac {\beta}{\alpha} = 1$, 则 $\beta$ 是 $\alpha$ **等阶的无穷小**, 记作 $\alpha ∼ \beta$
* **定理**:
    1. 在自变量的同一变化过程 $x \to x_0 /\infin$ 中, 函数 $f(x)$ 具有极限 $A$ $\iff$ $f(x)=A+a$, 其中 $a$ 是无穷小
    1. 在自变量的同一变化过程, 如果 $f(x)$ 为无穷大, 则 $\dfrac{1}{f(x)}$ 为无穷小; 反之, 如果 $f(x)$ 为无穷小, 则 $\dfrac{1}{f(x)}$ 为无穷大
    1. $\beta$ 与 $\alpha$ 等阶无穷小的 $\iff$ $\beta = \alpha + o(a)$
    1. 设 $\alpha ∼ \alpha', \beta ∼ \beta'$ 且 $\lim \dfrac {\beta'}{\alpha'}$ 存在, 则 $\lim \dfrac {\beta}{\alpha} = \lim \dfrac {\beta'}{\alpha'}$

## 2.4. 两个重要的极限
* $\displaystyle \lim _{x \to 0}{\dfrac {\sin x}{x}} = 1$
* $\displaystyle \lim _{x \to \infin}{(1 + \dfrac {1}{x})^x} = \mathrm{e}$

## 2.5. 极限运算法则
1. 有限个无穷小的和也是无穷小
1. 有界函数与无穷小的乘积是无穷小
    * 常数与无穷小的乘积是无穷小
    * 有限个无穷小的乘积是无穷小
1. $\lim f(x) = A, \lim g(x) = B$, 那么
    * $\lim [f(x) \pm g(x)] = \lim f(x) \pm \lim g(x) = A \pm B$
    * $\lim [f(x) \cdot g(x)] = \lim f(x) \cdot \lim g(x) = A \cdot B$
        </br>
    * 若 $B \ne 0$ 则 $\lim \dfrac{f(x)}{g(x)} = \dfrac {\lim f(x)}{\lim g(x)} = \dfrac A B$
        </br>
    * 如果 $c$ 为常数, 则 $\lim cf(x) = c \lim f(x)$
    * 如果 $n$ 为正整数, 则 $\lim [f(x)]^n = [\lim f(x)]^n$
1. 设有数列 $\{x_n\} 和 \{y_n\}$, 如果 $\lim x_n = A, \lim y_n = B$, 那么
    * $\displaystyle \lim _{n \to \infin} x_n \pm y_n = A \pm B$
    * $\displaystyle \lim _{n \to \infin} x_n \cdot y_n = A \cdot B$
    * $当 y_n \ne 0 (n = 1, 2, \cdots) 且 B \ne 0 时, \displaystyle \lim _{n \to \infin} \dfrac {x_n}{y_n} = \dfrac A B$
        </br>
1. 如果 $\varphi(x) \geqslant \psi (x)$, 而 $\lim \varphi(x) = a, \lim \psi(x) = b$, 那么 $a \geqslant b$
1. **复合函数的极限**: 设函数 $y=f[g(x)]$ 是由函数 $u=g(x)$ 与函数 $y=f(u)$ 复合而成, $y=f[g(x)]$ 在点 $x_0$ 的某去心邻域内有定义, 若 $\lim g(x)= u_0, \lim f(u) = A$, 且存在 $\delta _0 > 0$, 当 $x \in \mathring U(x_0, \delta _0)$ 时有 $g(x) \ne u_0$ 则 $\displaystyle \lim _{x \to x_0}f[g(x)] = \lim _{x \to x_0}f(u) = A$

## 2.6. 极限存在准则
* **准则 I: 夹逼准则**
    * 如果数列 $\{x_n\}, \{y_n\}, \{z_n\}$ 满足下列条件:
        1. 从某项起, 即 $\exists n_0 \in \N$, 当 $n > n_0$ 时, 有 $y_n \leqslant x_n \leqslant z_n$
        1. $\displaystyle \lim _{x \to \infin}y_n = a, \lim _{x \to \infin}z_n = a$

      那么数列 $\{x_n\}$ 的极限存在, 且 $\displaystyle \lim _{x \to \infin}x_n = a$

    * 如果
        1. $当 x \in \mathring U(x_0, r) (或 |x| > M) 时, g(x) \leqslant f(x) \leqslant h(x)$
        1. $\displaystyle \lim _{x \to x_0/\infin}{g(x)} =A, \lim _{x \to x_0/\infin}{h(x)} =A$

      那么 $\displaystyle \lim _{x \to x_0/\infin}{f(x)}$ 存在, 且等于 $A$
* **准则 II**
    * 单调有界数列必有极限
    * 设函数 $f(x)$ 在点 $x_0$ 的某个左邻域内单调并且有界, 则 $f(x)$ 在 $x_0$ 的左极限 $f(x_0^-)$ 必定存在

## 2.7. 函数的连续性
* **定义**:
    * **连续**: 设函数 $y=f(x)$在点 $x_0$ 的某一邻域内有定义, 如果 $$\displaystyle \lim _{\Delta x \to 0} \Delta y = \lim _{\Delta x \to 0} [f(x_0 + \Delta x) - f(x_0)] = 0$$ 那么就称函数 $y=f(x)$ 在点 $x_0$ **连续**
        * 也可表述为: 设函数 $y=f(x)$ 在点 $x_0$ 的某一邻域内有定义, 如果 $\displaystyle \lim _{x \to x_0} f(x) = f(x_0)$, 那么就称函数 $f(x)$ 在点 $x_0$ 连续
    * **左连续**: $f(x_0^-) = f(x_0)$
    * **右连续**: $f(x_0^+) = f(x_0)$
    * **不连续点** (**间断点**): 设函数 $f(x)$ 在点 $x_0$ 的某去心邻域内有定义, 在此前提下, 如果函数 $f(x)$ 有下列三种情形之一:
        1. 在 $x= x_0$ 没有定义
        1. 虽在 $x=x_0$ 有定义, 但 $\displaystyle \lim _{x \to x_0} f(x)$ 不存在
        1. 虽在 $x=x_0$ 有定义, 且 $\displaystyle \lim _{x \to x_0} f(x)$ 存在, 但 $\displaystyle \lim _{x \to x_0} f(x) \ne f(x_0)$

        则函数 $f(x)$ 在点 $x_0$ 为不连续, 而点 $x_0$ 称为函数 $f(x)$ 的不连续点或间断点
    * **间断点类型**: 无穷间断点, 振荡间断点, 可去间断点, 跳跃间断点
        * 第一类间断点: 可去间断点, 跳跃间断点
        * 第二类间断点: 无穷间断点, 振荡间断点
* **基本初等函数在它们的定义域内都是连续的**
* **连续函数的定理**
    1. 设函数 $f(x)$ 和 $g(x)$ 在点 $x_0$ 连续, 则它们的和（差）$f \pm g$、积 $f \cdot g$ 及商 $\dfrac f g$（当 $g(x_0) \ne 0$ 时）都在点 $x_0$ 连续.
    1. 如果函数 $y=f(x)$ 在区间 $I_x$ 上单调增加（或单调减少）且连续, 那么它的反函数 $x=f^{-1}(y)$ 也在对应的区间 $I_y=\{y|y=f(x), x \in I_x\}$ 上单调增加（或单调减少）且连续
    1. 设函数 $y=f[g(x)]$ 由函数 $u=g(x)$ 与函数 $y=f(u)$ 复合而成, $\mathring U(x_0) \subset D_{f \cdot g}$, 若 $\displaystyle \lim _{x \to x_0} g(x) = u_0$, 而函数 $y=f(u)$ 在 $u=u_0$ 连续, 则 $\displaystyle \lim _{x \to x_0} f[g(x)] = \lim _{x \to x_0} f(u) = f(u_0)$
    1. 设函数 $y=f[g(x)]$ 由函数 $u=g(x)$ 与函数 $y=f(u)$ 复合而成, $\mathring U(x_0) \subset D_{f \cdot g}$, 若函数 $u=g(x)$ 在 $x=x_0$ 连续, 且 $g(u_0) = u_0$, 而函数 $y=f(u)$ 在 $u=u_0$连续, 则复合函数 $y=f[g(x)]$ 在 $x=x_0$ 也连续
* **闭区间上的连续性的定理**:
    1. **有界性与最大值最小值定理**: 在闭区间上连续的函数在该区间上有界且一定能取得它的最大值和最小值
    1. **零点定理**: 设函数 $f(x)$ 在闭区间 $[a, b]$上连续, 且 $f(a)$ 与 $f(b)$ 异号 ( 即 $f(a)\cdot f(b)<0$), 那么在开区间 $(a, b)$ 内至少有一点 $s$, 使 $f(s)=0$
    1. **介值定理**: 设函数 $f(x)$ 在闭区间 $[a, b]$ 上连续, 且在这区间的端点取不同的函数值 $f(a)=A及f(b)=B$, 那么, 对于A与B之间的任意一个数 $C$, 在开区间 $(a, b)$ 内至少有一点 $\varepsilon$, 使得 $f(\varepsilon) = C (a < \varepsilon < b)$
        * **推论**: 在闭区间上连续的函数必取得介于最大值 $M$ 与最小值 $m$ 之间的任何值
    1. **一致连续性定理**: 如果函数 $f(x)$ 在闭区间 $[a, b]$ 上连续, 那么它在该区间上一致连续

# 3. 导数与微分

## 3.1. 导数与微分的基本概念
* **导数**: 设函数 $y=f(x)$ 在点 $x_0$ 的某个邻城内有定义, 当自变量 $x$ 在 $x_0$ 处取得增量 $\Delta x$ (点 $x_0 + \Delta x$ 仍在该邻域内) 时, 相应的函数取得增量 $\Delta y = f(x_0 +  \Delta x) - f(x_0)$; 如果 $\Delta y$ 与 $\Delta x$ 之比当 $\Delta x \to 0$ 时的极限存在, 则称函数$y=f(x)$ 在点 $x_0$ 处可导, 并称这个极限为函数 $y=f(x)$ 在点 $x_0$ 处的导数, 记为 $f'(x_0)$, 即 $$f'(x_0) = \lim _{\Delta x \to 0} \dfrac{\Delta y}{\Delta x} = \lim _{\Delta x \to 0} \dfrac {f(x_0 + \Delta x) - f(x)}{\Delta x}$$ 也可记作 $y'\Big|_{x=x_0}, \dfrac{\mathrm{d} y}{\mathrm{d} x}\Big|_{x=x_0}, 或 \dfrac{\mathrm{d} f(x)}{\mathrm{d} x}\Big|_{x=x_0}$
    > $\dfrac{\mathrm{d}}{\mathrm{d} x}$ 是莱布尼兹(Leibniz)符号
    * **单侧导数**: $f_+'(x), f_-'(x)$
    * **高阶导数**:
        * 二阶导数: 记作 $y''$ 或 $\dfrac {\mathrm{d}^2 y}{\mathrm{d} x^2}$, 即 $y''=(y')'$ 或 $\dfrac {\mathrm{d}^2 y}{\mathrm{d} x^2} = \dfrac{\mathrm{d}}{\mathrm{d}x}(\dfrac{\mathrm{d} y}{\mathrm{d}x})$
            </br>
        * 三阶导数, 四阶导数, ..., n阶导数, 分别记作 $y''', \dots, y^{(4)}, y^{(n)}$
    * 通常称导数等于零的点为函数的**驻点** (或**稳定点**, **临界点**)
    * 导数的意义
        * 曲线的斜率
        * 里程与速度, 速度与加速度
        * ...
* **微分**:设函数 $y=f(x)$ 在某区间内有定义, $x_0$ 及 $x_0 + \Delta x$ 在这区间内, 如果增量 $\Delta y = f(x_0 + \Delta x) - f(x_0)$ 可表示为 $\Delta y = A \Delta x - o(\Delta x)$, 其中 $A$ 是不依赖于 $\Delta x$ 的常数, 那么称函数 $y=f(x)$ 在点 $x_0$ 是可微的, 而 $A \Delta x$ 叫做函数 $y=f(x)$ 在点 $x_0$ 相应于自变量增量 $\Delta x$ 的微分, 记作 $\mathrm{d} y$, 即 $$\mathrm{d} y = A \Delta x$$
    * 微分的意义: 工程上的近似
        * $\sqrt[n]{1+x} \approx 1 + \dfrac 1 n x$
        * $\sin x \approx x$
        * $\tan \approx x$
        * $e^x \approx 1 + x$
        * $ln(1+x) \approx x$

## 3.2. 求导法则
1. 设 $u,v$ 为函数, $C \in \R$,
    * $(u \pm v)' = u' \pm v'$
    * $(Cu)' = Cu'$
    * $(uv)' = u' \cdot v + u \cdot v'$
    * $(uv)^{(n)} = \displaystyle \sum^{n}_{k=0}  C_n^k u^{(n-k)}v^{(k)}$
    * $(\dfrac u v)' = \dfrac {u' \cdot v - u \cdot v'}{v^2}$
1. 反函数求导
    * $[f^{-1}(x)]' = \dfrac {1}{f'(y)}$ 或 $\dfrac {\mathrm{d} y}{\mathrm{d} x} = \dfrac {1}{\dfrac {\mathrm{d} x}{\mathrm{d} y}}$
1. 复合函数求导（链式法则）
    * $f[g(x)]' = f'(u) \cdot g'(x)$ 或 $\dfrac {\mathrm{d} y}{\mathrm{d} x} = \dfrac {\mathrm{d} y}{\mathrm{d} u} \cdot \dfrac {\mathrm{d}u}{\mathrm{d} x}$
        * $\dfrac {\mathrm{d} y}{\mathrm{d} x} = \dfrac {\mathrm{d} y}{\mathrm{d} u} \cdot \dfrac {\mathrm{d}u}{\mathrm{d} v} \cdot \dfrac {\mathrm{d}v}{\mathrm{d} x}$

## 3.3. 基本导数
* $(C)' = 0$
* $(x^\mu) = \mu x^{\mu - 1}$
* $(a^x)' = a^x \ln a$, 特别地 $(e^x)' = e^x$
* $(\log_a x)' = \dfrac{1}{x \ln a}$ , 特别地 $(\ln x)' = \dfrac{1}{x}$
    </br>

* $(\csc x)' = - \csc x \cdot \cot x$
* $(\arcsin x)' = \dfrac {1}{\sqrt {1-x^2}}$
    </br>
* $(\arccos x)' = - \dfrac {1}{\sqrt {1-x^2}}$
    </br>
* $(\arctan x)' = \dfrac {1}{1+x^2}$
    </br>
* $(\mathrm {arccot} x)' = - \dfrac {1}{1+x^2}$
    </br>
* $(\sinh x)' = \cosh x$
* $(\cosh x)' = \sinh x$

    </br>

    </br>
* $(\mathrm{arch} x)' = \dfrac {1}{\sqrt {x^2-1}}$
    </br>

## 3.4. 隐函数、参数方程、复函数的导数
* **隐函数求导**: 等式两边分别求导, 算子 $\dfrac {\mathrm{d}}{\mathrm{d} x}$
    > 如: $e^y+xy-e=0$ 的导数:
    > $\displaystyle \frac {\mathrm{d}}{\mathrm{d} x}(e^y+xy-e=0) = e^y \frac {\mathrm{d} y}{\mathrm{d} x} + y + x \frac {\mathrm{d} y}{\mathrm{d} x} = 0$
    > $\displaystyle \frac {\mathrm{d} y}{\mathrm{d} x} = - \frac{y}{x+e^y} \ (x+e^y \ne 0)$
* **参数方程求导**: $\dfrac {\mathrm{d} y}{\mathrm{d} x} = \dfrac {\ \dfrac {\mathrm{d} y}{\mathrm{d} t} \ }{\dfrac {\mathrm{d} x}{\mathrm{d} t}}$
* **复变函数的导数**: $f(x) = \alpha (x) + \mathrm{i} \beta (x)$ 的导数为 $f'(x) = \alpha' (x) + \mathrm{i} \beta' (x)$

## 3.5. 微分中值定理
* **费马引理**: 设函数 $f(x)$ 在点 $x_0$ 的某邻城 $U(x_0)$ 内有定义, 并且在 $x_0$ 处可导, 如果对任意的 $x \in U(x_0)$, 有 $f(x) \leqslant f(x_0) (或 f(x) \geqslant f(x_0) )$ 那么 $f'(x_0) = 0$
* **罗尔定理**: 如果函数 $f(x)$ 满足
    1. 在闭区间 $[a, b]$ 上连续
    1. 在闭区间 $(a, b)$ 内可导
    1. 在区间断点处的函数值相等, 即 $f(a) = f(b)$

    那么在 $(a, b)$ 内至少有一点 $\xi(a < \xi < b)$, 使得 $f'(\xi) = 0$
* **拉格朗日中值定理**: 如果函数 $f(x)$ 满足
    1. 在闭区间 $[a, b]$ 上连续
    1. 在闭区间 $(a, b)$ 内可导

    那么在 $(a, b)$ 内至少有一点 $\xi(a < \xi < b)$, 使等式 $f(b) - f(a) = f'(\xi)(b-a)$ 成立
* **定理**: 如果函数 $f(x)$ 在区间 $I$ 上的导数恒为零, 那么 $f(x)$ 在区间 $I$ 上是一个常数
* **柯西中值定理**: 如果函数 $f(x)$ 及 $F(x)$ 满足
    1. 在闭区间 $[a, b]$ 上连续
    1. 在闭区间 $(a, b)$ 内可导
    1. 对任一 $x \in (a, b), F'(x) \ne 0$ 内可导

    那么在 $(a, b)$ 内至少有一点 $\xi$, 使等式 $\dfrac{f(b)-f(a)}{F(b)-F(a)} = \dfrac{f'(\xi)}{F'(\xi)}$ 成立

## 3.6. 洛必达法则
1. $\displaystyle \lim _{x \to a} \dfrac{f(x)}{F(x)} = \lim _{x \to a} \dfrac{f'(x)}{F'(x)} = \cdots = \lim _{x \to a} \dfrac{f^{(n)}(x)}{F^{(n)}(x)}$
    * 成立的前提:
        1. 当 $x \to a$ 时, 函数 $f(x)$ 及 $F(x)$ 都趋于零
        1. 在点 $a$ 的某去心邻域内, $f'(x)$ 及 $F'(x)$ 都存在且 $F'(x) \ne 0$
        1. $\displaystyle \lim _{x \to a} \dfrac{f'(x)}{F'(x)}$ 存在 (或为无穷大)
1. $\displaystyle \lim _{x \to \infin} \dfrac{f(x)}{F(x)} = \lim _{x \to \infin} \dfrac{f'(x)}{F'(x)}= \cdots = \lim _{x \to \infin} \dfrac{f^{(n)}(x)}{F^{(n)}(x)}$
    * 成立的前提:
        1. 当 $x \to \infin$ 时, 函数 $f(x)$ 及 $F(x)$ 都趋于零
        1. 在 $|x| > N$ 时 $f'(x)$ 及 $F'(x)$ 都存在, 且 $F'(x) \ne 0$
        1. $\displaystyle \lim _{x \to \infin} \dfrac{f'(x)}{F'(x)}$ 存在 (或为无穷大)

## 3.7. 泰勒公式
* **泰勒 (Taylor) 中值定理**: 如果函数 $f(x)$ 在含有 $x_0$ 的某个开区间 $(a, b)$ 内具有直到 $(n+1)$ 阶的导数, 则对任一 $x \in (a, b)$ , 有$$f(x) = f(x_0) + f'(x_0)(x-x_0) + \dfrac{f''(x_0)}{2!}(x-x_0)^2 + \cdots + \dfrac{f^{(n)}(x_0)}{n!}(x-x_0)^n + R_n(x)$$ 其中 $R_n(x) = \dfrac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}$ (拉格朗日余项), 这里 $\xi$ 是 $x_0$ 与 $x$ 之间的某个值.
    * 在不需要余项的精确表达式时, n 阶泰勒公式也可以写成 $$f(x) = f(x_0) + f'(x_0)(x-x_0) + \dfrac{f''(x_0)}{2!}(x-x_0)^2 + \cdots + \dfrac{f^{(n)}(x_0)}{n!}(x-x_0)^n + o[(x-x_0)^n]$$
* **麦克劳林 (Maclaurin) 公式**: $$f(x) = f(0) + f'(0)(x) + \dfrac{f''(0)}{2!}x^2 + \cdots + \dfrac{f^{(n)}(0)}{n!}x^n + \dfrac{f^{(n+1)}(\theta x)}{(n+1)!}x^{n+1} \quad (0 < \theta < 1)$$

## 3.8. 函数极值、单调性、凹凸性
* **极值**:
    1. **必要条件**: 设函数 $f(x)$ 在 $x_0$ 处可导, 且在 $x_0$ 处取得极值 $\implies$ $f'(x_0) = 0$
    1. **第一充分条件**: 设函数 $f(x)$ 在 $x_0$ 处连续, 且在 $x_0$ 的某去心邻域 $\mathring U(x_0, \delta)$ 处可导
        * 若 $x \in (x_0- \delta, x_0)$ 时, $f'(x) > 0$, 而 $x \in (x_0, x_0 + \delta)$ 时, $f'(x) < 0$, 则 $f(x)$ 在 $x_0$ 处取得最大值
        * 若 $x \in (x_0- \delta, x_0)$ 时, $f'(x) < 0$, 而 $x \in (x_0, x_0 + \delta)$ 时, $f'(x) > 0$, 则 $f(x)$ 在 $x_0$ 处取得最小值
        * 若 $x \in \mathring U(x_0, \delta)$ 时, $f'(x)$ 的符号保持不变, 则 $f(x)$ 在 $x_0$ 处没有极值
    1. **第二充分条件**: 设函数 $f(x)$ 在 $x_0$ 处具有二阶导数且 $f'(x_0) = 0, f''(x_0) \ne 0$,
        * 当 $f''(x_0) < 0$ 时, $f(x)$ 在 $x_0$ 处取得最大值
        * 当 $f''(x_0) > 0$ 时, $f(x)$ 在 $x_0$ 处取得最小值
* **单调性**: 设函数 $f(x)$ 在 $[a, b]$ 上连续, 在 $(a, b)$ 内可导,
    1. 在 $(a, b)$ 内 $f'(x) > 0$ $\implies$ 函数 $f(x)$ 在 $[a, b]$ 上单调增加
    1. 在 $(a, b)$ 内 $f'(x) < 0$ $\implies$ 函数 $f(x)$ 在 $[a, b]$ 上单调减少
* **凹凸性**: 设函数 $y = f(x)$ 在 $[a, b]$ 上连续, 在 $(a, b)$ 内具有一阶和二阶导数,
    1. 在 $(a, b)$ 内 $f''(x) > 0$ $\implies$ 函数 $f(x)$ 在 $[a, b]$ 上的图形是凹的
    1. 在 $(a, b)$ 内 $f''(x) < 0$ $\implies$ 函数 $f(x)$ 在 $[a, b]$ 上的图形是凸的
    1. 在 $(a, b)$ 内 $f''(x_0) = 0$ $\implies$  $\big(x_0, f(x_0)\big)$ 是拐点

# 4. 多元微分

## 4.1. 多元函数
* **点集**: 内点, 外点, 边界点, 聚点, 开集, 闭集, 连通集, 开区域, 闭区域, 有界集, 无界集
* **定义**:
    * **多元函数**: 设 $D$ 是 $R^2$ 的一个非空子集, 称映射 $f: D \to \R$ 为定义在 $D$ 上的二元函数, 通常记为 $$z = f(x,y), \ (x,y) \in D \ 或\ z = f(P), \ P \in D$$ 其中点集 $D$ 称为该函数的**定义域**, $x, y$ 称为**自变量**, $z$ 称为**因变量**
        * $n$ 元函数: $u = f(x_1, x_2, \cdots, x_n), \ (x_1, x_2, \cdots, x_n) \in D$
    * **极限**: 设二元函数 $f(P) = f(x,y)$ 的定义域为 $D$, $P_0(x_0, y_0)$ 是 $D$ 的聚点, 如果存在常数 $A$, 对于任意给定的正数 $\varepsilon$, 总存在正数 $\delta$, 使得当点 $P(x,y) \in D \cap \mathring U(P_0, \delta)$ 时, 都有 $|f(P) -A| = |f(x,y) -A| < \varepsilon$ 成立, 那么就称常数 $A$ 为函数 $f(x,y)$ 当 $(x,y) \to (x_0, y_0)$ 时的极限, 记作 $$\lim_{(x,y) \to (x_0, y_0)} f(x,y) = A 或 f(x,y) \to A \ ((x,y) \to (x_0, y_0))$$ 也记作 $$\lim_{P \to P_0} f(P) = A 或 f(P) \to A \ (P \to P_0)$$
        * 二元函数的极限也叫**二重极限**
    * **连续**: 设二元函数 $f(P) = f(x,y)$ 的定义域为 $D$, $P_0(x_0, y_0)$ 为 $D$ 的聚点, 且 $P_0 \in D$, 如果 $\displaystyle \lim _{(x,y) \to (x_0, y_0)} f(x,y) = f(x_0, y_0)$, 则称函数 $f(x,y)$ 在点 $P_0(x_0, y_0)$ 连续
    * **间断点**: 设二元函数 $f(P) = f(x,y)$ 的定义域为 $D$, $P_0(x_0, y_0)$ 为 $D$ 的聚点, 如果函数 $f(x,y)$ 在点 $P_0(x_0, y_0)$ 不连续, 则称 $P_0(x_0, y_0)$ 为间断点
* **连续性的定理**:
    1. **有界性与最大值最小值定理**: 在有界闭区域 $D$ 上的多元连续函数, 必定在 $D$ 上有界, 且能取得它的最大值和最小值
    1. **介值定理**: 在有界闭区域 $D$ 上的多元连续函数必取得介于最大值和最小值之间的任何值
    1. **一致连续性定理**: 在有界闭区域 $D$ 上的多元连续函数必定在 $D$ 上一致连续

## 4.2. 偏导数与全微分
* **偏导数**: 设函数 $z=f(x,y)$ 在点 $(x_0, y_0)$ 的某一领域内有定义, 当 $y$ 固定在 $y_0$ 而 $x$ 在 $x_0$ 处有增量 $\Delta x$ 时, 相应的函数有增量 $f(x_0 + \Delta x, y_0) - f(x_0, y_0)$, 如果 $\displaystyle \lim _{\Delta x \to 0} \dfrac {f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x}$ 存在, 则称此极限为函数 $z=f(x,y)$ 在点 $(x_0, y_0)$ 处对 $x$ 的**偏导数**, 记作 $$\dfrac{\partial z}{\partial x} \bigg | _{\begin{gathered} x = x_0 \\ y=y_0 \end{gathered}}, \dfrac{\partial f}{\partial x} \bigg | _{\begin{gathered} x = x_0 \\ y=y_0 \end{gathered}}, \dfrac{\partial z_x}{\partial x} \bigg | _{\begin{gathered} x = x_0 \\ y=y_0 \end{gathered}}, f_x(x_0, y_0)$$
* **高阶偏导数**
    * $\dfrac {\partial}{\partial x}(\dfrac {\partial z}{\partial x}) = \dfrac {\partial ^2 z}{\partial x^2}= f_{xx} (x,y)$
        </br>
    * $\dfrac {\partial}{\partial y}(\dfrac {\partial z}{\partial x}) = \dfrac {\partial ^2 z}{\partial x \partial y}= f_{xy} (x,y)$
        </br>
    * $\dfrac {\partial}{\partial x}(\dfrac {\partial z}{\partial y}) = \dfrac {\partial ^2 z}{\partial y \partial x}= f_{yx} (x,y)$
        </br>
    * $\dfrac {\partial}{\partial y}(\dfrac {\partial z}{\partial y}) = \dfrac {\partial ^2 z}{\partial y^2}= f_{yy} (x,y)$
        </br>
        * 第二三个称为**混合偏导数**
    * **定理**: 如果函数 $z=f(x,y)$ 的两个二阶混合偏导数 $\dfrac {\partial ^2 z}{\partial x \partial y}$ 及 $\dfrac {\partial ^2 z}{\partial y \partial x}$ 在区域 $D$ 内连续, 那么在该区域内这两个二阶偏导数必相等, 即 $f_{xy} (x,y) = f_{yx} (x,y)$
* **全微分**: 设函数 $z=f(x,y)$ 在点 $(x,y)$ 的某邻域内有定义, 如果函数在点 $(x,y)$ 的全增量 $\Delta z = f(x+\Delta x, y+\Delta y) - f(x,y)$ 可表示为 $\Delta z = A \Delta x + B \Delta y + o(\rho)$, 其中 $A, B$ 不依赖于 $\Delta x, \Delta y$ 而仅与 $x, y$ 有关, $\rho = \sqrt {(\Delta x)^2 + (\Delta y)^2}$, 则称函数 $z=f(x,y)$ 在点 $(x,y)$ **可微分**, 而 $A \Delta x + B \Delta y$ 称为函数 $z=f(x,y)$ 在点 $(x,y)$ 的**全微分**, 记作 $\mathrm{d} z$, 即 $$\mathrm{d} z = A \Delta x + B \Delta y$$
    * 偏增量, 偏微分, 全增量
* **全微分定理**:
    1. **必要条件**: 如果函数 $z=f(x,y)$ 在点 $(x,y)$ 可微分, 则该函数在点 $(x,y)$ 的偏导数 $\dfrac {\partial z}{\partial x}, \dfrac {\partial z}{\partial y}$ 必定存在, 且函数 $z=f(x,y)$ 在点 $(x,y)$ 的全微分为 $$\mathrm{d} z = \dfrac{\partial z}{\partial x} \Delta x + \dfrac {\partial z}{\partial y} \Delta y$$
    1. **充分条件**: 如果函数 $z=f(x,y)$ 的偏导数 $\dfrac {\partial z}{\partial x}, \dfrac {\partial z}{\partial y}$ 点 $(x,y)$ 连续, 则函数在该点可微分

## 4.3. 多元复合函数的求导法则
1. 如果函数 $u=\varphi(t)$ 及 $v=\psi(t)$ 都在点 $t$ 可导, 函数 $z=f(u,v)$ 在对应点 $(u,v)$ 具有连续偏导数, 则复合函数 $z=f[\varphi(t), \psi(t)]$ 在点 $t$ 可导, 且有 $$\dfrac{\mathrm{d} z}{\mathrm{d} t} = \dfrac {\partial z}{\partial u} \dfrac {\mathrm{d} u}{\mathrm{d} t} + \dfrac {\partial z}{\partial v} \dfrac {\mathrm{d} v}{\mathrm{d} t}$$
1. 如果函数 $u=\varphi(x,y)$ 及 $v=\psi(x,y)$ 都在点 $(x,y)$ 具有对 $x$ 及对 $y$ 的偏导数, 函数 $z=f(u,v)$ 在对应点 $(u,v)$ 具有连续偏导数, 则复合函数 $z=f[\varphi(x,y), \psi(x,y)]$ 在点 $(x,y)$ 的两个偏导数都存在, 且有 $$\begin{aligned} \dfrac{\partial z}{\partial x} = \dfrac {\partial z}{\partial u} \dfrac {\partial u}{\partial x} + \dfrac {\partial z}{\partial v} \dfrac {\partial v}{\partial x} \\ \\ \dfrac{\partial z}{\partial y} = \dfrac {\partial z}{\partial u} \dfrac {\partial u}{\partial y} + \dfrac {\partial z}{\partial v} \dfrac {\partial v}{\partial y} \end{aligned}$$
1. 如果函数 $u=\varphi(x,y)$ 在点 $(x,y)$ 具有对 $x$ 及对 $y$ 的偏导数, 函数 $v=\psi(y)$ 在点 $y$ 可导,  函数 $z=f(u,v)$ 在对应点 $(u,v)$ 具有连续偏导数, 则复合函数 $z=f[\varphi(x,y), \psi(y)]$ 在点 $(x,y)$ 的两个偏导数都存在, 且有 $$\begin{aligned} \dfrac{\partial z}{\partial x} &= \dfrac {\partial z}{\partial u} \dfrac {\partial u}{\partial x} \\ \\ \dfrac{\partial z}{\partial y} &= \dfrac {\partial z}{\partial u} \dfrac {\partial u}{\partial y} + \dfrac {\partial z}{\partial v} \dfrac {\mathrm{d} v}{\mathrm{d} y} \end{aligned}$$
1. **全微分形式不变性**: 设函数 $z=f(u,v)$ 具有连续偏导数, 则有全微分 $$\mathrm{d} z = \dfrac {\partial z}{\partial u} \mathrm{d} u + \dfrac {\partial z}{\partial v} \mathrm{d} v$$

## 4.4. 隐函数的求导
1. 设函数 $F(x,y)$ 在点 $P(x_0, y_0)$ 的某一邻域内具有连续偏导数, 且 $F(x_0,y_0) = 0, F_y(x_0, y_0) \ne 0$, 则方程 $F(x,y)=0$ 在点 $(x_0, y_0)$ 的某一邻域内恒能唯一确定一个连续且具有连续导数的函数 $y=f(x)$, 它满足条件 $y_0 = f(x_0)$, 并有 $$\dfrac {\mathrm{d} y}{\mathrm{d} x} = - \dfrac {F_x}{F_y}$$
1. 设函数 $F(x,y,z)$ 在点 $P(x_0, y_0,z_0)$ 的某一邻域内具有连续偏导数, 且 $F(x_0,y_0, z_0) = 0, F_z(x_0, y_0, z_0) \ne 0$, 则方程 $F(x,y,z)=0$ 在点 $(x_0, y_0, z_0)$ 的某一邻域内恒能唯一确定一个连续且具有连续导数的函数 $z=f(x,y)$, 它满足条件 $z_0 = f(x_0, y_0)$, 并有 $$\dfrac {\partial z}{\partial x} = - \dfrac {F_x}{F_z}, \dfrac {\partial z}{\partial y} = - \dfrac {F_y}{F_z}$$
1. 设函数 $F(x,y,u,v), G(x,y,u,v)$ 在点 $P(x_0, y_0, u_0, v_0)$ 的某一邻域内具有对各个变量的连续偏导数, 有 $F(x_0, y_0, u_0, v_0) = 0, G(x_0, y_0, u_0, v_0) = 0$, 且偏导数所组成的函数行列式 (或称雅可比(Jacobi)式) $$ J = \dfrac {\partial (F,G)}{\partial (u,v)} = \begin{vmatrix}
        \dfrac {\partial F}{\partial u} &\dfrac {\partial F}{\partial v} \\
        \\
        \dfrac {\partial G}{\partial u} &\dfrac {\partial G}{\partial v} \end{vmatrix}$$ 在点 $P(x_0, y_0, u_0, v_0)$ 不等于 0, 则方程组 $F(x,y,u,v) =0, G(x,y,u,v)=0$ 在点 $(x_0, y_0, u_0, v_0)$ 的某一邻域内恒能唯一确定一个连续且具有连续导数的函数 $u=u(x,y), v=v(x,y)$, 它们满足条件 $u_0 = u(x_0, y_0), v_0 = v(x_0, y_0)$, 并有$$\begin{aligned}
                \dfrac {\partial u}{\partial x} = - \dfrac{1}{J} \dfrac{\partial (F,G)}{\partial (x,v)}=- \dfrac{\begin{vmatrix} F_x &F_v \\ G_x &G_v\end{vmatrix}}{\begin{vmatrix} F_u &F_v \\ G_u &G_v\end{vmatrix}} \\ \\
                \dfrac {\partial v}{\partial x} = - \dfrac{1}{J} \dfrac{\partial (F,G)}{\partial (u,x)}=- \dfrac{\begin{vmatrix} F_u &F_x \\ G_u &G_x\end{vmatrix}}{\begin{vmatrix} F_u &F_v \\ G_u &G_v\end{vmatrix}} \\ \\
                \dfrac {\partial u}{\partial y} = - \dfrac{1}{J} \dfrac{\partial (F,G)}{\partial (y,v)}=- \dfrac{\begin{vmatrix} F_y &F_v \\ G_y &G_v\end{vmatrix}}{\begin{vmatrix} F_u &F_v \\ G_u &G_v\end{vmatrix}} \\ \\
                \dfrac {\partial v}{\partial y} = - \dfrac{1}{J} \dfrac{\partial (F,G)}{\partial (u,y)}=- \dfrac{\begin{vmatrix} F_u &F_y \\ G_u &G_y\end{vmatrix}}{\begin{vmatrix} F_u &F_v \\ G_u &G_v\end{vmatrix}}
            \end{aligned}$$

## 4.5. 多元函数微分学的几何应用

### 4.5.1. 一元向量值函数及其导数
* **定义**:
    * **一元向量值函数**: 设数集 $D \subset \R$, 则称映射 $f: D \to \R^n$ 为一元**向量值函数**, 通常记为 $\bm r = f(t), t \in D$, 其中数集 $D$ 称为函数的**定义域**, $t$ 称为**自变量**, $\bm r$ 称为**因变量**
    * **极限**: 设向量值函数 $f(t)$ 在 $t_0$ 的某一去心邻域内有定义, 如果存在一个常向量 $\bm r_0$ 时, 对应的函数值 $f(t)$ 都满足不定式 $|f(t)-\bm r_0| < \varepsilon$, 那么, 常向量 $\bm r_0$ 就叫做向量值函数 $f(t)$ 当 $t \to t_0$ 时的**极限**, 记作 $$\lim _{t \to t_0} f(t) = \bm r_0 或 f(t) \to \bm r_0, t \to t_0$$
    * **导向量**: 设向量值函数 $\bm r = f(t)$ 在点 $t_0$ 的某一邻域内有定义, 如果 $\displaystyle \lim _{\Delta t \to 0} \dfrac {\Delta \bm r}{\Delta t} = \lim _{\Delta t \to 0} \dfrac {f(t_0 + \Delta t) - f(t_0)}{\Delta t}$ 存在, 那么就称这个极限向量为向量值函数 $\bm r = f(t)$ 在 $t_0$ 处的导数或**导向量**, 记作 $$f'(t_0) 或 \dfrac {\mathrm{d} \bm r}{\mathrm{d} t} \bigg |_{t=t_0}$$
* **向量值函数运算法则**:
    * $\dfrac {\mathrm{d}}{\mathrm{d}t} \bm{C} = 0$
        </br>
    * $\dfrac {\mathrm{d}}{\mathrm{d}t} [c \bm u (t)] = c \bm u'(t)$
        </br>
    * $\dfrac {\mathrm{d}}{\mathrm{d}t} [\bm u(t) \pm \bm v(t)] = \bm u'(t) \pm  \bm v'(t)$
        </br>
    * $\dfrac {\mathrm{d}}{\mathrm{d}t} [\varphi(t) \bm u(t)] = \varphi'(t) \bm u(t) + \varphi(t) \bm u'(t)$
        </br>
    * $\dfrac {\mathrm{d}}{\mathrm{d}t} [\bm u(t) \cdot \bm v(t)] = \bm u'(t) \cdot \bm v(t) + \bm u(t) \cdot \bm v'(t)$
        </br>
    * $\dfrac {\mathrm{d}}{\mathrm{d}t} [\bm u(t) \times \bm v(t)] = \bm u'(t) \times \bm v(t) + \bm u(t) \times \bm v'(t)$
        </br>
    * $\dfrac {\mathrm{d}}{\mathrm{d}t} \bm u [\varphi (t)] = \varphi '(t) \bm u'[\varphi(t)]$

### 4.5.2. 不同坐标下的微分
* 直角坐标系:
    * $x,y,z$ 轴的微分: $\mathrm{d}x, \mathrm{d}y, \mathrm{d}z$
* 极坐标系:
    * $\rho$ 轴的微分: $\mathrm{d}\rho$
    * $\theta$ 轴的微分: $\rho \mathrm{d} \theta$
* 圆柱坐标系:
    * $\rho$ 轴的微分: $\mathrm{d}\rho$
    * $\theta$ 轴的微分: $\rho \mathrm{d} \theta$
    * $z$ 轴的微分: $\mathrm{d}z$
* 球坐标系:
    * $r$ 轴的微分: $\mathrm{d}r$
    * $\varphi$ 轴的微分: $r \mathrm{d} \varphi$
    * $\theta$ 轴的微分: $r \sin \varphi \mathrm{d} \theta$

### 4.5.3. 空间曲线的切线与法平面
* 设空间曲线 $\Gamma$ 的参数方程为 $\begin{cases} x =\varphi (t) \\ y = \psi (t) \\ z = \omega (t) \end{cases}$, $t \in [\alpha, \beta]$
    假定三个函数在 $[\alpha, \beta]$ 上可导, 且三个导数不同时为 0. 设点 $M$ 对应的参数为 $t_0$, 则在点 $M$ 处:
    * **切向量**为 $$T = f'(t_0) = \bigg(\varphi '(t_0), \psi '(t_0), \omega '(t0) \bigg)$$
    * **切线**方程为 $$\dfrac {x - x_0}{\varphi '(t_0)} = \dfrac {y - y_0}{\psi '(t_0)} = \dfrac {z - z_0}{\omega '(t_0)}$$
    * **法平面**方程为 $$\varphi '(t_0)(x - x_0) + \psi '(t_0)(y - y_0) + \omega '(t_0)(z - z_0) = 0$$
* 空间曲线 $\begin{cases} F(x,y,z) = 0 \\ G(x,y,z) = 0 \end{cases}$ 在点 $(x_0, y_0, z_0)$ 处
    * **切向量**为 $$T = \Bigg(\begin{vmatrix} F_y &F_z \\ G_y &G_z \end{vmatrix},
                               \begin{vmatrix} F_z &F_x \\ G_z &G_x \end{vmatrix},
                               \begin{vmatrix} F_x &F_y \\ G_x &G_y \end{vmatrix} \Bigg)$$
    * **切线**方程为 $$\dfrac {x - x_0}{\begin{vmatrix} F_y &F_z \\ G_y &G_z \end{vmatrix}} =
                       \dfrac {y - y_0}{\begin{vmatrix} F_z &F_x \\ G_z &G_x \end{vmatrix}} =
                       \dfrac {z - z_0}{\begin{vmatrix} F_x &F_y \\ G_x &G_y \end{vmatrix}}$$
    * **法平面**方程为 $$\begin{vmatrix} F_y &F_z \\ G_y &G_z \end{vmatrix}(x - x_0) +
                         \begin{vmatrix} F_z &F_x \\ G_z &G_x \end{vmatrix}(y - y_0) +
                         \begin{vmatrix} F_x &F_y \\ G_x &G_y \end{vmatrix}(z - z_0) = 0$$

### 4.5.4. 曲面的切平面与法线
* 曲面方程 $F(x, y ,z)$ 在点 $(x_0, y_0, z_0)$ 处
    * **切平面**方程为 $$F_x(x_0, y_0, z_0)(x-x_0) + F_y(x_0, y_0, z_0)(y-y_0) + F_z(x_0, y_0, z_0)(z-z_0) = 0$$
    * **法线**方程为 $$\dfrac {x - x_0}{F_x(x_0, y_0, z_0)} = \dfrac {y - y_0}{F_y(x_0, y_0, z_0)} = \dfrac {z - z_0}{F_z(x_0, y_0, z_0)}$$
    * **法向量**为 $$\bm n = [F_x(x_0, y_0, z_0), F_y(x_0, y_0, z_0), F_z(x_0, y_0, z_0)]$$

## 4.6. 方向导数与梯度

### 4.6.1. 方向导数
* **定理**: 如果函数 $f(x,y)$ 在点 $P_0(x_0, y_0)$ 可微分, 那么函数在该点沿任一方向 $l$ 的**方向导数**存在, 且有 $$\dfrac {\partial f}{\partial l} \bigg | _{(x_0, y_0)} = f_x(x_0, y_0) \cos \alpha + f_y(x_0, y_0) \cos \beta$$ 其中 $\cos \alpha, \cos \beta$ 是方向 $l$ 的方向余弦

### 4.6.2. 梯度
* 函数 $f(x,y)$ 在点 $P_0(x_0, y_0)$ 的**梯度**: $f_x(x_0, y_0) \bm i + f_y(x_0, y_0) \bm j$, 记作 $\bold {grad} f(x_0, y_0)$ 或 $\nabla f(x_0, y_0)$, 即 $$\bold {grad} f(x_0, y_0) = \nabla f(x_0, y_0) = f_x(x_0, y_0) \bm i + f_y(x_0, y_0) \bm j$$ 其中 $\nabla = \dfrac {\partial}{\partial x} \bm i + \dfrac {\partial}{\partial y} \bm j$ 称为**向量微分算子**或**Nabla算子** $$\nabla f= \dfrac {\partial f}{\partial x} \bm i + \dfrac {\partial f}{\partial y} \bm j$$
* 如果函数 $f(x,y)$ 在点 $P_0(x_0, y_0)$ 可微分, $\bm {e_l} = (\cos \alpha, \cos \beta)$ 是方向 $l$ 同向的单位向量, 则 $$\dfrac {\partial f}{\partial l} \bigg | _{(x_0, y_0)} = f_x(x_0, y_0) \cos \alpha + f_y(x_0, y_0) \cos \beta = \bold {grad} f(x_0, y_0) \cdot \bm {e_l} = |\bold {grad} f(x_0, y_0)| \cos \theta$$ 其中 $\theta = (\widehat {{\bold {grad} f(x_0, y_0)}, {\bm {e_l}}})$
* 概念:
    * 等值面
    * 数量场, 向量场
    * 势函数, 势场, 引力场, 引力势, 电力场, 电势

## 4.7. 多元函数的极值及其求法
* **定理1 (必要条件)**: 设函数 $z = f(x,y)$ 在点 $(x_0, y_0)$ 具有偏导数, 且在点 $(x_0, y_0)$ 处具有极值, 则有 $$f_x(x_0, y_0) = 0, f_y(x_0, y_0) = 0$$
* **定理2 (充分条件)**: 设函数 $z = f(x,y)$ 在点 $(x_0, y_0)$ 的某邻域内连续且具有一阶及二阶连续偏导数, 又 $f_x(x_0, y_0) = 0, f_y(x_0,y_0) = 0$, 令 $$f_{xx}(x_0, y_0) = A, f_{xy}(x_0, y_0) = B, f_{yy}(x_0, y_0) = C$$ 则 $f(x,y)$ 在点 $(x_0, y_0)$ 处是否取得极值的条件如下:
    1. $AC-B^2 > 0$ 时具有极值, 且当 $A < 0$ 时有极大值, 当 $A > 0$ 时有极小值
    1. $AC-B^2 < 0$ 时没有极值
    1. $AC-B^2 = 0$ 时不确定是否有极值
* 条件极值: 对自变量有附加条件的极值
* **拉格朗日乘法**求条件极值: 要找函数 $z = f(x,y)$ 在附加条件 $\varphi(x, y) = 0$ 下可能的极值点, 可以先作拉格朗日函数 $$L(x, y) = f(x,y) + \lambda \varphi(x, y)$$ 其中 $\lambda$ 为参数, 求其对 $x, y$ 的一阶偏导数, 并使之为 0, 然后求解方程组 $$\begin{cases} f_x(x, y) + \lambda \varphi _x (x, y) = 0 \\ f_y(x, y) + \lambda \varphi _y (x, y) = 0 \\ \varphi (x, y) = 0 \end{cases}$$ 这样得到的 $(x, y)$ 就是函数 $z = f(x,y)$ 在附加条件 $\varphi(x, y) = 0$ 下可能的极值点
    > 方法可以推广, 例如求函数 $$u = f(x, y, z, t)$$ 在附加条件 $$\varphi(x,y,z,t)=0, \psi (x,y,z,t)=0$$ 下的极值, 可以先作拉格朗日函数 $$L(x,y,z,t)=f(x,y,z,t) + \lambda \varphi(x,y,z,t) + \mu \psi (x,y,z,t)$$

## 4.8. 二元函数的泰勒公式
* 设 $z = f(x,y)$ 在点 $(x_0, y_0)$ 的某一去心邻域内连续且具有直到 $(n+1)$ 阶的连续偏导数, $(x_0 + h, y_0+k)$ 为此邻域内的任一点, 则有 $$\begin{aligned}
        f(x_0 + h, y_0+k) = & f(x_0, y_0) + \\
        & (h \dfrac {\partial}{\partial x} + k \dfrac {\partial}{\partial y})f(x_0, y_0) + \\
        & \dfrac {1}{2!}(h \dfrac {\partial}{\partial x} + k \dfrac {\partial}{\partial y})^2 f(x_0, y_0) + \\
        & \cdots + \\
        & \dfrac {1}{n!}(h \dfrac {\partial}{\partial x} + k \dfrac {\partial}{\partial y})^n f(x_0, y_0) + \\
        & \dfrac {1}{(n+1)!}(h \dfrac {\partial}{\partial x} + k \dfrac {\partial}{\partial y})^{n+1} f(x_0 + \theta h, y_0 + \theta k) \ \ (0 < \theta < 1)
    \end{aligned}$$ 其中记号
    * $(h \dfrac {\partial}{\partial x} + k \dfrac {\partial}{\partial y})f(x_0, y_0)$ 表示 $h f_x (x_0, y_0) + k f_y (x_0, y_0)$
        </br>
    * $(h \dfrac {\partial}{\partial x} + k \dfrac {\partial}{\partial y})^2 f(x_0, y_0)$ 表示 $h^2 f_{xx} (x_0, y_0) + 2hk f_{xy}(x_0, y_0) + k^2 f_{yy} (x_0, y_0)$
        </br>
    * $(h \dfrac {\partial}{\partial x} + k \dfrac {\partial}{\partial y})^m f(x_0, y_0)$ 表示 $\displaystyle \sum _{p=0}^{m} C_m^p h^p k^{m-p} \dfrac {\partial ^m f}{\partial x^p \partial y^{m-p}} \bigg| _{(x_0, y_0)}$

# 5. 积分

## 5.1. 不定积分的概念及性质
* **定义**:
    * **原函数**: 如果在区间 $I$ 上, 可导函数 $F(x)$ 的导函数为 $f(x)$, 即对任一 $x \in I$, 都有 $F'(x) = f(x) 或 \mathrm{d}F(x) = f(x)\mathrm{d}x$, 那么函数 $F(x)$ 就称为 $f(x)(或 f(x)\mathrm{d}x)$ 在区间 $I$ 上的原函数
    * **不定积分**: 区间 $I$ 上, 函数 $f(x)$ 的带有任意常数项的原函数称为 $f(x)(或 f(x)\mathrm{d}x)$ 在区间 $I$ 上的**不定积分**, 记作 $$\int f(x) \mathrm{d}x$$ 其中记号 $\displaystyle \int$ 称为**积分号**, $f(x)$ 称为**被积函数**, $f(x)\mathrm{d}x$ 称为**被积表达式**, x称为**积分变量**
* **原函数存在定理**: 如果函数 $f(x)$ 在区间 $I$ 上连续, 那么在区间 $I$ 上存在可导函数 $F(x)$, 使对任一 $x \in I$ 都有 $F'(x) = f(x)$
    * 即: 连续函数一定有原函数
* **性质**:
    1. 设函数 $f(x)$ 及 $g(x)$ 的原函数都存在, 则 $\displaystyle \int [f(x)+g(x)] \mathrm{d}x= \int f(x) \mathrm{d}x + \int g(x) \mathrm{d}x$
    1. 设函数 $f(x)$ 的原函数存在, $k$ 为非0常数, 则 $\displaystyle \int kf(x) \mathrm{d}x = k\int f(x) \mathrm{d}x$

## 5.2. 定积分的概念及性质
* **定积分**: $\displaystyle \int_a^b f(x) \mathrm{d}x$, 其中 $a$ 叫做**积分下限**, $b$ 叫做**积分上限**, $[a, b]$ 叫做**积分区间**
* **定理**:
    1. 设函数 $f(x)$ 在区间 $[a, b]$ 上连续, 则 $f(x)$ 在区间 $[a, b]$ 上可积
    1. 设函数 $f(x)$ 在区间 $[a, b]$ 上有界, 且只有有限个间断点, 则 $f(x)$ 在区间 $[a, b]$ 上可积
    1. 设 $M$ 及 $m$ 分别是函数 $f(x)$ 区间 $[a, b]$ 上的最大值和最小值, 则 $m(b-a) \leqslant \int_a^b f(x) \mathrm{d}x \leqslant M(b-a) \quad (a < b)$
    1. **定积分中值定理**: 如果函数 $f(x)$ 在区间 $[a, b]$ 上连续, 则区间 $[a, b]$ 上至少存在一个点 $\xi$, 满足 $\displaystyle \int_a^b f(x) \mathrm{d}x = f(\xi)(b-a) \quad (a < b)$
* **性质**:
    1. $\displaystyle \int_a^b [f(x) + g(x)] \mathrm{d}x = \int_a^b f(x) \mathrm{d}x + \int_a^b g(x) \mathrm{d}x$
    1. $\displaystyle \int_a^b kf(x) \mathrm{d}x = k\int_a^b f(x) \mathrm{d}x \ (k是常数)$
    1. 设$a < c < b$, 则 $\displaystyle \int_a^b f(x) \mathrm{d}x = \int_a^c f(x) \mathrm{d}x + \int_c^b f(x) \mathrm{d}x$
    1. 如果在区间 $[a, b]$ 上 $f(x) \equiv 1$, 则 $\displaystyle \int_a^b 1 \mathrm{d}x = \int_a^b \mathrm{d}x  = b-a$
    1. 如果在区间 $[a, b]$ 上 $f(x) \geqslant 0$, 则 $\displaystyle \int_a^b f(x) \mathrm{d}x \geqslant 0 \quad (a < b)$
        * 如果在区间 $[a, b]$ 上 $f(x) \leqslant g(x)$, 则 $\displaystyle \int_a^b f(x) \mathrm{d}x \leqslant  \int_a^b g(x) \mathrm{d}x \quad (a < b)$
        * $\displaystyle |\int_a^b f(x) \mathrm{d}x| \leqslant \int_a^b |f(x)| \mathrm{d}x \quad (a < b)$
* **积分上限函数**
    1. 如果函数 $f(x)$ 在区间 $[a, b]$ 上连续, 则积分上限函数 $\displaystyle \Phi(x) = \int_a^x f(t) \mathrm{d}t$ 在 $[a, b]$ 上可导, 并且其导数 $\displaystyle \Phi'(x) = \dfrac{\mathrm{d}}{\mathrm{d}x} \int_a^x f(t) \mathrm{d}t = f(x) \quad (a\leqslant x \leqslant b)$
    1. 如果函数 $f(x)$ 在区间 $[a, b]$ 上连续, 则函数 $\displaystyle \Phi(x) = \int_a^x f(t) \mathrm{d}t$ 就是 $f(x)$ 在 $[a, b]$ 上的一个原函数
* **牛顿-莱布尼兹公式**
    * 如果函数 $F(x)$ 是连续函数 $f(x)$ 在区间 $[a, b]$ 上的一个原函数, 则 $\displaystyle \int_a^b f(x) \mathrm{d}x = F(b) - F(a)$

## 5.3. 积分法则
* **第一类换元法**: 设函数 $f(\mu)$ 具有原函数, $\mu = \varphi(x)$ 可导, 则有换元法 $$\displaystyle \int f[\varphi(x)] \varphi'(x) \ \mathrm{d}x = \Big [\int f(u) \mathrm{d}u \Big ]_{\mu = \varphi(x)}$$
* **第二类换元法**: 设 $x=\psi (t)$ 是单调+可导的函数, 并且 $\psi' (t) \ne 0$; 又设 $f[\psi (t)]\psi' (t)$ 具有原函数, 则有换元公式 $$\displaystyle \int f(x) \mathrm{d}x = \Big[ \int f[\psi (t)]\psi' (t) \mathrm{d}t \Big]_{t=\psi^{-1}(x)}$$
* **分部积分法**: 设 $u=u(x), v = v(x)$ 具有连续导数, 则$$\int u v' \mathrm{d}x = u v - \int u' v \mathrm{d}x$$ $$\int u \mathrm{d}v = u v - \int v \mathrm{d}u$$

# 6. 重积分

## 6.1. 二重积分
* **定义**: 设 $f(x,y)$ 是有界闭区域 $D$ 上的有界函数, 将闭区域 $D$ 任意分成 n 个小闭区域 $\Delta \sigma _1, \Delta \sigma _2, \cdots, \Delta \sigma _n$, 其中 $\Delta \sigma _i$ 表示第 $i$ 个小闭区域, 也表示它的面积, 在每个 $\Delta \sigma _i$ 上任取一点 $(\varepsilon _i, \eta _i)$, 作乘积 $f(\varepsilon _i, \eta _i) \Delta \sigma _i (i = 1,2,\cdots,n)$, 并作和 $\displaystyle \sum _{i=1}^{n}f(\varepsilon _i, \eta _i) \Delta \sigma _i$. 如果当各个小闭区域的直径中的最大值 $\lambda$ 趋近于0时, 这和的极限总存在, 则称此极限为函数 $f(x,y)$ 在闭区域 $D$ 上的**二重积分**, 记作 $\displaystyle \iint _D f(x,y) \mathrm{d} \sigma$, 即 $$\iint _D f(x,y) \mathrm{d} \sigma = \lim _{\lambda \to 0} \sum _{i=1}^{n}f(\varepsilon _i, \eta _i) \Delta \sigma _i$$ 其中 $f(x,y)$ 叫**做被积函数**, $f(x,y) \mathrm{d} \sigma$ 叫做**被积表达式**, $\mathrm{d} \sigma$ 叫做**面积元素**, $x,y$ 叫做**积分变量**, $D$ 叫做**积分区域**, $\displaystyle \sum _{i=1}^{n}f(\varepsilon _i, \eta _i) \Delta \sigma _i$ 叫做**积分和**
    * 二重积分也记作 $$\iint _D f(x,y) \mathrm{d}x \mathrm{d} y$$ 其中 $\mathrm{d}x \mathrm{d} y$ 叫做**直角坐标系中的面积元素**
        * **极坐标系中的面积元素** $\mathrm{d} \rho \cdot \rho \mathrm{d} \theta = \rho \mathrm{d} \rho \mathrm{d} \theta$
* **性质**
    1. 设 $\alpha, \beta$ 为常数, 则 $$\iint _D [\alpha f(x, y) + \beta g(x,y)] \mathrm{d} \sigma = \alpha \iint _D f(x, y) \mathrm{d} \sigma + \beta \iint _D g(x,y) \mathrm{d} \sigma$$
    1. 如果闭区域 $D$ 被有限条曲线分为有限个部分闭区域, 则在 $D$ 上的二重积分等于各个部分闭区域上的二重积分的和, 即二重积分对于积分区域具有**可加性**
    1. 如果在 $D$ 上, $f(x, y) = 1$, $\sigma$ 为 $D$ 的面积, 则 $$\sigma = \iint 1 \mathrm{d} \sigma = \iint \mathrm{d} \sigma$$
    1. 如果在 $D$ 上, $f(x, y) \leqslant \varphi(x,y)$, 则有 $$\iint _D f(x,y) \mathrm{d} \sigma \leqslant \iint _D \varphi(x,y) \mathrm{d} \sigma$$
        特殊地, 由于 $-|f(x,y)| \leqslant f(x,y) \leqslant |f(x,y)|$, 又有 $$\bigg| \iint _D f(x,y) \mathrm{d} \sigma \bigg| \leqslant \iint _D f(x,y) \mathrm{d} \sigma$$
    1. 设 $M, m$ 分别是 $f(x,y)$ 在闭区域 $D$ 上的最大值和最小值, $\sigma$ 是 $D$ 的面积, 则有 $$m\sigma \leqslant \iint _D f(x,y) \mathrm{d} \sigma \leqslant M \sigma$$
    1. (二重积分的中值定理) 设函数 $f(x,y)$ 在闭区域 $D$ 上连续, $\sigma$ 是 $D$ 的面积, 则在 $D$ 上至少存在一点 $(\varepsilon, \eta)$, 使得 $$\iint _D f(x,y) \mathrm{d} \sigma = f(\varepsilon, \eta) \cdot \sigma$$
* **计算**: 化为两次单积分
* **二重积分的换元公式**: 设 $f(x,y)$ 在 $xOy$ 平面上的闭区域 $D$ 上连续, 变换 $$T:x=x(u,v),y=y(u,v)$$ 将 $uOv$ 平面上的闭区域 $D'$ 变为 $xOy$ 平面上的 $D$, 且满足:
    1. $x(u,v),y(u,v)$ 在 $D'$ 上具有一阶连续偏导数
    1. 在 $D'$ 上雅可比式 $$J(u,v) = \dfrac {\partial(x,y)}{\partial(u,v)} \ne 0$$
    1. 变换 $T: D' \to D$ 是一对一的

    则有 $$\iint _D f(x,y) \mathrm{d} x \mathrm{d} y = \iint _{D'} f\big[x(u,v), y(u,v)\big] |J(u,v)| \mathrm{d} u \mathrm{d} v$$

## 6.2. 三重积分
* **定义** 设 $f(x,y,z)$ 是空间有界闭区域 $\Omega$ 上的有界函数, 将 $\Omega$ 任意分成 n 个小闭区域 $\Delta v_1, \Delta v_2, \cdots, \Delta v_n$, 其中 $\Delta v_i$ 表示第 $i$ 个小闭区域, 也表示它的体积, 在每个 $\Delta v_i$ 上任取一点 $(\varepsilon _i, \eta _i, \xi _i)$, 作乘积 $f(\varepsilon _i, \eta _i, \xi _i) \Delta v_i \ (i = 1,2,\cdots, n)$, 并作和 $\displaystyle \sum _{i = 1}^{n} f(\varepsilon _i, \eta _i, \xi _i) \Delta v_i$, 如果当各小闭区域直径中的最大值 $\lambda$ 趋于0时这和的极限总存在, 则此极限为函数 $f(x,y,z)$ 在闭区域 $\Omega$ 上的**三重积分**, 记作 $\displaystyle \iiint _\Omega f(x,y,z) \mathrm{d} v$, 即 $$\iiint _\Omega f(x,y,z) \mathrm{d} v = \lim _{\lambda \to 0} \sum _{i = 1}^{n} f(\varepsilon _i, \eta _i, \xi _i) \Delta v_i$$ 其中 $\mathrm{d} v$ 叫做**体积元素**
    * 三重积分也记作 $$\iiint _\Omega f(x,y,z) \mathrm{d} x \mathrm{d} y \mathrm{d} z$$ 其中 $\mathrm{d} x \mathrm{d} y \mathrm{d} z$ 叫做**直角坐标系中的体积元素**
        * **柱面坐标系中的体积元素**: $\mathrm{d} \rho \cdot \rho \mathrm{d} \theta \cdot \mathrm{d} z = \rho \mathrm{d} \rho \mathrm{d} \theta \mathrm{d} z$
        * **球面坐标系中的体积元素**: $\mathrm{d}r \cdot r \mathrm{d} \varphi \cdot r \sin \varphi \mathrm{d} \theta  = r^2 \sin \varphi \mathrm{d} r \mathrm{d} \varphi \mathrm{d} \theta$
* **计算**: 化为三次单积分
* **二重积分的换元公式**: 参见**二重积分的换元公式**

## 6.3. 含参变量的积分
1. 如果函数 $f(x,y)$ 在矩形 $R=[a,b] \times [c,d]$ 上连续, 那么 $\varphi(x) = \int _c^d f(x,y) \mathrm{d} y \ (a \leqslant x \leqslant b)$ 在 $[a,b]$ 上也连续
1. 如果函数 $f(x,y)$ 在矩形 $R=[a,b] \times [c,d]$ 上连续, 则 $$\int _a^b \bigg[\int _c^d f(x,y) \mathrm{d} y \bigg] \mathrm{d}x = \int _c^d \bigg[\int _a^b f(x,y) \mathrm{d}x \bigg] \mathrm{d}y$$ 也可写成 $$\int _a^b \mathrm{d}x \int _c^d f(x,y) \mathrm{d} y = \int _c^d \mathrm{d}y \int _a^b f(x,y) \mathrm{d}x$$
1. 如果函数 $f(x,y)$ 及其偏导数 $f_x(x,y)$ 都在矩形 $R=[a,b] \times [c,d]$ 上连续, 那么 $\int _a^b f(x,y) \mathrm{d} y$ 在 $[a,b]$ 上可微分, 且 $$\dfrac {\mathrm{d}}{\mathrm{d} x} \int _c^d f(x,y) \mathrm{d} y = \int _c^d f_x(x,y) \mathrm{d} y$$
1. 如果函数 $f(x,y)$ 在矩形 $R=[a,b] \times [c,d]$ 上连续, 函数 $\alpha (x)$ 与 $\beta (x)$ 在区间 $[a,b]$ 上连续, 且 $c \leqslant \alpha(x) \leqslant d, c \leqslant \beta(x) \leqslant d \ (a \leqslant x \leqslant b)$, 则函数 $\Phi(x) = \int _{\alpha(x)}^{\beta(x)} f(x,y) \mathrm{d} y$ 在 $[a,b]$ 上也连续
1. 如果函数 $f(x,y)$ 及其偏导数 $f_x(x,y)$ 都在矩形 $R=[a,b] \times [c,d]$ 上连续, 函数 $\alpha (x)$ 与 $\beta (x)$ 在区间 $[a,b]$ 上可微, 且 $c \leqslant \alpha(x) \leqslant d, c \leqslant \beta(x) \leqslant d \ (a \leqslant x \leqslant b)$, 则函数 $\Phi(x) = \int _{\alpha(x)}^{\beta(x)} f(x,y) \mathrm{d} y$ 在 $[a,b]$ 上可微, 且 $$\Phi'(x) = \dfrac {\mathrm{d}}{\mathrm{d} x}\int _{\alpha(x)}^{\beta(x)} f(x,y) \mathrm{d} y = \int _{\alpha(x)}^{\beta(x)} f_x(x,y) \mathrm{d} y + f[x, \beta(x)]\beta '(x) - f[x, \alpha(x)]\alpha '(x)$$

## 6.4. 重积分的应用

### 6.4.1. 曲面的面积
* 曲面的面积 $$A=\iint _D \sqrt{1 + f_x^2(x,y) + f_y^2(x,y)} \mathrm{d} \sigma = \iint _D \sqrt{1 + f_x^2(x,y) + f_y^2(x,y)} \mathrm{d} x \mathrm{d} y$$
* 利用曲面的参数方程求曲面面积:
    曲面 $S$: $$\begin{cases} x = x(u,v) \\ y = y(u,v) \\ z = z(u,v) \end{cases}, \ (u,v) \in D$$ 的面积为 $$A= \iint _D \sqrt{EG-F^2} \mathrm{d} u \mathrm{d} v$$ 其中, $$\begin{aligned} E &= x_u^2 + y_u^2 + z_u^2 \\ F &= x_u \cdot x_v + y_u \cdot y_v + z_u \cdot z_v \\ G &= x_v^2 + y_v^2 + z_v^2\end{aligned}$$

### 6.4.2. 质心
* 质心坐标: $$\bar x = \dfrac {\iint _D x \mu(x,y) \mathrm{d} \sigma}{\iint _D \mu(x,y) \mathrm{d} \sigma}, \bar y = \dfrac {\iint _D y \mu(x,y) \mathrm{d} \sigma}{\iint _D \mu(x,y) \mathrm{d} \sigma}$$

# 7. 曲线积分与曲面积分

## 7.1. 对弧长的曲线积分
* **定义**: 设 $L$ 为 $xOy$ 面内的一条光滑曲线弧, 函数 $f(x,y)$ 在 $L$ 上有界, 在 $L$ 上任意插入一点列 $M_1, M_2, \cdots, M_{n-1}$ 把 $L$ 分成 $n$ 个小段, 设第 $i$ 个小段的长度为 $\Delta s_i$, 又 $(\xi_i, \eta _i)$ 为第 $i$ 个小段上任意取定的一点, 作乘积 $f(\xi _i, \eta _i)\Delta s_i \ (i=1,2,\cdots, n)$, 并作和 $\displaystyle \sum _{i=1}^{n} f(\xi _i, \eta _i)\Delta s_i$, 如果当各小弧段的长度的最大值 $\lambda \to 0$ 时, 这和的极限总存在, 则称此极限为函数 $f(x,y)$ 在曲线弧 $L$ 上**对弧长的曲线积分**或**第一类曲线积分**, 记作 $\displaystyle \int _L f(x,y) \mathrm{d} s$, 即$$\int _L f(x,y) \mathrm{d} s = \lim _{\lambda \to 0} \sum _{i=1}^{n}f(\xi _i, \eta _i) \Delta s_i$$ 其中 $f(x,y)$ 叫做**被积函数**, $L$ 叫做**积分弧段**
* **性质**:
    1. 设 $\alpha, \beta$ 为常数, 则 $\displaystyle \int _L [\alpha f(x,y) + \beta g(x,y)] \mathrm{d} s = \alpha \int _L f(x,y) \mathrm{d} s + \beta \int _L g(x,y) \mathrm{d} s$
    1. 设积分弧段 $L$ 可分成两段光滑曲线 $L_1, L_2$, 则 $\displaystyle \int _L f(x,y) \mathrm{d} s = \int _{L_1} f(x,y) \mathrm{d} s + \int _{L_2} f(x,y) \mathrm{d} s$
    1. 设在 $L$ 上 $f(x,y) \leqslant g(x,y)$, 则 $\displaystyle \int _L f(x,y) \mathrm{d} s \leqslant \int _L g(x,y) \mathrm{d} s$
        特别地, 有 $\displaystyle \bigg| \int _L f(x,y) \mathrm{d} s \bigg| \leqslant \int _L \bigg|f(x,y)\bigg| \mathrm{d} s$
* **计算**: 设 $f(x,y)$ 在曲线弧 $L$ 上有定义且连续, 如果 $L$ 的参数方程为 $\begin{cases} x=\varphi (t) \\ y = \psi(t) \end{cases}\ (\alpha \leqslant t \leqslant \beta)$, 其中 $\varphi(t), \psi(t)$ 在 $[\alpha, \beta]$ 上具有一阶连续导数, 且 ${\varphi'\ }^2 (t) + {\psi'\ }^2 \ne 0$, 则曲线积分 $\displaystyle \int _L f(x,y) \mathrm{d} s$ 存在, 且 $$\int _L f(x,y) \mathrm{d} s = \int _{\alpha}^{\beta} f \Big[\varphi(t), \psi(t) \Big] \sqrt {{\varphi'\ }^2 (t) + {\psi'\ }^2 (t)} \ \mathrm{d}t \quad (\alpha < \beta)$$
    * 如果曲线 $L$ 由方程 $y=\varphi(t) \ (x_0 \leqslant x \leqslant X)$ 给出, 则 $$\int _L f(x,y) \mathrm{d} s = \int _{x_0}^{X} f \Big[x, \varphi(x) \Big] \sqrt {1 + {\varphi'\ }^2 (x)} \ \mathrm{d}t \quad (x_0 < X)$$
    * 推广到空间曲线弧 $\varGamma$ 由参数方程 $x=\varphi(t), y=\psi(t), z=\omega(t) \ (\alpha \leqslant t \leqslant \beta)$ 给出的情形时有 $$\int _\varGamma f(x,y,z) \mathrm{d} s = \int _{\alpha}^{\beta} f \Big[\varphi(t), \psi(t), \omega(t) \Big] \sqrt{{\varphi'\ }^2 (t) + {\psi'\ }^2  (t) + {\omega ' \ }^2 (t)} \ \mathrm{d} t$$

## 7.2. 对坐标的曲线积分
* **定义**: 设 $L$ 为 $xOy$ 面内从点 $A$ 到点 $B$ 的一条有向光滑曲线弧, 函数 $P(x,y), Q(x,y)$ 在 $L$ 上有界, 在 $L$ 上沿 $L$ 的方向任意插入一点列 $M_1(x_1, y_1), M_2(x_2,y_2), \cdots, M_{n-1}(x_{n-1}, y_{n-1})$, 把 $L$ 分成 $n$ 个有向小弧段 $\overgroup {M_{i-1}M_i} \ (i=1,2,\cdots, n; M_0 = A, M_n=B)$, 设 $\Delta x_i = x_i - x_{i-1}, \Delta y_i = y_i - y_{i-1}$, 点 $(\xi _i, \eta _i)$ 为 $\overgroup {M _{i-1}M _i}$ 上任意取定的点, 如果当各个小弧段的最大值 $\lambda \to 0$ 时, $\displaystyle \sum _{i=1}^{n} P(\xi _i, \eta _i) \Delta x_i$ 的极限总存在, 则称此极限为函数 $P(x,y)$ 在有向曲线弧 $L$ 上**对坐标 $x$ 的曲线积分**, 记作 $\displaystyle \int _L P(x,y) \mathrm{d} x$, 类似地, 如果 $\displaystyle \lim _{\lambda \to 0}\sum _{i=1}^{n} Q(\xi _i, \eta _i) \Delta y_i$ 总存在, 则称此极限为函数 $Q(x,y)$ 在有向曲线弧 $L$ 上**对坐标 $y$ 的曲线积分**, 记作 $\displaystyle \int _L Q(x,y) \mathrm{d} y$, 即 $$\begin{aligned}
        \displaystyle \int _L P(x,y) \mathrm{d} x &= \lim _{\lambda \to 0}\sum _{i=1}^{n} P(\xi _i, \eta _i) \Delta x_i \\
        \displaystyle \int _L Q(x,y) \mathrm{d} y &= \lim _{\lambda \to 0}\sum _{i=1}^{n} Q(\xi _i, \eta _i) \Delta y_i \\
        \end{aligned}$$ 其中 $P(x,y), Q(x,y)$ 叫做**被积函数**, $L$ 叫做**被积弧段**
    以上两个积分也叫做**第二类曲线积分**
* **性质**
    1. 设 $\alpha, \beta$ 为常数, 则 $\displaystyle \int _L [\alpha \bm {F_1}(x,y) + \beta \bm {F_2}(x,y)] \cdot \mathrm{d} \bm r = \alpha \int _L \bm {F_1}(x,y) \cdot \mathrm{d} \bm r + \beta \int _L \bm {F_2}(x,y) \cdot \mathrm{d} \bm r$
    1. 设有向曲线弧 $L$ 可分成两段光滑的有向曲线弧 $L_1, L_2$, 则 $$\int _L \bm F(x,y) \cdot \mathrm{d} \bm r = \int _{L_1} \bm F(x,y) \cdot \mathrm{d} \bm r + \int _{L_2} \bm F(x,y) \cdot \mathrm{d} \bm r$$
    1. 设 $L$ 时有向光滑曲线弧, $L^-$ 是 $L$ 的反向曲线弧, 则 $\displaystyle \int _{L ^-} \bm F(x,y) \cdot \mathrm{d} \bm r = -\int _L \bm F(x,y) \cdot \mathrm{d} \bm r$
* **计算**: 设 $P(x,y), Q(x,y)$ 在有向曲线弧 $L$ 上有定义且连续, $L$ 的参数方程为 $\begin{cases} x=\varphi (t) \\ y = \psi(t) \end{cases}$, 当参数 $t$ 单调地由 $\alpha$ 变到 $\beta$ 时, 点 $M(x,y)$ 从 $L$ 的起点 $A$ 沿 $L$ 运动到终点 $B$, $\varphi(t), \psi(t)$ 在以 $\alpha$ 及 $\beta$ 为端点的闭区间上具有一阶连续导数, 且  ${\varphi'\ }^2 (t) + {\psi'\ }^2 \ne 0$, 则曲线积分 $\displaystyle \int _L P(x,y) \mathrm{d}x + Q(x,y) \mathrm{d} y$ 存在, 且 $$\int _L P(x,y) \mathrm{d}x + Q(x,y) \mathrm{d} y = \int _{\alpha}^{\beta} \bigg \{ P \Big[\varphi(t), \psi(t) \Big] \varphi ' (t) + Q \Big[\varphi(t), \psi(t) \Big]\psi' (t) \bigg \} \mathrm{d} t$$
    * 当 $L$ 由 $y=\psi(x)$ 给出时,  $$\int _L P(x,y) \mathrm{d}x + Q(x,y) \mathrm{d} y = \int _a^b \bigg \{ P \Big[x, \psi(x) \Big] + Q \Big[x, \psi(x) \Big]\psi ' (x) \bigg \} \mathrm{d} x$$
    * 推广到空间曲线弧 $\varGamma$ 由参数方程 $x=\varphi(t), y=\psi(t), z=\omega(t)$ 给出的情形时有 $$\int _\varGamma P(x,y,z) \mathrm{d}x + Q(x,y,z) \mathrm{d}y + R(x,y,z) \mathrm{d}z$$ $$ =\int _{\alpha}^{\beta} \bigg \{ P \Big[\varphi(t), \psi(t), \omega(t) \Big] \varphi'(t) + Q \Big[\varphi(t), \psi(t), \omega(t) \Big] \psi'(t) + R \Big[\varphi(t), \psi(t), \omega(t) \Big] \omega'(t) \bigg \} \mathrm{d} t$$
* **两类曲线积分之间的联系**
    * 平面曲线 $L$ 上两类曲线积分之间的的联系: $$\int _L P \mathrm{d}x + Q \mathrm{d}y = \int _L (P \cos \alpha + Q \cos \beta) \mathrm{d}s$$ 其中 $\alpha(x,y), \beta(x,y)$ 为有向曲线弧 $L$ 在点 $(x,y)$ 处的切向量的方向角
    * 空间曲线 $\varGamma$ 上两类曲线积分之间的的联系: $$\int _L P \mathrm{d}x + Q \mathrm{d}y + R \mathrm{d}z = \int _L (P \cos \alpha + Q \cos \beta + R \cos \gamma) \mathrm{d}s$$ 其中 $\alpha(x,y,z), \beta(x,y,z), \gamma(x,y,z)$ 为有向曲线弧 $\varGamma$ 在点 $(x,y,z)$ 处的切向量的方向角
    * 两类曲线积分之间的联系也可用向量的形式表达: $$\int _\varGamma \bm A \cdot \mathrm{d} \bm r = \int _\varGamma \bm A \cdot \bm \tau \mathrm{d}s = \int _\varGamma \bm {A _\tau} \mathrm{d}s$$ 其中 $\bm A = (P,Q,R), \bm \tau = (\cos \alpha, \cos \beta, \cos \gamma)$ 为有向曲线弧 $\varGamma$ 在点 $(x,y,z)$ 处的单位切向量, $\mathrm{d} \bm r = \bm \tau \mathrm{d}s = (\mathrm{d}x, \mathrm{d}y, \mathrm{d}z)$ 称为**有向曲线元**, $\bm {A _\tau}$ 为向量 $\bm A$ 在向量 $\bm \tau$ 上的投影

## 7.3. 格林公式及其应用
* **定义**:
    * **连通区域**:
        * **单连通区域**: 闭合面, 中间没洞
        * **复连通区域 (多连通区域)**: 闭合面, 但中间有洞
* **定理**:
    * **格林公式**: 设闭区域 $D$ 由分段光滑的曲线 $L$ 围观, 函数 $P(x,y), Q(x,y)$ 在 $D$ 上具有一阶连续偏导数, 则有 $$\iint _D \bigg(\dfrac {\partial Q}{\partial x} - \dfrac {\partial P}{\partial y} \bigg) \mathrm{d}x \mathrm{d}y = \oint _L P \mathrm{d}x + Q \mathrm{d}y$$ 其中 $L$ 是 $D$ 的取正向 ($D$ 面在线 $L$ 左侧) 的边界曲线
    * 设区域 $G$ 是一个单连通域, 函数 $P(x,y), Q(x,y)$ 在 $G$ 上具有一阶连续偏导数, 则曲线积分 $\displaystyle \int _L P \mathrm{d}x + Q \mathrm{d}y$ 在 $G$ 内与路径无关 (或沿 $G$ 内任意闭曲线的曲线积分为0) $\iff$ $\dfrac {\partial P}{\partial y} = \dfrac {\partial Q}{\partial x}$ 在 $G$ 内恒成立
    * 设区域 $G$ 是一个单连通域, 函数 $P(x,y), Q(x,y)$ 在 $G$ 上具有一阶连续偏导数, 则曲线积分 $P(x,y) \mathrm{d}x + Q(x,y) \mathrm{d}y$ 在 $G$ 内为某一函数 $\mu(x,y)$ 的全微分 $\iff$ $\dfrac {\partial P}{\partial y} = \dfrac {\partial Q}{\partial x}$ 在 $G$ 内恒成立
        * **推论**: 设区域 $G$ 是一个单连通域, 函数 $P(x,y), Q(x,y)$ 在 $G$ 上具有一阶连续偏导数, 则曲线积分 $\displaystyle \int _L P \mathrm{d}x + Q \mathrm{d}y$ 在 $G$ 内与路径无关 (或沿 $G$ 内任意闭曲线的曲线积分为0) $\iff$ $G$ 内存在函数 $\mu(x,y)$, 使 $\mathrm{d} \mu = P \mathrm{d}x + Q \mathrm{d}y$
    * **曲线积分的基本定理**: 设 $\bm F (x,y) = P(x,y) \bm i + Q(x,y) \bm j$ 是区域 $G$ 内的一个向量场, $P(x,y), Q(x,y)$ 都在 $G$ 内连续, 且存在一个数量函数 $f(x,y)$, 使得 $\bm F = \nabla f$, 则曲线积分 $\displaystyle \int _L \bm F \cdot \mathrm{d} \bm r$  $G$ 内与路径无关, 且 $$\int _L \bm F \cdot \mathrm{d} \bm r = f(B) - f(A)$$ 其中 $L$ 是位于 $G$ 内起点为 $A$, 终点为 $B$ 的任一分段光滑曲线

## 7.4. 对面积的曲面积分
* **定义**: 设曲面 $\Sigma$ 是光滑的, 函数 $f(x,y,z)$ 在 $\Sigma$ 上有界, 把 $\Sigma$ 任意分成 $n$ 个小块 $\Delta S_i$ ($\Delta S_i$ 同时也代表第 $i$ 小块曲面的面积), 设 $(\xi _i, \eta _i, \zeta _i)$ 是 $\Delta S_i$ 上任意取定的一点, 作乘积 $f(\xi _i, \eta _i, \zeta _i) \Delta S_i (i=1,2,\cdots, n)$, 并作和 $\displaystyle \sum _{i=1}^n f(\xi _i, \eta _i, \zeta _i) \Delta S_i$, 如果当各小块曲面的直径的最大值 $\lambda \to 0$ 时, 这和的极限总存在, 则称此极限为函数 $f(x,y,z)$ 在曲面 $\Sigma$ 上**对面积的曲面积分** 或 **第一类曲面积分**, 记作 $\displaystyle \iint _\Sigma f(x,y,z) \mathrm{d} S$, 即 $$\iint _\Sigma f(x,y,z) \mathrm{d} S = \lim _{\lambda \to 0} \sum _{i=1}^n f(\xi _i, \eta _i, \zeta _i) \Delta S_i$$ 其中 $f(x,y,z)$ 叫做**被积函数**, $\Sigma$ 叫做**积分曲面**
* **性质**: 与曲线积分类似
* **计算**: $\displaystyle \iint _\Sigma f(x,y,z) \mathrm{d} S = \iint _{D _{xy}} f \Big [x,y,z(x,y) \Big] \sqrt {1 + z_x^2(x,y) + z_y^2(x,y)} \ \mathrm{d}x \mathrm{d}y$

## 7.5. 对坐标的曲面积分
* 概念: 有向曲面, 投影
* **定义**: 设 $\Sigma$ 为光滑的有向曲面, 函数 $R(x,y,z)$ (是向量场 $\bm A(x,y,z) = P(x,y,z) \bm i + Q(x,y,z) \bm j + R(x,y,z) \bm k$ 的一个向量函数) 在 $\Sigma$ 上有界, 把 $\Sigma$ 任意分成 $n$ 块小曲面 $\Delta S_i$ ($\Delta S_i$ 同时又表示第 $i$ 块小曲面的面积), $\Delta S_i$ 在 $xOy$ 面上的投影为 $(\Delta S_i)_{xy}$, $(\xi _i, \eta _i, \zeta _i)$ 是 $\Delta S_i$ 上任意取定的一点, 如果当各小块曲面的直径的最大值 $\lambda \to 0$ 时, $\displaystyle \lim _{\lambda \to 0} \sum _{i=1}^n R(\xi _i, \eta _i, \zeta _i) (\Delta S_i) _{xy}$ 总存在, 则称此极限为函数 $R(x,y,z)$ 在有向曲面 $\Sigma$ 上**对坐标 $x,y$ 的曲面积分**, 记作 $\displaystyle \iint _\Sigma R(x,y,z) \mathrm{d}x \mathrm{d}y$, 即 $$\iint _\Sigma R(x,y,z) \mathrm{d}x \mathrm{d}y = \lim _{\lambda \to 0} \sum _{i=1}^n R(\xi _i, \eta _i, \zeta _i) (\Delta S_i) _{xy}$$ 其中 $R(x,y,z)$ 叫做**被积函数**, $\Sigma$ 叫做**积分曲面**
    类似地, 可以定义函数 $P(x,y,z)$ 在有向曲面 $\Sigma$ 上**对坐标 $y,z$ 的曲面积分** $\displaystyle \iint _\Sigma P(x,y,z) \mathrm{d}y \mathrm{d}z$ 及函数 $Q(x,y,z)$ 在有向曲面 $\Sigma$ 上**对坐标 $z,x$ 的曲面积分** $\displaystyle \iint _\Sigma Q(x,y,z) \mathrm{d}z \mathrm{d}x$ 分别为 $$\begin{aligned}
            \displaystyle \iint _\Sigma P(x,y,z) \mathrm{d}y \mathrm{d}z &= \lim _{\lambda \to 0} \sum _{i=1}^n P(\xi _i, \eta _i, \zeta _i) (\Delta S_i) _{yz} \\
            \displaystyle \iint _\Sigma Q(x,y,z) \mathrm{d}z \mathrm{d}x &= \lim _{\lambda \to 0} \sum _{i=1}^n Q(\xi _i, \eta _i, \zeta _i) (\Delta S_i) _{zx}
        \end{aligned}$$
    以上三个曲面积分也称为**第二类曲面积分**
* **性质**: 与曲线积分类似
* **计算**: $$\begin{aligned}
        \displaystyle \iint _\Sigma R(x,y,z) \mathrm{d}x \mathrm{d}y = \pm \iint _{D _{xy}} R \Big [x,y,z(x,y) \Big] \mathrm{d}x \mathrm{d}y \\
        \displaystyle \iint _\Sigma P(x,y,z) \mathrm{d}y \mathrm{d}z = \pm \iint _{D _{yz}} P \Big [x(y,z),y,z \Big] \mathrm{d}y \mathrm{d}z \\
        \displaystyle \iint _\Sigma Q(x,y,z) \mathrm{d}z \mathrm{d}x = \pm \iint _{D _{zx}} Q \Big [x,y(z,x),z \Big] \mathrm{d}z \mathrm{d}x \\
    \end{aligned}$$
* **两类曲面积分之间的联系**: $\displaystyle \iint _\Sigma P \mathrm{d}y \mathrm{d}z + Q \mathrm{d}z \mathrm{d}x + R \mathrm{d}x \mathrm{d}y = \iint _\Sigma (P \cos \alpha + Q \cos \beta + R \cos \gamma) \mathrm{d}S$
    其中 $\cos \alpha, \cos \beta, \cos \gamma$ 是有向曲面 $\Sigma$ 在点 $(x,y,z)$ 处的法向量的方向余弦
    * 可用向量的形式表达: $$\iint _\Sigma \bm A \cdot \mathrm{d} \bm S = \iint _\Sigma \bm A \cdot \bm n \mathrm{d}S = \iint _\Sigma \bm {A _n} \mathrm{d}S$$ 其中 $\bm A = (P,Q,R), \bm n = (\cos \alpha, \cos \beta, \cos \gamma)$ 为有向曲面 $\Sigma$ 在点 $(x,y,z)$ 处的单位法向量, $\mathrm{d} \bm S = \bm n \mathrm{d}S = (\mathrm{d}y \mathrm{d}z, \mathrm{d}z \mathrm{d}x, \mathrm{d}x \mathrm{d}y)$ 称为**有向曲面元**, $\bm {A _n}$ 为向量 $\bm A$ 在向量 $\bm n$ 上的投影

## 7.6. 高斯公式, 通量和散度
* **高斯公式**: 设空间闭区域 $\Omega$ 是由分片光滑的闭曲面 $\Sigma$ 所围成, 函数 $P(x,y,z), Q(x,y,z), R(x,y,z)$ 在 $\Omega$ 上具有一阶连续偏导数, 则有 $$\iiint _\Omega \bigg(\dfrac {\partial P}{\partial x} + \dfrac {\partial Q}{\partial y} + \dfrac {\partial R}{\partial z}\bigg) \mathrm{d}v = \oiint _\Sigma P \mathrm{d}y \mathrm{d}z + Q \mathrm{d}z \mathrm{d}x + R \mathrm{d}x \mathrm{d}y$$ 或 $$\iiint _\Omega \bigg(\dfrac {\partial P}{\partial x} + \dfrac {\partial Q}{\partial y} + \dfrac {\partial R}{\partial z}\bigg) \mathrm{d}v = \oiint _\Sigma (P \cos \alpha + Q \cos \beta + R \cos \gamma) \mathrm{d}S$$ 这里 $\Sigma$ 是 $\Omega$ 的整个边界曲面的外侧, $\cos \alpha, \cos \beta, \cos \gamma$ 是 $\Sigma$ 在点 $(x,y,z)$ 处的法向量的方向余弦
* **定义**:
    * **空间二维单连通区域**: 对于空间区域 $G$, 如果 $G$ 内任一闭曲面所围成的区域全属于 $G$, 则称 $G$ 是空间二维单连通区域
    * **空间一维单连通区域**: 如果 $G$ 内任一闭曲线总可以张成一片完全属于 $G$ 的曲面, 则称 $G$ 为空间一维单连通区域
        > 例如: 球面既是...二维...又是...一维...; 环面是...二维...; 同心球面之间的区域是...一维...
    * **通量**: 设有向量场 $\bm A(x,y,z) = P(x,y,z) \bm i + Q(x,y,z) \bm j + R(x,y,z) \bm k$, 其中函数 $P,Q,R$ 均具有一阶连续偏导数, $\Sigma$ 是场内的一片有向曲面, $\bm n$ 是 $\Sigma$ 在 $(x,y,z)$ 处的单位法向量, 则积分 $$\iint _\Sigma \bm A \cdot \bm n \mathrm{d}S$$ 称为向量场 $\bm A$ 通过曲面 $\Sigma$ 向着指定侧的**通量** (或**流量**), 由两类曲面积分的关系, 通量又可表达为 $$\iint _\Sigma \bm A \cdot \bm n \mathrm{d}S = \iint _\Sigma \bm A \cdot \mathrm{d} \bm S = \iint _\Sigma P \mathrm{d}y \mathrm{d}z + Q \mathrm{d}z \mathrm{d}x + R \mathrm{d}x \mathrm{d}y$$
    * **散度**: 对于一般的向量场 $\bm A(x,y,z) = P(x,y,z) \bm i + Q(x,y,z) \bm j + R(x,y,z) \bm k$, $\dfrac {\partial P}{\partial x} + \dfrac {\partial Q}{\partial y} + \dfrac {\partial R}{\partial z}$ 叫做向量场 $\bm A$ 的散度, 记作 $\mathrm{div} \bm A$, 即 $$\mathrm{div} \bm A = \dfrac {\partial P}{\partial x} + \dfrac {\partial Q}{\partial y} + \dfrac {\partial R}{\partial z}$$ 利用向量微分算子 $\nabla$, $\bm A$ 的散度 $\mathrm {div} \bm A$ 也可以表达为 $$\mathrm{div} \bm A = \nabla \cdot \bm A$$
        如果向量场 $\bm A$ 的散度 $\mathrm{div} \bm A$ 处处为0, 则称向量场 $\bm A$ 为**无源场**
* **定理**:
    1. 设 $G$ 是空间二维单连通区域, $P(x,y,z), Q(x,y,z), R(x,y,z)$ 在 $G$ 内具有一阶连续偏导数, 则曲面积分 $\displaystyle \iint _\Sigma P \mathrm{d}y \mathrm{d}z + Q \mathrm{d}z \mathrm{d}x + R \mathrm{d}x \mathrm{d}y$ 在 $G$ 内与所取曲面 $\Sigma$ 无关而只取决与 $\Sigma$ 的边界曲线 (或沿 $G$ 内任一闭曲面的曲面积分为0) $\iff$ $\dfrac {\partial P}{\partial x} + \dfrac {\partial Q}{\partial y} + \dfrac {\partial R}{\partial z} = 0$ 在 $G$ 内恒成立
* **格林第一公式**: 设函数 $u(x,y,z), v(x,y,z)$ 在闭区域 $\Omega$ 上具有一阶及二阶连续偏导数, $$\iiint _\Omega u \Delta v \mathrm{d}x \mathrm{d}y \mathrm{d}z = \oiint _\Sigma u \dfrac {\partial v}{\partial n} \mathrm{d}S - \iiint _\Omega \bigg( \dfrac{\partial u}{\partial x} \dfrac{\partial v}{\partial x} + \dfrac{\partial u}{\partial y} \dfrac{\partial v}{\partial y} + \dfrac{\partial u}{\partial z} \dfrac{\partial v}{\partial z} \bigg) \mathrm{d}x \mathrm{d}y \mathrm{d}z$$ 其中 $\Sigma$ 是闭区域 $\Omega$ 的整个边界曲面, $\dfrac {\partial v}{\partial n}$ 为函数 $v(x,y,z)$ 沿 $\Sigma$ 的外法线方向的方向导数, 符号 $\Delta = \dfrac{\partial ^2}{\partial x^2} + \dfrac{\partial ^2}{\partial y^2} + \dfrac{\partial ^2}{\partial z^2}$, 称为**拉普拉斯 (Laplace) 算子**

## 7.7. 斯托克斯公式, 环流量与旋度
* **斯托克斯公式**: 设 $\varGamma$ 为分段光滑的空间有向闭曲线, $\Sigma$ 是以 $\varGamma$ 为边界的分片光滑的有向曲面, $\varGamma$ 的正向与 $\Sigma$ 的侧符合右手规则, 函数 $P(x,y,z), Q(x,y,z), R(x,y,z)$ 在曲面 $\Sigma$ (连同边界 $\varGamma$ ) 上具有一阶连续偏导数, 则有 $$\iint _\Sigma \bigg(\dfrac {\partial R}{\partial y} - \dfrac {\partial Q}{\partial z} \bigg) \mathrm{d}y \mathrm{d}z + \bigg(\dfrac {\partial P}{\partial z} - \dfrac {\partial R}{\partial x} \bigg) \mathrm{d}z \mathrm{d}x + \bigg(\dfrac {\partial Q}{\partial x} - \dfrac {\partial P}{\partial y} \bigg) \mathrm{d}x \mathrm{d}y = \oint _\varGamma P \mathrm{d}x + Q \mathrm{d}y + R \mathrm{d}z$$
    * 利用两类曲面积分间的联系, 可得斯托克斯公式的另一种形式 $$\iint _\Sigma \begin{vmatrix}
        \cos \alpha & \cos \beta & \cos \gamma \\ \\
        \dfrac{\partial}{\partial x} & \dfrac{\partial}{\partial y} & \dfrac{\partial}{\partial z} \\ \\
        P & Q & R \end{vmatrix} \mathrm{d} S = \oint _\varGamma P \mathrm{d}x + Q \mathrm{d}y + R \mathrm{d}z$$ 其中 $\bm n = (\cos \alpha, \cos \beta, \cos \gamma)$ 为有向曲面 $\Sigma$ 在点 $(x,y,z)$ 处的单位法向量
* **定义**:
    * **环流量**: 设有向量场 $\bm A(x,y,z) = P(x,y,z) \bm i + Q(x,y,z) \bm j + R(x,y,z) \bm k$, 其中函数 $P,Q,R$ 均连续, $\varGamma$ 是 $\bm A$ 的定义域内的一条分段光滑的有向闭曲线, $\bm \tau$ 是 $\varGamma$ 在 $(x,y,z)$ 处的单位切向量, 则积分 $$\oint _\varGamma \bm A \cdot \bm \tau \mathrm{d}s$$ 称为向量场 $\bm A$ 沿有向闭曲线 $\varGamma$ 的**环流量**, 由两类曲线积分的关系, 环流量又可表达为 $$\oint _\varGamma \bm A \cdot \bm \tau \mathrm{d}s = \oint _\varGamma \bm A \cdot \mathrm{d} \bm r = \oint _\varGamma P \mathrm{d}x + Q \mathrm{d}y + R \mathrm{d}z$$
    * **旋度**: 对有一向量场 $\bm A(x,y,z) = P(x,y,z) \bm i + Q(x,y,z) \bm j + R(x,y,z) \bm k$, 其中 $P,Q,R$ 均具有一阶连续偏导数, 则向量 $\bigg(\dfrac {\partial R}{\partial y} - \dfrac {\partial Q}{\partial z} \bigg) \bm i + \bigg(\dfrac {\partial P}{\partial z} - \dfrac {\partial R}{\partial x} \bigg) \bm j + \bigg(\dfrac {\partial Q}{\partial x} - \dfrac {\partial P}{\partial y} \bigg) \bm k$ 称为向量场 $\bm A$ 的**旋度**, 记作 $\mathrm{rot} \bm A$, 即 $$\mathrm{rot} \bm A = \bigg(\dfrac {\partial R}{\partial y} - \dfrac {\partial Q}{\partial z} \bigg) \bm i + \bigg(\dfrac {\partial P}{\partial z} - \dfrac {\partial R}{\partial x} \bigg) \bm j + \bigg(\dfrac {\partial Q}{\partial x} - \dfrac {\partial P}{\partial y} \bigg) \bm k$$
        * 利用向量微分算子 $\nabla$, $\bm A$ 的旋度  $\mathrm {rot} \bm A$ 可表达为 $\nabla \times \bm A$, 即 $$\mathrm {rot} \bm A = \nabla \times \bm A = \begin{vmatrix} \bm i & \bm j & \bm k \\ \\
            \dfrac{\partial}{\partial x} & \dfrac{\partial}{\partial y} & \dfrac{\partial}{\partial z} \\ \\
            P & Q & R \end{vmatrix}$$
        * 如果向量场 $\bm A$ 的散度 $\mathrm{rot} \bm A$ 处处为0, 则称向量场 $\bm A$ 为**无旋场**, 而一个无源、无旋的向量场称为**调和场**
* **定理**:
    1. 设空间区域 $G$ 是一维单连通区域, 函数 $P(x,y,z), Q(x,y,z), R(x,y,z)$ 在 $G$ 内具有一阶连续偏导数, 则空间曲线积分 $\displaystyle \oint _\varGamma P \mathrm{d}x + Q \mathrm{d}y + R \mathrm{d}z$ 在 $G$ 内与路径无关 (或沿 $G$ 内任意闭曲线的曲线积分为0) $\iff$ $$\dfrac {\partial P}{\partial y} = \dfrac {\partial Q}{\partial x}, \quad
        \dfrac {\partial Q}{\partial z} = \dfrac {\partial R}{\partial y}, \quad
        \dfrac {\partial R}{\partial x} = \dfrac {\partial P}{\partial z}$$ 在 $G$ 内恒成立
    1. 设区域 $G$ 是空间一维单连通区域, 函数 $P(x,y,z), Q(x,y,z), R(x,y,z)$ 在 $G$ 内具有一阶连续偏导数, 则表达式 $\displaystyle P \mathrm{d}x + Q \mathrm{d}y + R \mathrm{d}z$ 在 $G$ 内称为某一函数 $u(x,y,z)$ 的全微分 $\iff$ $$\dfrac {\partial P}{\partial y} = \dfrac {\partial Q}{\partial x}, \quad
        \dfrac {\partial Q}{\partial z} = \dfrac {\partial R}{\partial y}, \quad
        \dfrac {\partial R}{\partial x} = \dfrac {\partial P}{\partial z}$$ 在 $G$ 内恒成立; 此时, 这函数 (不计一常数之差) 可用下式求出 $$u(x,y,z) = \int _{(x_0, y_0, z_0)}^{(x,y,z)} P \mathrm{d}x + Q \mathrm{d}y + R \mathrm{d}z$$ 或用定积分表示为 $$u(x,y,z) = \int _{x_0}^x P(x,y,z) \mathrm{d}x + \int _{y_0}^y Q(x,y,z) \mathrm{d}y + \int _{z_0}^z R(x,y,z) \mathrm{d}z$$ 其中 $M_0(x_0, y_0, z_0)$ 为 $G$ 内某一定点, 点 $M(x,y,z) \in G$

# 8. 无穷级数

## 8.1. 常数项级数
* **定义**: 如果级数 $\displaystyle \sum _{n=1} u_n$ 的部分和数列 $\{s_n\}$ 有极限 $s$, 即 $\displaystyle \lim _{n \to \infin} s_n = s$, 则称该无穷级数**收敛**, 这时极限 $s$ 叫做这级数的 **和**, 并写作 $s = u_1 + u_2 + \cdots + u_n + \cdots$; 如果没有极限, 则称该无穷级数**发散**
* **性质**:
    1. 如果级数 $\displaystyle \sum _{n=1} u_n$ 收敛于和 $s$, 则级数 $\displaystyle \sum _{n=1} k u_n$ 也收敛, 且其和为 $k s$
    1. 如果级数 $\displaystyle \sum _{n=1} u_n, \sum _{n=1} v_n$ 分别收敛于和 $s, \delta$, 则级数 $\displaystyle \sum _{n=1} u_n \pm v_n$ 也收敛, 且其和为 $s \pm \delta$
    1. 在级数中去掉、加上或改变有限项, 不会改变级数的收敛性
    1. 如果级数 $\displaystyle \sum _{n=1} u_n$ 收敛, 则对这级数的项任意加括号后所成的级数 $(u_1 + \cdots + u_{n_1}) + (u_{n_1 + 1} + \cdots + u_{n_2}) + \cdots + (u _{n _{k-1}+1} + \cdots + u _{n_k}) + \cdots$ 仍收敛, 且其和不变
    1. **级数收敛的必要条件** 如果级数 $\displaystyle \sum _{n=1} u_n$ 收敛, 则它的一般项 $u_n$ 趋于0, 即 $\displaystyle \lim _{n \to \infin} u_n = 0$
* **柯西审敛原理**: 级数 $\displaystyle \sum _{n=1} u_n$ 收敛的充分必要条件为: 对于任意给定的正数 $\varepsilon$, 总存在正整数 $N$, 使得当 $n > N$ 时, 对于任意正整数 $p$, 都有 $|u _{n+1} + u _{n+2} + \cdots + u _{n+p}| < \varepsilon$ 成立
* **常数项级数的审敛法**:
    1. 正项级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 收敛 $\iff$ 它的部分和数列 $\{s_n\}$ 有界
    1. **比较审敛法**: 设 $\displaystyle \sum _{n=1}^{\infin}u_n$ 和 $\displaystyle \sum _{n=1}^{\infin}v_n$ 都是正项级数, 且 $u_n \leqslant v_n (n=1,2,\cdots)$, 若级数 $\displaystyle \sum _{n=1}^{\infin}v_n$ 收敛, 则级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 收敛; 反之, 若级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 发散, 则级数 $\displaystyle \sum _{n=1}^{\infin}v_n$ 发散
        * 设 $\displaystyle \sum _{n=1}^{\infin}u_n$ 和 $\displaystyle \sum _{n=1}^{\infin}v_n$ 都是正项级数, 若级数 $\displaystyle \sum _{n=1}^{\infin}v_n$ 收敛, 且存在正整数 $N$, 使得当 $n \geqslant N$ 时有 $u_n \leqslant kv_n (k > 0)$ 成立, 则级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 收敛; 如果级数 $\displaystyle \sum _{n=1}^{\infin}v_n$ 发散, 且当 $n \geqslant N$ 时有 $u_n \geqslant kv_n (k > 0)$ 成立, 则级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 发散
    1. **比较审敛法的极限形式**: 设 $\displaystyle \sum _{n=1}^{\infin}u_n$ 和 $\displaystyle \sum _{n=1}^{\infin}v_n$ 都是正项级数,
        1. 如果 $\displaystyle \lim _{n \to \infin} \dfrac{u_n}{v_n} = l (0 \leqslant l < +\infin)$, 且级数 $\displaystyle \sum _{n=1}^{\infin}v_n$ 收敛, 则级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 收敛;
        1. 如果 $\displaystyle \lim _{n \to \infin} \dfrac{u_n}{v_n} = l > 0$ 或 $\displaystyle \lim _{n \to \infin} \dfrac{u_n}{v_n} = +\infin$, 且级数 $\displaystyle \sum _{n=1}^{\infin}v_n$ 发散, 则级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 发散
    1. **比较审敛法, 达朗贝尔 (d'Alembert) 判别法**: 设 $\displaystyle \sum _{n=1}^{\infin}u_n$ 是正项级数, 如果 $\displaystyle \lim _{n \to \infin} \dfrac{u_{n+1}}{u_n} = \rho$, 则当 $\rho < 1$ 时级数收敛; $\rho > 1$ (或 $\displaystyle \lim _{n \to \infin} \dfrac{u_{n+1}}{u_n} = \infin$) 时级数发散; $\rho = 1$ 时级数可能收敛也可能发散
    1. **根值审敛法, 柯西判别法**: 设 $\displaystyle \sum _{n=1}^{\infin}u_n$ 是正项级数, 如果 $\displaystyle \lim _{n \to \infin} \sqrt[n]{u_n} = \rho$, 则当 $\rho < 1$ 时级数收敛; $\rho > 1$ (或 $\displaystyle \lim _{n \to \infin} \dfrac{u_{n+1}}{u_n} = \infin$) 时级数发散; $\rho = 1$ 时级数可能收敛也可能发散
    1. **根值审敛法**: 设 $\displaystyle \sum _{n=1}^{\infin}u_n$ 是正项级数,
        1. 如果 $\displaystyle \lim _{n \to \infin} nu_n = l > 0$ (或 $\displaystyle \lim _{n \to \infin} nu_n = +\infin$), 则级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 发散
        1. 如果 $p > 1$, 而 $\displaystyle \lim _{n \to \infin} n^p u_n = l (0 \leqslant l < +\infin)$, 则级数 $\displaystyle \sum _{n=1}^{\infin}u_n$ 收敛
    1. **莱布尼兹定理**: 设交错级数 $\displaystyle \sum _{n=1}^{\infin}(-1)^{n-1} u_n$ 满足条件: (1) $u_n \geqslant u_{n+1} (n=1,2,3,\cdots)$ (2) $\displaystyle \lim _{n \to \infin} u_n =0$, 则级数收敛, 且其和 $s \leqslant u_1$, 其余项 $r_n$ 的绝对值 $|r_n| \leqslant u_{n+1}$
        > 如果级数 $\displaystyle \sum _{n=1}^{\infin} u_n$ 各项的绝对值所构成的正项级数 $\displaystyle \sum _{n=1}^{\infin} |u_n|$ 收敛, 则称级数 $\displaystyle \sum _{n=1}^{\infin} u_n$ 绝对收敛; </br>如果级数 $\displaystyle \sum _{n=1}^{\infin} u_n$ 收敛, 而级数 $\displaystyle \sum _{n=1}^{\infin} |u_n|$ 发散, 则称级数 $\displaystyle \sum _{n=1}^{\infin} u_n$ 条件收敛
    1. 如果 $\displaystyle \sum _{n=1}^{\infin} u_n$ 绝对收敛, 则级数$\displaystyle \sum _{n=1}^{\infin} u_n$ 必定收敛
    1. 绝对收敛级数经改变项的位置后构成的级数也收敛, 且与原级数有相同的和 (即绝对收敛级数具有可交换性)
    1. **绝对收敛级数的乘法**: 设级数 $\displaystyle \sum _{n=1}^{\infin} u_n$ 和 $\displaystyle \sum _{n=1}^{\infin} v_n$ 都绝对收敛, 其和分别为 $s$ 和 $\sigma$, 则它们的柯西乘积 $u_1 v_1 + (u_1 v_2 + u_2 v_1) + \cdots + (u_1 v_n + u_2 v_{n-1} + \cdots + u_n v_1) + \cdots$ 也是绝对收敛的, 且其和为 $s \cdot \sigma$

## 8.2. 幂级数
* **定义**:
    * **级数**: 如果给定一个定义在区间 $I$ 上的函数列 $$u_1(x), u_2(x), u_3(x), \cdots, u_n(x), \cdots$$ 则由这函数列构成的表达式 $$u_1(x) + u_2(x) + u_3(x) + \cdots + u_n(x) + \cdots \quad (1)$$ 称为定义在区间 $I$ 上的 **(函数项) 无穷级数**, 简称 **(函数项) 级数**
    * 对于每一个确定的值 $x_0 \in I$, 函数项级数(1)成为常数项级数 $$u_1(x_0) + u_2(x_0) + u_3(x_0) + \cdots + u_n(x_0) + \cdots \quad (2)$$ 如果级数(2)收敛, 就称点 $x_0$ 是(1)的**收敛点**; 如果发散, 称为**发散点**. 收敛点的全体称为**收敛域**, 发散点的全体称为**发散域**
    * **幂级数**: $\displaystyle \sum _{n=0}^{\infin} a_nx^n = a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n + \cdots$, 其中$a_0, a_1, \cdots, a_n, \cdots$ 叫做**幂级数的系数**
* **定理**:
    1. **阿贝尔 (Abel)定理** 如果级数 $\displaystyle \sum _{n=0}^{\infin} a_nx^n$ 当 $x=x_0(x_0 \ne 0)$ 时收敛, 则适合不等式 $|x| < |x_0|$ 的一切 $x$ 使这个幂级数绝对收敛, 反之, 如果级数 $\displaystyle \sum _{n=0}^{\infin} a_nx^n$ 当 $x=x_0(x_0 \ne 0)$ 时发散, 则适合不等式 $|x| > |x_0|$ 的一切 $x$ 使这个幂级数发散
        * 如果幂级数 $\displaystyle \sum _{n=0}^{\infin} a_nx^n$ 不是仅在 $x=0$ 一点收敛, 也不是在整个数轴上都收敛, 则必有一个确定的正数 $R$ 存在, 使得
            1. 当 $|x| < R$ 时, 幂级数绝对收敛
            1. 当 $|x| > R$ 时, 幂级数发散
            1. 当 $|x| = R$ 时, 幂级数可能收敛也可能发散
            * 正数 $R$ 通常叫做**收敛半径**, 开区间 $(-R, R)$ 叫做**收敛区间**
    1. 如果 $\displaystyle \lim _{n \to \infin} \bigg| \dfrac {a _{n+1}}{a_n} \bigg| = \rho$, 其中 $a_n, a_{n+1}$ 是幂级数 $\displaystyle \sum _{n=0}^{\infin} a_nx^n$ 的相邻两项的系数, 则这幂级数的收敛半径 $R= \begin{cases} \dfrac {1}{\rho}, &\rho \ne 0 \\ +\infin, &\rho = 0 \\ 0, &\rho = + \infin   \end{cases}$
* 和函数的性质
    1. 幂级数 $\displaystyle \sum _{n=0}^{\infin} a_nx^n$ 的和函数 $s(x)$ 在其收敛域 $I$ 上连续
    1. 幂级数 $\displaystyle \sum _{n=0}^{\infin} a_nx^n$ 的和函数 $s(x)$ 在其收敛域 $I$ 上可积, 并有逐项积分公式 $$\int _0^x s(x) \mathrm{d}x = \int _0^x \Big[\sum _{n=0}^{\infin} a_nx^n \Big] \mathrm{d}x = \sum _{n=0}^{\infin} \int _0^x a_nx^n \mathrm{d}x = \sum _{n=0}^{\infin} \dfrac{a_n}{n+1}x^{n+1} \ (x \in I)$$ 逐项积分后所得到的幂级数和原级数有相同的收敛半径
    1. 幂级数 $\displaystyle \sum _{n=0}^{\infin} a_nx^n$ 的和函数 $s(x)$ 在其收敛区间 $(-R, R)$ 内可导, 且有逐项求导公式 $$s'(x) = \Big(\sum _{n=0}^{\infin} a_nx^n \Big)' = \sum _{n=0}^{\infin}(a_nx^n)' = \sum _{n=0}^{\infin} n a_n x^{n-1} \ (|x| < R)$$ 逐项求导后所得到的幂级数和原级数有相同的收敛半径
    * 反复应用上述结论可得: 幂级数 $\displaystyle \sum _{n=0}^{\infin} a_nx^n$ 的和函数 $s(x)$ 在其收敛区间 $(-R, R)$ 内具有任意阶导数
* **函数展开成幂级数**
    * 函数 $f(x)$ 在点 $x_0$ 处的 **泰勒级数**: $$f(x_0) + f'(x_0)(x-x_0) + \cdots + \dfrac{1}{n!}f^{(n)}(x_0)(x-x_0)^n + \cdots = \sum _{n=0}^{\infin} \dfrac{1}{n!}f^{(n)}(x_0)(x-x_0)^n$$
    * **泰勒展开式**: $\displaystyle f(x) = \sum _{n=0}^{\infin} \dfrac{1}{n!}f^{(n)}(x_0)(x-x_0)^n, \ (x \in U(x_0))$
    * **定理**: 设函数 $f(x)$ 在点 $x_0$ 的某一邻域 $U(x_0)$ 内具有各阶导数, 则 $f(x)$ 在该邻域内能展开成泰勒级数 $\iff$ 在该邻域内 $f(x)$ 的泰勒公式中的余项 $R_n(x)$ 当 $n \to \infin$ 时的极限为0, 即 $\displaystyle \lim _{n \to \infin} R_n(x) = 0, x \in U(x_0)$
    * **麦克劳林级数**: $$f(0) + f'(0)(x) + \cdots + \dfrac{1}{n!}f^{(n)}(0)(x)^n + \cdots = \sum _{n=0}^{\infin} \dfrac{1}{n!}f^{(n)}(0)(x)^n$$
    * **泰勒展开式**: $\displaystyle f(x) = \sum _{n=0}^{\infin} \dfrac{1}{n!}f^{(n)}(0)(x)^n, \ (|x| < r)$ (前提: $f(x)$ 能在 $(-r, r)$ 内展开称 $x$ 的幂级数)
    * ==常见的幂级数==:
        * $\displaystyle \mathrm{e}^x = 1 + x + \dfrac{x^2}{2!} + \cdots + \dfrac{x^n}{n!} + \cdots = \sum _{n=0}^{\infin} \dfrac{1}{n!}x^n \quad (-\infin < x < +\infin)$
        </br>
        * $\displaystyle \sin x = x - \dfrac{x^3}{3!} + \dfrac{x^5}{5!} - \cdots + (-1)^k \dfrac{x^{2k+1}}{(2k+1)!} + \cdots = \sum _{k=0}^{\infin} \dfrac{(-1)^k}{(2k+1)!}x^{2k+1} \quad (-\infin < x < +\infin)$
        </br>
        * $\displaystyle \cos x = 1 - \dfrac{x^2}{2!} + \dfrac{x^4}{4!} - \cdots + (-1)^k \dfrac{x^{2k}}{(2k)!} + \cdots = \sum _{k=0}^{\infin} \dfrac{(-1)^k}{(2k)!}x^{2k} \quad (-\infin < x < +\infin)$
        </br>
        * $\displaystyle \dfrac{1}{1+x} = \sum _{n=0}^{\infin} (-1)^n x^n \quad (-1 < x < 1)$
        </br>
        * $\displaystyle \dfrac{\pi}{4} = 1 - \dfrac{1}{3} + \dfrac{1}{5} - \cdots = \sum _{k=0}^{\infin} (-1)^k \dfrac {1}{2k+1}$
        </br>
    * 应用:
        1. 近似计算
        1. 微分方程的幂级数解法
        1. **欧拉公式**: $\mathrm{e}^{\mathrm{i}x} = \cos x + \mathrm{i} \sin x$

## 8.3. 傅里叶级数
* 周期为 $2\pi$ 的周期函数 $f(x)$ 的**傅里叶级数**展开式: $$f(x) = \dfrac{a_0}{2} + \sum _{n=1}^{\infin}(a_n \cos nx + b_n \sin nx), x \in C$$ 其中 $$\begin{aligned} a_n = \dfrac {1}{\pi} \int _{-\pi}^{\pi}f(x) \cos nx \mathrm{d}x \quad &(n=0,1,2,3,\cdots) \\ b_n = \dfrac {1}{\pi} \int _{-\pi}^{\pi}f(x) \sin nx \mathrm{d}x \quad &(n=1,2,3,\cdots) \end{aligned}$$
    * 当 $f(x)$ 为奇函数时, $$\begin{aligned} a_n &= 0  &(n=0,1,2,3,\cdots) \\ b_n &= \dfrac {2}{\pi} \int _{0}^{\pi}f(x) \sin nx \mathrm{d}x &(n=1,2,3,\cdots) \end{aligned}$$ 即知奇函数的傅里叶级数是只包含正弦项的**正弦级数** $\displaystyle \sum _{n=1}^{\infin} b_n \sin nx$
    * 当 $f(x)$ 为偶函数时, $$\begin{aligned} a_n &= \dfrac {2}{\pi} \int _{0}^{\pi}f(x) \cos nx \mathrm{d}x  &(n=0,1,2,3,\cdots) \\ b_n &= 0 &(n=1,2,3,\cdots) \end{aligned}$$ 即知偶函数的傅里叶级数是只包含常数项和余弦项的**余弦级数** $\displaystyle \frac {a_0}{2} + \sum _{n=1}^{\infin} a_n \cos nx$
* **定理**
    1. **收敛定理, 迪利克雷 (Dirichlet) 充分条件**: 设 $f(x)$ 是周期为 $2\pi$ 的周期函数, 如果它满足:
        1. 在一个周期内连续或只有有限个第一类间断点
        1. 在一个周期内至多只有有限个极值点

        则 $f(x)$ 的傅里叶级数收敛, 并且
        * 当 $x$ 是 $f(x)$ 的连续点时, 级数收敛于 $f(x)$
        * 当 $x$ 是 $f(x)$ 的间断点时, 级数收敛于 $\dfrac {1}{2}[f(x^-) + f(x^+)]$
    1. 设周期为 $2l$ 的周期函数 $f(x)$ 满足收敛定理的条件, 则它的傅里叶级数展开式为 $$f(x) = \dfrac{a_0}{2} + \sum _{n=1}^{\infin}(a_n \cos \dfrac {n \pi x}{l} + b_n \sin \dfrac {n \pi x}{l}), x \in C$$ 其中 $$\begin{aligned} a_n &= \dfrac 1l \int _{-l}^l f(x) \cos \dfrac {n \pi x}{l} \mathrm{d}x \quad &(n=0,1,2,3,\cdots) \\ b_n &= \dfrac 1l \int _{-l}^l f(x) \sin \dfrac {n \pi x}{l} \mathrm{d}x \quad &(n=1,2,3,\cdots) \end{aligned}$$ $$C = \{x|f(x)= \frac{1}{2}[f(x^-) + f(x^+)]\}$$
        * 当 $f(x)$ 为奇函数时, $$f(x) = \sum _{n=1}^{\infin} b_n \sin \dfrac {n \pi x}{l}, x \in C$$ 其中 $$b_n = \dfrac {2}{l} \int _{0}^{l} f(x) \sin \dfrac {n \pi x}{l} \mathrm{d}x \quad (n=1,2,3,\cdots)$$
        * 当 $f(x)$ 为偶函数时,  $$f(x) = \frac {a_0}{2} + \sum _{n=1}^{\infin} a_n \cos \dfrac {n \pi x}{l}, x \in C$$ 其中 $$a_n = \dfrac {2}{l} \int _{0}^{l} f(x) \cos \dfrac {n \pi x}{l} \mathrm{d}x \quad (n=0,1,2,3,\cdots)$$
* **傅里叶级数的复数形式** $$f(x) = \sum _{n=-\infin}^{\infin} \large c_n \mathrm{e}^{i \frac {n \pi x}{l}}$$ 其中 $$c_n = \frac {1}{2l} \int _{-l}^{l} f(x) \mathrm{e}^{-i \frac {n \pi x}{l}} \mathrm{d}x \quad (n = 0,\pm 1, \pm 2, \cdots)$$

# 9. 常微分方程

## 9.1. 基本概念
* **微分方程**: 表示未知函数、未知函数的导数与自变量之间的关系的方程, 有时也简称方程
* **微分方程的阶**: 未知函数的最高级导数的阶数
* **解**, **通解**, **特解**
* **初始条件**
    > 如 $x_0, y_0, y_0'$

## 9.2. 可分离变量的微分方程
* 如果一阶微分方程可化成 $g(y) \mathrm{d}y = f(x) \mathrm{d}x$ 的形式, 那么原方程称为**可分离变量的微分方程**
    * 求解: 两边同时积分 $\displaystyle \int g(x) \mathrm{d}y = \int f(x) \mathrm{d}x$, 得 $G(y) = F(x) + C$

## 9.3. 齐次方程
* 如果一阶微分方程可化成 $\dfrac{\mathrm{d}y}{\mathrm{d}x} = \varphi (\dfrac{y}{x})$ 的形式, 那么该方程为**齐次方程**
    * 求解:
        1. 引入未知函数 $u = \dfrac{y}{x}$, 则 $y=ux, \dfrac{\mathrm{d}y}{\mathrm{d}x} = u+x \dfrac{\mathrm{d}u}{\mathrm{d}x}$
        </br>
        1. 代入方程后并简化, 得 $\dfrac{\mathrm{d}u}{\varphi(u)-u} = \dfrac{\mathrm{d}x}{x}$
        </br>
        1. 两端同时积分, 得 $\displaystyle \int \dfrac{\mathrm{d}u}{\varphi(u)-u} = \int \dfrac{\mathrm{d}x}{x}$
        </br>
        1. 求出积分后, 再以 $\dfrac{y}{x}$ 代替 $u$
    </br>
* 可化为齐次方程的方程 $\dfrac {\mathrm{d}y}{\mathrm{d}x} = \dfrac{ax+by+c}{a_1 x+b_1 y+c_1}$,
    * 当 $c=c_1=0$ 时是齐次的
    * 在非齐次的情形,可以变化为齐次方程
        1. 令 $x=X+h, y = Y+k (其中 h, k 是待定的常数)$, 得 $\mathrm{d}x=\mathrm{d}X, \mathrm{d}y=\mathrm{d}Y$
            </br>
        1. 方程变为 $\dfrac {\mathrm{d}Y}{\mathrm{d}X} = \dfrac{aX+bY+ah+bk+c}{a_1 X+b_1 Y+a_1 h + b_1 k + c_1}$
            </br>
        1. 如果方程组 $\begin{cases} ah+bk+c= 0\\ a_1 h + b_1 k + c_1 =0 \end{cases}$ 的系数行列式 $\begin{vmatrix} a &b \\ a_1 &b_1 \end{vmatrix} \ne 0$, 即 $\dfrac{a_1}{a} \ne \dfrac{b_1}{b}$, 那么可以求解 $h, k$
            </br>
            1. 微分方程便化为齐次方程 $\dfrac{\mathrm{d}Y}{\mathrm{d}X} = \dfrac{aX+bY}{a_1 X+b_1 Y}$
                </br>
            1. 得到通解后, 代入 $X=x-h, Y=y-k$
                </br>
        1. 如果方程组$\begin{cases} ah+bk+c= 0\\ a_1 h + b_1 k + c_1 =0 \end{cases}$ 的系数行列式 $\begin{vmatrix} a &b \\ a_1 &b_1 \end{vmatrix} = 0$, 即 $\dfrac{a_1}{a} = \dfrac{b_1}{b}$, $h,k$ 无法求解, 此时令 $\dfrac{a_1}{a} = \dfrac{b_1}{b} = \lambda$
            1. 微分方程可化为 $\dfrac{\mathrm{d}y}{\mathrm{d}x} = \dfrac{ax+by+c}{\lambda(ax+by)+c_1}$
                </br>
            1. 引入新变量 $v=ax+by$, 则 $\dfrac{\mathrm{d}v}{\mathrm{d}x} = a+b\dfrac{\mathrm{d}y}{\mathrm{d}x}$ 或 $\dfrac{\mathrm{d}y}{\mathrm{d}x} = \dfrac{1}{b}(\dfrac{\mathrm{d}v}{\mathrm{d}x}-a)$
                </br>
            1. 微分方程化为 $\dfrac{1}{b}(\dfrac{\mathrm{d}v}{\mathrm{d}x}-a) = \dfrac{v+c}{\lambda v + c_1}$, 这是可分离变量的微分方程

## 9.4. 一阶线性微分方程
* 方程 $\dfrac{\mathrm{d}y}{\mathrm{d}x} + P(x)y = Q(x)$ 叫做**一阶线性微分方程**
    * 如果 $Q(x) \equiv 0$, 则方程称为齐次的; 否则是非齐次的
    * 通解 $$\displaystyle y=C\mathrm{e}^{-\int P(x) \mathrm{d} x} + \mathrm{e}^{-\int P(x) \mathrm{d} x}\int Q(x) \mathrm{e}^{\int P(x) \mathrm{d} x} \mathrm{d}x$$
        * 第一项是对应的齐次方程的通解
        * 第二项是非齐次方程的一个特解

* 伯努利方程 $\dfrac{\mathrm{d}y}{\mathrm{d}x} + P(x)y = Q(x)y^n \quad (n \ne 0, 1)$
    * 当 $n = 0,1$ 时, 方程是线性微分方程, 否则是非线性的
    * 求解:
        1. 两端除以 $y^n$, 得 $y^{-n} \dfrac{\mathrm{d}y}{\mathrm{d}x} + P(x)y^{1-n} = Q(x)$
        1. 引入新变量 $z=y^{1-n}$, 则 $\dfrac{\mathrm{d}z}{\mathrm{d}x} = (1-n)y^{-n}\dfrac{\mathrm{d}y}{\mathrm{d}x}$
        1. 方程继续化为 $\dfrac{\mathrm{d}z}{\mathrm{d}x} + (1-n)P(x)z=(1-n)Q(x)$
        1. 求通解
        1. 代入 $z=y^{1-n}$

## 9.5. 可降阶的高阶微分方程
* $y^{(n)} = f(x)$ 型的求解
    1. 将 $y^{(n-1)}$ 作为新的未知函数, 则
        $\displaystyle y^{(n-1)} = \int f(x) \mathrm{d}x + C_1$
        $\displaystyle y^{(n-2)} = \int \Big [ \int f(x) \mathrm{d}x + C_1\Big ] \mathrm{d}x + C_2$
    1. 连续积分 $n$ 次
    </br>
* $y''=f(x, y')$ 型的求解
    1. 设 $y' = p$, 则 $y'' = p'$
    1. 方程变为 $p' = f(x, p)$, 这是一个关于 $x, p$ 的一阶微分方程, 设其通解为 $p = \varphi (x, C_1) = \dfrac{\mathrm{d}y}{\mathrm{d}x}$
    1. 分离变量并积分, 得通解 $$\displaystyle y = \int \varphi (x, C_1) \mathrm{d}x +C_2$$
* $y'' = f(y, y')$ 型的求解
    1. 令 $y' = p$, 则 $y'' = p \dfrac{\mathrm{d}p}{\mathrm{d}y}$
    1. 方程变为 $p \dfrac{\mathrm{d}p}{\mathrm{d}y} = f(y, p)$, 这是关于$y, p$ 的一阶微分方程, 设其通解为 $y' = p = \varphi (y, C_1)$
    1. 分离变量并积分, 得通解 $$\displaystyle \int \frac{\mathrm{d}y}{\varphi (y, C_1)} = x + C_2$$

## 9.6. 高阶线性微分方程

### 9.6.1. 二维齐次线性方程
二维齐次线性方程: $y'' + P(x) y' + Q(x)y = 0$
* **定理1**: 如果 $y_1(x)$ 与 $y_2(x)$ 是方程的两个解, 那么 $$y = C_1 y_1(x) + C_2 y_2(x)$$ 也是其解, 其中 $C_1, C_1$ 是任意常数
* **定理2**: 如果 $y_1(x)$ 与 $y_2(x)$ 是方程的两个线性无关的特解, 那么 $$y = C_1 y_1(x) + C_2 y_2(x) \ (C_1, C_2是任意常数)$$ 是方程的通解
    * **推论**:如果 $y_1(x), y_2(x), \cdots, y_n(x)$ 是 $n$ 阶齐次方程 $$y^{(n)} + a_1(x) y^{(n-1)} + \cdots + a_{n-1}(x) y^{(n-1)} + a_n(x)y = 0 $$ 的 $n$ 个线性无关的解, 那么其通解为 $$ y = C_1 y_1(x) + C_2 y_2(x) + \cdots + C_n y_n(x_0) \ (C_1, C_2, \cdots, C_n是任意常数)$$
* **定理3**: 如果 $y^* (x)$ 是二维非齐次线性方程 $y'' + P(x) y' + Q(x)y = f(x)$ 的一个特解, $Y(x)$ 是对应的齐次方程 $y'' + P(x) y' + Q(x)y = 0$ 的通解, 那么 $$y = Y(x) + y^* (x)$$ 是二维非齐次线性方程的通解
* **定理4**: 如果 $y_1^*(x), y_2^*(x)$ 分别是二维非齐次线性方程 $$\begin{alignedat}{2}
                y'' + P(x) y' + Q(x)y = f_1(x) \\
                y'' + P(x) y' + Q(x)y = f_2(x) \end{alignedat}$$ 的一个特解, 那么 $$y_1^*(x) + y_2^*(x)$$ 是方程 $y'' + P(x) y' + Q(x)y = f_1(x)+f_2(x)$ 的一个特解
### 9.6.2. 常数变易法
* 如果 $Cy_1(x)$ 是齐次线性方程的通解, 那么可以利用变换 $y=uy_1(x)$ (这变换是把齐次方程的通解中的任意常数C换成未知函数 $u$ 而得到的) 去解非齐次线性方程

## 9.7. 常系数齐次线性微分方程
* 二阶**常系数**齐次微分线性方程: $y'' + py' + qy = 0 \quad (p, q是常数)$
    * 求解:
        1. 写出特征方程 $r^2+pr+q=0$
        1. 求特征方程的两个根 $r_1, r_2$
        1. 根据两个根的不同情形, 得到通解

            特征方程的两个根 $r_1, r_2$ | 微分方程 $y'' + py' + qy = 0$ 的通解
            :-------------------------- | :----------------------------------:
            两个不相等的实根            | $y = C_1 \mathrm{e}^{r_1 x}+C_2 \mathrm{e}^{r_2 x}$
            两个相等的实根              | $y = (C_1 + C_2 x) \mathrm{e}^{r_1 x}$
            一对共轭复根 $r_{1,2} = \alpha \pm \mathrm{i} \beta$ | $y = \mathrm{e}^{\alpha x}(C_1 \cos {\beta x} + C_2 \sin {\beta x})$
* 二阶**变系数**齐次微分线性方程: $y'' + py' + qy = 0 \quad (p, q不全是常数)$
* n 阶**常系数**齐次微分线性方程: $y^{(n)} + p_1 y^{(n-1)} + p_2 y^{n-2} + \dots + p_{n-1} y' + p_n y = 0$ 其中 $p_1, p_2, \cdots, p_{n-1}, p_n$ 都是常数
    * 可以用记号 $\mathrm {D}$ (微分算子) 表示 $\dfrac{\mathrm{d}}{\mathrm{d}x}$, 把 $\dfrac{\mathrm{d}y}{\mathrm{d}x}$ 记 $\mathrm{D} y$, 把 $\dfrac{\mathrm{d}^n y}{\mathrm{d}x^n}$ 记 $\mathrm{D}^n y$
    </br>
    * 把 **n 阶常系数齐次微分线性方程**记作 $(\mathrm{D}^n + p_1 \mathrm{D}^{n-1} + \cdots + p_{n-1} \mathrm{D} + p_n)y=0$, 记 $\mathrm{L}(\mathrm{D}) = (\mathrm{D}^n + p_1 \mathrm{D}^{n-1} + \cdots + p_{n-1} \mathrm{D} + p_n)$, $\mathrm{L}(\mathrm{D})$ 叫做**微分算子 $\mathrm{D}$ 的 $n$ 次多项式**
    * **n 阶常系数齐次微分线性方程** 的特征方程是 $\mathrm{L}(r) = 0$ 即 $r^n + p_1 r^{n-1} + \cdots + p_{n-1} r + p_n = 0$
    * 根据特征方程的解,可以得到对应的微分方程的解

        特征方程的根 | 微分方程通解中的对应项
        :----------- | :---------------------
        单实根 $r$   | 给出一项 $C e^{rx}$
        一对单复根 $r_{1,2} = \alpha \pm \mathrm{i} \beta$ | 给出两项 $\mathrm{e}^{\alpha x}(C_1 \cos {\beta x} + C_2 \sin {\beta x})$
        $k$ 重实根 $r$ | 给出 $k$ 项 $e^{rx}(C_1 + C_2 x + \cdots + C_k x^{k-1})$
        一对 $k$ 重复根 $r_{1,2} = \alpha \pm \mathrm{i} \beta$ | 给出 $2k$ 项 $\mathrm{e}^{\alpha x}[(C_1 + C_2 x + \cdots + C_k x^{k-1}) \cos {\beta x} + (D_1 + D_2 x + \cdots + D_k x^{k-1}) \sin {\beta x}]$

## 9.8. 常系数非齐次线性微分方程
* 通解: 齐次方程的通解 $Y(x)$ + 非齐次方程的一个特解$y^* (x)$, 即 $y = Y(x) + y^* (x)$
* 二阶常系数非齐次微分线性方程: $y'' + py' + qy = f(x) \quad (p, q是常数)$
    * $f(x)$ 的两种形式
        1. $f(x) = P_m(x) \mathrm{e}^{\lambda x}$, 其中 $\lambda$ 是常数, $P_m(x)$ 是 $x$ 的一个 $m$ 次多项式 $P_m(x) = a_0 x^{m} + a_1 x^{m-1} + \cdots + a_{m-1} x + a_m$
        1. $f(x) = e^{\lambda x}[P_l(x) \cos {\omega x} + P_n(x) \sin {\omega x}]$, 其中 $\lambda \omega$ 是常数, $P_l(x), P_n(x)$ 分别是 $x$ 的 $l, n$ 次多项式, 且有一个可为 0
* $f(x) = \mathrm{e}^{\lambda x} P_m(x)$ 型的特解 $$y^* = x^k Q_m(x) e^{\lambda x}$$ 其中 $Q_m(x)$ 是与 $P_m(x)$ 同次 ($m$ 次) 的多项式, 而 $k=\begin{cases}
        0, &\lambda 不是特征方程的根\\
        1, &\lambda 是特征方程的单根\\
        2, &\lambda 是特征方程的重根\\
        \end{cases}$
    * 将特解代入微分方程, 可以求得 $Q_m(x)$ 的系数
* $f(x) = e^{\lambda x}[P_l(x) \cos {\omega x} + P_n(x) \sin {\omega x}]$ 型的特解 $$y^* = x^k \mathrm{e}^{\lambda x}[R_m^{(1)}(x) \cos {\omega x} + R_m^{(2)}(x) \sin {\omega x}]$$ 其中 $R_m^{(1)}(x), R_m^{(2)}(x)$ 是 $m$ 次多项式, $m = \max (l, n)$, 而 $k=\begin{cases}
        0, &\lambda + \mathrm{i} \omega (或 \lambda - \mathrm{i} \omega) 不是特征方程的根\\
        1, &\lambda + \mathrm{i} \omega (或 \lambda - \mathrm{i} \omega) 是特征方程的单根
        \end{cases}$
    * 将特解代入微分方程, 可以求得 $R_m^{(1)}(x), R_m^{(2)}(x)$ 的系数

## 9.9. 欧拉方程
* 欧拉方程 $x^n y^{(n)} + p_1 x^{n-1} y^{(n-1)}+ \cdots + p_{n-1} x y' + p_n y = f(x)$ 的求解
    1. 作变换 $x = e^t$, 或 $t = \ln x$, 则 $$\begin{gathered}
        \dfrac{\mathrm{d}y}{\mathrm{d}x} = \dfrac{\mathrm{d}y}{\mathrm{d}t} \dfrac{\mathrm{d}t}{\mathrm{d}x} = \dfrac {1}{x} \dfrac{\mathrm{d}y}{\mathrm{d}t} \\
        \dfrac{\mathrm{d}^2 y}{\mathrm{d}x^2} = \dfrac {1}{x^2} (\dfrac{\mathrm{d}^2 y}{\mathrm{d}t^2} - \dfrac{\mathrm{d}y}{\mathrm{d}t}) \\
        \dfrac{\mathrm{d}^3 y}{\mathrm{d}x^3} = \dfrac {1}{x^3} (\dfrac{\mathrm{d}^3 y}{\mathrm{d}t^3} - 3\dfrac{\mathrm{d}^2 y}{\mathrm{d}t^2} + 2\dfrac{\mathrm{d}y}{\mathrm{d}t}) \\
        \end{gathered}$$
    1. 如果采用记号 $\mathrm{D}$ 表示对 $t$ 求导的运算 $\dfrac{\mathrm{d}}{\mathrm{d}t}$, 那么上述计算结果可写为  $$\begin{gathered}
        xy' = \mathrm{D} y \\
        x^2y'' = \dfrac{\mathrm{d}^2 y}{\mathrm{d}t^2} - \dfrac{\mathrm{d}y}{\mathrm{d}t} = (\dfrac{\mathrm{d}^2}{\mathrm{d}t^2} - \dfrac{\mathrm{d}}{\mathrm{d}t})y = (\mathrm{D}^2-\mathrm{D})y = \mathrm{D}(\mathrm{D}-1)y\\
        x^3y''' = \dfrac{\mathrm{d}^3 y}{\mathrm{d}t^3} - 3\dfrac{\mathrm{d}^2 y}{\mathrm{d}t^2} + 2\dfrac{\mathrm{d}y}{\mathrm{d}t} = (\mathrm{D}^3 - 3\mathrm{D}^2+2 \mathrm{D}) =\mathrm{D}(\mathrm{D}-1)(\mathrm{D}-2)y\\
        \end{gathered}$$
    1. 一般的, $x^k y^{(k)} = \mathrm{D}(\mathrm{D}-1)\cdots(\mathrm{D}-k+1)y$, 代入欧拉方程, 得到一个以 $t$ 为自变量的常系数线性微分方程
    1. 求解后, 用 $\ln x$ 替换 $t$

# 11. 复变函数

## 11.1. 复数与复变函数
* **复数**: 参见 [1.1.1. 复数](#111-复数)
* **复球面**: 球面上的每一点都有唯一的复数与之对应
    ![复球面](/assets/分析数学-复球面.png)
    * 复数 $\leftrightarrow$ 复平面上的点 $\xleftrightarrow{除北极 N 外}$ 球面上的点
    * $\infin \leftrightarrow$ 无穷远点 $\leftrightarrow$ $N$ 点
* **复变函数**: 设 $G$ 是一个复数的集合, 如果有一个确定的法则存在, 按照这一法则, 对于集合中的每一个复数 $z \in G$, 就有一个或几个复数 $w=u+iv$ 与之对应, 那么称复变数 $w$ 是复变数 $z$ 的函数, 即**复变函数**, 记做 $w = f(z)$
    * 若 $z$ 的一个值仅对应着 $w$ 的一个值, 称 $f(z)$ 为**单值函数**, 否则称为**多值函数**
    * $G$ 称为**定义域**
    * $w = f(z) = u(x, y) + \mathrm{i} v(x,y) \iff u(x, y), v(x,y)$ 为两个二元函数
* **复变函数定理**:
    1. 设函数 $f(z) = u(x, y) + \mathrm{i} v(x,y), A = u_0 + \mathrm{i} v_0, z_0 = x_0 + \mathrm{i} y_0$, 则 $$\displaystyle \lim _{z \to z_0} f(z) = A \iff \begin{cases} \displaystyle \lim _{\substack{x \to x_0 \\ \\ y \to y_0}} u(x,y) = u_0 \\ \displaystyle \lim _{\substack{x \to x_0 \\ \\ y \to y_0}} v(x,y) = v_0 \end{cases}$$
    1. 若 $\displaystyle \lim _{z \to z_0} f(z) = A, \lim _{z \to z_0} g(z) = B$, 则:
        * $\displaystyle \lim _{z \to z_0} [f(z) \pm g(z)] = A \pm B$
        * $\displaystyle \lim _{z \to z_0} [f(z) \cdot g(z)] = A \cdot B$
        * $\displaystyle \lim _{z \to z_0} \dfrac {f(z)}{g(z)} = \dfrac A B \quad (B \ne 0)$
    1. 函数 $f(z) = u(x, y) + \mathrm{i} v(x,y)$ 在点 $z_0 = x_0 + \mathrm{i} y_0$ 处连续 $\iff$ 二元函数 $u(x, y), v(x,y)$ 在 $(x_0, y_0)$ 处连续
    1. 连续函数的和、差、积、商是连续函数, 复合函数是连续函数
    1. 有界闭区域 $\bar D$ 上的连续函数是有界的, 且其模 $|f(z)|$ 可取得最大值和最小值

## 11.2. 解析函数与调和函数
* **定义**:
    * **解析函数**:
        * 如果函数 $f(z)$ 在 $z_0$ 及 $z_0$ 的某一邻域内处处可导, 那么称 $f(z)$ 在 $z_0$ 处**解析**
        * 如果函数 $f(z)$ 在 $D$ 内每一点都解析, 那么称 $f(z)$ 在 $D$ 内解析, 或称 $f(z)$ 是 $D$ 内的一个**解析函数** (**全纯函数**或**正则函数**)
        * 若函数 $f(z)$ 在 $z_0$ 处不解析, 那么称 $z_0$ 为函数 $f(z)$ 的**奇点**
    * **调和函数**: 如果二元函变函数 $\varphi (x, y)$ 满足:
        1. 在区域 $D$ 内具有二阶连续偏导数
        1. 满足二维拉普拉斯 (Laplace) 方程: $\dfrac {\partial^2 \varphi}{\partial x^2} + \dfrac {\partial^2 \varphi}{\partial y^2} = 0$

        则称 $\varphi (x,y)$ 为区域 $D$ 内的调和函数
    * **共轭调和函数**: 设函数 $\varphi (x, y)$ 及 $\psi  (x, y)$ 均为区域D内的调和函数, 且满足柯西-黎曼(Cauchy -Riemann, C-R) 方程 $\dfrac {\partial \varphi}{\partial x} = \dfrac {\partial \psi}{\partial y}, \dfrac {\partial \psi}{\partial x} = -\dfrac {\partial \varphi}{\partial y}$, 则称 $\psi  (x, y)$ 是 $\varphi (x, y)$ 的共轭调和函数
* **求导法则**: 与实函数类似
* **定理**:
    1. 在区域 $D$ 内解析函数的和、差、积、商 (除去分母为 0 的点) 在 $D$ 内解析
    1. 设函数 $h=g(z)$ 在 $z$ 平面上的区域 $D$ 内解析, $w = f (h)$ 在 $h$ 平面上的区域 $G$ 内解析, 如果对 $D$ 内的每一个点 $z$, 函数 $g(z)$ 的对应值都属于 $G$, 那么复合函数 $w=f [g(z)]$ 在 $G$ 内解析
    1. 任何有理分式函数 $\dfrac {P(z)}{Q(z)}$ 在分母不为 0 的点的区域内是解析函数, 使分母为 0 的点是函数奇点
    1. 函数 $f(z) = u(x, y) + \mathrm{i} v(x, y)$ 定义在区域 $D$ 内, 则 $f(z)$ 在 $D$ 内一点 $z = x + \mathrm{i} y$ 处可导
        $\iff$ $u(x, y)$ 与 $v(x, y)$ 与在点 $(x, y)$ 可微, 且在该点满足 C-R 方程: $\dfrac {\partial u}{\partial x} = \dfrac {\partial v}{\partial y}, \dfrac {\partial u}{\partial y} = - \dfrac {\partial v}{\partial x}$
        $\iff$ 在区域 $D$ 内, $f(z)$ 的虚部 $v(x, y)$ 是实部 $u(x, y)$ 的共轭调和函数
        $\implies$ $f(z)$ 的实部 $u(x, y)$ 和虚部 $v(x, y)$ 都是调和函数
* **解析函数与调和函数的关系**: 如果已知一个调和函数 $u(x, y)$, 则可利用 C−R 方程求得它的共轭调和函数 $v(x, y)$, 从而构成一个解析函数 $f(z) = u(x, y) + \mathrm{i} v(x, y)$

## 11.3. 复数初等函数
* **指数函数**: $f(z) = \mathrm{e}^z = \mathrm{e}^x(\cos y + \mathrm{i} \sin y)$
    1. $\mathrm{e}^z \ne 0$
    1. 模: $|\mathrm{e}^z| = \mathrm{e}^x$
    1. 幅角: $\mathrm{Arg} \mathrm{e}^z = y + 2k\pi \quad (k = 0, \pm 1, \pm 2, \cdots)$
    1. 周期: $\mathrm{e}^{z+\mathrm{i}2k\pi} = \mathrm{e}^z(\cos 2k\pi + \mathrm{i}\sin 2k\pi) = \mathrm{e}^z$, 即 $T = \mathrm{i} 2k\pi$
    1. 在整个复平面上是解析函数, $$(\mathrm{e}^z)'=\mathrm{e}^z$$
* **对数函数**: $f(z) = \mathrm{Ln} z = \ln |z| + i(\theta + 2k\pi) = \ln |z| + \mathrm{i} \mathrm{Arg} z \quad (k = 0, \pm 1, \pm 2, \cdots)$
    1. $\mathrm{Ln} z$ 是多值函数, 每两值相差 $\mathrm{i}2\pi$ 的整数倍, 主值: $\ln |z| + \mathrm{i} \arg z$
    1. $\mathrm{Ln} (z_1 \cdot z_2) = \mathrm{Ln} z_1 + \mathrm{Ln} z_2, \mathrm{Ln} \dfrac {z_1}{z_2} = \mathrm{Ln} z_1 - \mathrm{Ln} z_2$
    1. 由于 $\mathrm{Ln} z$ 是多值函数, $\mathrm{Ln} z^n \ne n \mathrm{Ln} z, \mathrm{Ln} \sqrt[n]z \ne \dfrac {1}{n} \mathrm{Ln} z$
    1. $\ln z$ 的各个分支在除原点及负实轴的复平面内解析, $$(\ln z)' = \dfrac 1 z$$
* **幂函数**: $z^a = \mathrm{e}^{a \ln z} \quad (a为复常数, z \ne 0)$
    1. 当 $a = 0$ 时, $z^a = z^0 = 1$
    1. 当 $a$ 为正整数 $n$ 时, $$\large z^a = z^n = \mathrm{e}^{n\mathrm{Ln} z} = \mathrm{e}^{n[\ln |z| + \mathrm{i}(\arg z + 2k\pi)]} = |z|^n \mathrm{e}^{\mathrm{i} n \arg z}$$ 在复平面内是单值解析函数, $$(z^n)' = n z^{n-1}$$
    1. 当 $a = \dfrac {1}{n} (n 为正整数)$ 时, $$\large z^{\frac{1}{n}} = \mathrm{e}^{\frac{1}{n}\mathrm{Ln} z} = \mathrm{e}^{\frac{1}{n}\big[\ln|z| + \mathrm{i}(\arg z + 2k\pi)\big]} = |z|^{\frac{1}{n}} \mathrm{e}^{\mathrm{i} \frac{\arg z + 2k\pi}{n}} \normalsize \quad (k=0,1,2,\cdots, n-1)$$ 是一个 $n$ 值函数, 它的各分支在除去原点和负实轴外的复平面上是解析的, $$(z^{\frac{1}{n}})' = \large \frac{1}{n} z ^{\frac{1}{n}-1}$$
    1. 当 $a = \dfrac {p}{q} \ (p 和 q 为互质的整数, q > 0)$ 时 $$\large z^{\frac{p}{q}} = \mathrm{e}^{\frac{p}{q}\ln |z| + \mathrm{i} \frac{p}{q}(\arg z + 2k\pi)} \normalsize \quad (k=0,1,2,\cdots, q-1)$$ 是一个 $q$ 值函数
    1. 当 $a$ 为无理数或复数时, $z^a$ 有无穷多个值, 各分支在除去原点和负实轴外的复平面上是解析的, $$(z^a)' = a z^{a-1}$$
* **三角函数**: $\cos z = \dfrac{\mathrm{e}^{\mathrm{i}z} + \mathrm{e}^{-\mathrm{i}z}}{2}, \quad \sin z = \dfrac{\mathrm{e}^{\mathrm{i}z} - \mathrm{e}^{-\mathrm{i}z}}{2 \mathrm{i}}$
    1. 根据欧拉公式: $\cos y = \dfrac{\mathrm{e}^{\mathrm{i}y} + \mathrm{e}^{-\mathrm{i}y}}{2}, \ \sin y = \dfrac{\mathrm{e}^{\mathrm{i}y} - \mathrm{e}^{-\mathrm{i}y}}{2 \mathrm{i}}$
    1. $\cos z, \sin z$ 均为单值函数, 周期为 $2\pi$
    1. $\cos z$ 是偶函数, $\sin z$ 是奇函数
    1. $\cos z, \sin z$ 在整个复平面上处处解析, $$(\cos z)' = -\sin z, \quad (\sin z)' = \cos z$$
    1. 三角公式:
        * $\cos (z_1 \pm z_2) = \cos z_1 \cos z_2 \mp \sin z_1 \sin z_2$
        </br>
        * $\sin (z_1 \pm z_2) = \sin z_1 \cos z_2 \pm \cos z_1 \sin z_2$
        </br>
        * $\sin ^2 z + \cos ^2 z = 1$
        </br>
        * $\tan z = \dfrac {\sin z}{\cos z}, \ \cot z = \dfrac {\cos z}{\sin z}, \ \sec z = \dfrac {1}{\cos z}, \ \csc z = \dfrac {1}{\sin z}$
        </br>
* **双曲函数**:
    * 双曲余弦: $\cosh z = \dfrac {\mathrm{e}^z + \mathrm{e}^{-z}}{2}$
    </br>
    * 双曲正弦: $\sinh z = \dfrac {\mathrm{e}^z - \mathrm{e}^{-z}}{2}$
    </br>
    * 双曲正切: $\tanh z = \dfrac{\sinh z}{\cosh z} = \dfrac {\mathrm{e}^z - \mathrm{e}^{-z}}{\mathrm{e}^z + \mathrm{e}^{-z}}$
    </br>
    * 双曲余切: $\coth z = \dfrac{\cosh z}{\sinh z} = \dfrac {\mathrm{e}^z + \mathrm{e}^{-z}}{\mathrm{e}^z - \mathrm{e}^{-z}}$
    * 性质:
        1. $\cosh z$ 和 $\sinh z$ 的周期都是 $\mathrm{i} 2\pi$
        1. $\cosh z$ 是偶函数, $\sinh z$ 是奇函数
        1. $\cosh z, \sinh z$ 在整个复平面上处处解析, $$(\cosh z)' = \sinh z, \quad (\sinh z)' = \cosh z$$
    * 双曲公式
        1. $\cosh \mathrm{i} y = \cos y, \quad \sinh \mathrm{i} y = \mathrm{i} \sin y$
        </br>
        1. $\cos \mathrm{i} y = \cosh y, \quad \sin \mathrm{i} y = \mathrm{i} \sinh y$
        </br>
        1. $\cos (x + \mathrm{i}y) = \cos x \cos \mathrm{i}y - \sin x \sin \mathrm{i} y = \cos x \cosh y - \mathrm{i} \sin x \sinh y$
        </br>
        1. $\sin (x + \mathrm{i}y) = \sin x \cos \mathrm{i}y + \cos x \sin \mathrm{i} y = \sin x \cosh y + \mathrm{i} \cos x \sinh y$
        </br>
        1. $\cosh (x + \mathrm{i}y) = \cosh x \cos y + \mathrm{i} \sinh x \sin y$
        </br>
        1. $\sinh (x + \mathrm{i}y) = \sinh x \cos y + \mathrm{i} \cosh x \sin y$
        </br>
* **反三角函数**:
    * $\arccos z = - \mathrm{i} \ln (z + \sqrt {z^2 - 1})$
    </br>
    * $\arcsin z = - \mathrm{i} \ln (\mathrm{i} z + \sqrt {1 - z^2})$
    </br>
    * $\arctan z = \dfrac {\mathrm{i}}{2} \ln \dfrac {1 + \mathrm{i}z}{1-\mathrm{i}z}$
    </br>
* **反双曲角函数**:
    * $\mathrm {arsh} z = \ln (z + \sqrt {z^2 + 1})$
    </br>
    * $\mathrm {arch} z = \ln (z + \sqrt {z^2 - 1})$
    </br>
    * $\mathrm {arth} z = \dfrac {1}{2} \ln \dfrac {1 + z}{1-z}$
    </br>

## 11.4. 复变函数的积分与高阶导数
* **定理**:
    1. **积分存在条件**: 设函数 $f(z) = u(x, y) + \mathrm{i} v(x, y)$ 在光滑曲线C上连续, 则复积分 $\displaystyle \int _C f(z) \mathrm{d} z$ 存在, 且有积分公式 $\displaystyle \int _C f(z) \mathrm{d} z = \int _C u(x,y) \mathrm{d} x - v(x,y) \mathrm{d} y + \mathrm{i} \int _C v(x,y) \mathrm{d} x + u(x,y) \mathrm{d} y$
    1. **柯西—古萨基本积分定理**: 设函数 $f(z)$ 在单连通区域 $D$ 内处处解析, 那么函数 $f(z)$ 在 $D$ 内沿任何一条封闭曲线 $C$ 的积分为零, 即 $\displaystyle \oint f(z) \mathrm{d}z = 0$
        > 适用情况:
        > 1. 若曲线 $C$ 是 $D$ 的边界, 如果函数 $f (z)$ 在 $D$ 内和 $C$ 上解析, 适用
        > 1. 若函数 $f(z)$ 在 $D$ 内解析, 在闭区域 $\bar{D} = D + C$ 上连续, 适用
    1. 设函数 $f(z)$ 在单连通区域 $D$ 内解析, $z_0$ 与 $z_1$ 为 $D$ 内任意两点, $C_1$ 与 $C_2$ 为连接 $z_0$ 与 $z_1$ 的积分路线, $C_1$ 与 $C_2$ 都含于 $D$ 内, 则 $\displaystyle \int _{C_1} f(z) \mathrm{d} z = \int _{C_2} f(z) \mathrm{d} z$
    1. **闭路变形定理**: 设 $C_1$ 与 $C_2$ 是两条简单闭曲线, $C_1$ 在 $C_2$ 内部, 函数 $f(z)$ 在 $C_1$ 与$C_2$ 所围成的二连通区域 $D$ 内解析, 而在 $\bar D = D + C_1  + C_2^-$ 上连续, 则 $\displaystyle \oint _{C_1} f(z) \mathrm{d} z = \oint _{C_2} f(z) \mathrm{d} z$
        * **复合闭路定理**: 设 $C$ 为多连通区域 $D$ 的一条简单闭曲线, $C_1,C_2 \cdots, C_n$ 是在 $C$ 内部的简单闭曲线, 它们互不包含也互不相交, 并且以 $C, C_1, C_2, \cdots, C_n$ 为边界的区域全包含于 $D$, 如果函数 $f (z)$ 在 $D$ 内解析, 则 $\displaystyle \oint _C f(z) \mathrm{d} z = \sum _{k=1}^n \oint _{C_k} f(z) \mathrm{d} z$, 其中 $C$ 及 $C_k$ 均取正方向
    1. 设函数 $f(z)$ 在单连通区域 $D$ 内处处解析, 则积分 $\displaystyle \int _C f(z) \mathrm{d} z$ 与连接起点及终点的路线 $C$ 无关
    1. **原函数**: 设函数 $f(z)$ 在单连通区域 $D$ 内解析, 则函数 $F(z)$ 必为 $D$ 内的一个解析函数, 且 $\displaystyle F'(z) = \Big[\int _{z_0}^{z} f(\zeta) \mathrm{d} \zeta \Big]' = f(z)$
    1. 设函数 $f(z)$ 在单连通区域 $D$ 内解析, 函数 $F(z)$ 为 $f(z)$ 的一个原函数, 则 $\displaystyle \int _{z_0}^{z_1} f(z) \mathrm{d} z = F(z_1) - F(z_0)$, 其中 $z_0, z_1$ 为区域 $D$ 内的两点
    1. **柯西积分公式**: 设函数 $f(z)$ 在简单闭曲线 $C$ 所围成的区域 $D内$ 解析, 在 $\bar D = D \cup C$ 上连续, $z_0$ 为 $D$ 内任意一点, 则 $\displaystyle f(z_0) = \dfrac {1} {\mathrm{i} 2\pi} \oint _C \dfrac {f(z)} {z-z_0} \mathrm{d} z$
        * **平均值公式**: 设函数 $f(z)$ 在 $|z-z_0| < R$ 内解析, 在 $|z-z_0| = R$ 上连续, 则 $\displaystyle f(z_0) = \dfrac {1} {2\pi} \int _0^{2\pi} f(z_0 + R \mathrm{e}^{\mathrm{i}\theta}) \mathrm{d} \theta$
            > 这表明解析函数在圆心处的值等于它在圆周上的平均值
        * 设函数 $f(z)$ 在简单闭曲线 $C_1, C_2$ 所围成的二连通域 $D$ 并在 $C_1, C_2$ 上连续, $C_2$ 在 $C_1$ 的内部, $z_0$ 为 $D$ 内一点, 则 $\displaystyle f(z_0) = \dfrac{1}{\mathrm{i} 2\pi} \Big[\oint _{C_1} \dfrac{f(z)}{z-z_0} \mathrm{d} z - \oint _{C_2} \dfrac{f(z)}{z-z_0} \mathrm{d} z \Big]$
    1. **最大模原理**: 设函数 $f(z)$ 在区域 $D$ 内解析, 又函数 $f(z)$ 不是常数, 则在 $D$ 内 $|f(z)|$ 没有最大值
        * 区域 $D$ 内解析的函数, 若其模在 $D$ 的内点达到最大值, 则此函数必恒为常数
        * 若函数 $f(z)$ 在有界区域 $D$ 内解析, 在 $\bar D$ 上连续, 则 $|f(z)|$ 必在 $D$ 的边界上达到最大值
* **法则**:
    1. $\displaystyle \int f(z) \mathrm{d} z = \int _C (u + \mathrm{i}v) (\mathrm{d} x + \mathrm{i} \mathrm{d} y)$
    </br>
    1. $\displaystyle \int f(z) \mathrm{d} z = \int _a^b f\big[z(t) \big] z'(t) \mathrm{d} t$
    </br>
    1. $\displaystyle \oint _C \dfrac {\mathrm{d} z}{(z - z_0)^{n+1}} = \begin{cases} \mathrm{i} 2\pi, & n = 0 \\ 0, & n \ne 0\end{cases}$ 其中: $C$ 以 $z_0$ 为中心, $r$ 为半径的正向圆周, $n$ 为正整数
        > 这个积分经常用的, 它的特点是与积分路线的中心和半径无关
* **性质**:
    1. $\displaystyle \int _C f(z) \mathrm{d} z = - \int _{C^-} f(z) \mathrm{d} z$
    </br>
    1. $\displaystyle \int _C k f(z) \mathrm{d} z = k \int _C f(z) \mathrm{d} z \quad (k \in \R)$
    </br>
    1. $\displaystyle \int _C \Big[f(z) \pm g(z) \Big] \mathrm{d} z = \int _C f(z) \mathrm{d} z \pm \int _C g(z) \mathrm{d} z$
    </br>
    1. $\displaystyle \int _C f(z) \mathrm{d} z = \int _{C_1} f(z) \mathrm{d} z \pm \int _{C_2} f(z) \mathrm{d} z \quad (C = C_1 + C_2)$
    </br>
    1. $\displaystyle \Big| \int _C f(z) \mathrm{d} z \Big| \leqslant \int _C \big|f(z)\big| \mathrm{d} s \quad (\mathrm{d}s = \sqrt {\mathrm{d} x^2 + \mathrm{d} y^2})$
    </br>
* **高阶导数**
    * **定理**:
        1. 设函数 $f(z)$ 在简单闭曲线 $C$ 所围成的区域 $D$ 内解析, 而且在 $\bar D = D \cup C$ 上解析, 则函数 $f(z)$ 的各阶导数均在 $D$ 内解析, 对 $D$ 内任一点 $z$, 有 $\displaystyle f^{(n)}(z) = \dfrac{n!}{\mathrm{i} 2\pi} \oint _C \dfrac{f(\zeta)}{(\zeta-z)^{n+1}} \mathrm{d} \zeta, \quad (n = 1,2,\cdots)$
        1. **柯西不等式**: 设函数 $f(z)$ 在 $|z-z_0| < R$ 内解析, 由当 $|z-z_0| < R$ 时, $|f(z)| \leqslant M$, 则有 $\big|f^{(n)}(z_0) \big| \leqslant \dfrac{n!M}{R^n} ,\quad (n=1,2,\cdots)$
        1. **刘维尔（Liouville）定理**: 设函数 $f(z)$ 在全平面上解析且有界, 则函数 $f(z)$ 为常数

## 11.5. 解析函数的级数表示

* **数列和级数**:
    1. 复数列 $\{z_n\}$ 收敛与 $z_0$ $\iff$ $\begin{cases} \displaystyle \lim _{n \to \infin} x_n = x_0 \\ \displaystyle \lim _{n \to \infin} y_n = y_0 \end{cases}$
    1. 复数项级数 $z$ 收敛
        * $\iff$ 实数项级数 $\displaystyle \sum _{n=1}^\infin x_n$ 和 $\displaystyle \sum _{n=1}^\infin y_n$ 都收敛
        * $\iff$ $\displaystyle \lim _{n \to \infin} z_n = 0 $
    1. 如果级数 $\displaystyle \sum _{n=1}^\infin |z_n|$ 收敛, 则级数 $\displaystyle \sum _{n=1}^\infin z_n$ 也收敛, 且不等式 $\displaystyle \Big|\sum _{n=1}^\infin z_n \Big| \leqslant \sum _{n=1}^\infin |z_n|$
        此时称 $\displaystyle \sum _{n=1}^\infin z_n$ 为绝对收敛, 非绝对收敛的收敛级数称为条件收敛


* **幂级数**: $\displaystyle \sum _{n=0}^\infin C_n (z-z_0)^n = C_0 + C_1(z-z_0) + C_2(z-z_0)^2 + \cdots + C_n(z-z_0)^n + \cdots$
    * **阿贝尔定理**:
        1. 如果幂级数 $\displaystyle \sum _{n=1}^\infin C_n z^n$ 在 $z = z_0 (z_0 \ne 0)$ 收敛, 则对满足 $|z| < |z_0|$ 的一切 $z$, 幂级数 $\displaystyle \sum _{n=1}^\infin C_n z^n$ 必绝对收敛
        1. 如果幂级数 $\displaystyle \sum _{n=1}^\infin C_n z^n$ 在 $z = z_1 (z_1 \ne 0)$ 发散, 则对满足 $|z| > |z_1|$ 的一切 $z$, 幂级数必发散
    * **收敛圆与收敛半径**: 若存在实数 $R > 0$,
        - 当 $|z| < R$ 时, 幂级数 $\displaystyle \sum _{n=0}^\infin C_n z^n$ 绝对收敛
        - 当 $|z| > R$ 时, 幂级数 $\displaystyle \sum _{n=0}^\infin C_n z^n$ 发散

        则称以 $R$ 为半径的圆周为幂级数的**收敛圆**, $R$ 称为**收敛半径**

    * **收敛半径的求法**:
        1. **比值法**: 设幂级数 $\displaystyle \sum _{n=0}^\infin C_n z^n$ 系数有: $\displaystyle \lim _{n \to \infin} \dfrac{|C _{n+1}|}{|C_n|} = \lambda \ne 0$, 则幂级数的收敛半径为 $R = \dfrac{1}{\lambda}$
        1. **根值法**: 设幂级数 $\displaystyle \sum _{n=0}^\infin C_n z^n$ 系数有: $\displaystyle \lim _{n \to \infin} \sqrt[n]{|C_n|} = \lambda \ne 0$, 则幂级数的收敛半径为 $R = \dfrac{1}{\lambda}$

    * **性质**:
        1. 幂级数的代数运算: 设幂级数 $\displaystyle f(z) = \sum _{n=0}^{\infin} a_n z^n, g(z) = \sum _{n=0}^{\infin} b_n z^n$ 的收敛半径分别为 $R_1, R_2$, 令 $R = \min(R_1, R_2)$, 则当 $|z| < R$ 时, 有: $$\begin{aligned}
                f(z) \pm g(z) &= \sum _{n=0}^{\infin} (a_n \pm b_n) z^n \\
                f(z) g(z) &= \Big(\sum _{n=0}^{\infin} a_n x^n \Big) \Big(\sum _{n=0}^{\infin} b_n x^n \Big) \\
                          &= a_0 b_0 + (a_0 b_1 + a_1 b_0) x + \cdots + (a_0 b_n + a_1 b_{n-1} + \cdots + a_n b_n) x^n + \cdots
                \end{aligned}$$
        1. 符合运算: 如果当 $|z| < r$ 时, $\displaystyle f(z)= \sum _{n=0}^{\infin} a_n z^n$, 又设在 $|z| < R$ 内函数 $g(z)$ 解析, 且满足 $|g(z)| < r$, 则当 $|z| < R$ 时, 有 $\displaystyle f\Big[g(z)\Big] = \sum _{n=0}^{\infin} a_n \Big[g(z) \Big]^n$
        1. 设幂级数 $\displaystyle \sum _{n=0}^\infin C_n (z-a)^n = f(z)$ 的收敛半径为 $R$, 则
            * $f(z)$ 在收敛圆 $|z - a| < R$ 内解析, 且可逐项求导 $\displaystyle f'(z) = \sum _{n=0}^\infin n C_n (z-a)^{n-1}$
            * $f(z)$ 在收敛圆 $|z - a| < R$ 内是可积函数, 且可逐项积分 $\displaystyle \int _C f(z) \mathrm{d} z = \sum _{n=0}^\infin \int _C C_n (z-a)^n \mathrm{d} z, \quad C \in |z-a| < R$ 或 $\displaystyle \int _a^z f(\zeta) \mathrm{d} \zeta = \sum _{n=0}^\infin \dfrac{C_n}{n+1} (z-a)^{n+1}$
* **泰勒级数**:
    * **定理**: 设函数 $f(z)$ 在区域 $D$ 内解析, $z_0$ 为 $D$ 内一点, $R$ 为 $z_0$ 到 $D$ 的边界上各点的最短距离, 则当 $|z - z_0| < R$ 时, 函数 $f(z)$ 可展为幂级数: $$\displaystyle f(z) = \sum _{n=0}^{\infin} C_n (z-z_0)^n, \quad 其中: C_n = \dfrac{1}{n!} f^{(n)} (z_0), n = 0,1,2\cdots$$
    * 常用的泰勒级数:
        1. $\displaystyle \dfrac{1}{1-z} = \sum _{n=0}^{\infin} z^n, \quad |z| < 1$
        </br>
        1. $\displaystyle \mathrm{e} ^z = \sum _{n=0}^{\infin} \dfrac{z^n}{n!}, \quad |z| < +\infin$
        </br>
        1. $\displaystyle \sin z = \sum _{n=0}^{\infin} (-1)^n \dfrac{z^{2n+1}}{(2n+1)!}, \quad |z| < +\infin$
        </br>
        1. $\displaystyle \cos z = \sum _{n=0}^{\infin} (-1)^n \dfrac{z^{2n}}{(2n)!}, \quad |z| < +\infin$
        </br>
* **洛朗级数**:
    * **洛朗定理**: 设函数 $f(z)$ 在圆环域 $R_1 < |z-z_0| < R_2$ 内处处解析, 则函数 $f(z)$ 一定能在环形域 $R_1 < |z-z_0| < R_2$ 中展开为 $$\displaystyle f(z) = \sum _{n=-\infin}^{\infin} C_n (z-z_0)^n, \quad 其中: C_n = \dfrac{1}{\mathrm{i} 2\pi} \oint _C \dfrac{f(\zeta)}{(\zeta - z_0)^{n+1}} \mathrm{d} \zeta, n = 0,\pm 1, \pm 2\cdots$$ 这里 $C$ 为圆环域内绕 $z_0$ 的任何一条正向简单闭曲线

## 11.6. 留数及其应用

### 11.6.1. 孤立奇点
* **定义**:
    * **孤立奇点**:
        * 若函数 $f(z)$ 在 $z_0$ 处不解析, 但在 $z_0$ 的某一去心邻域 $\mathring U(z_0, \delta)$ 即 $0 <|z - z_0| < \delta$ 内处处解析, 则称 $z_0$ 为函数 $f(z)$ 的**孤立奇点**
        * 设函数 $f(z)$ 在无穷远点的邻域 $R<|z|<+\infin$ 内解析, 则称无穷远点为函数 $f(z)$ 的**孤立奇点**
    * 函数 $f(z)$ 在孤立奇点 $z_0$ 的邻域 $\mathring U(z_0, \delta)$ 内展为洛朗级数为 $$\begin{aligned}
        f(z) = & \sum _{n=0}^{\infin} C_n (z-z_0)^n + & \sum _{n=1}^{\infin} C _{-n} (z-z_0)^{-n} \\
        & 解析部分 & 主要部分 \end{aligned}$$
        1. 主部消失, 即只有 $\displaystyle \sum _{n=0}^{\infin} C_n (z-z_0)^n$, 则称 $z_0$ 为函数 $f(z)$ 的**可去奇点**
        1. 主部仅含有限项 ($m$ 项), 则称 $z_0$ 为函数 $f(z)$ 的 **$m$ 阶极点**
            * 一阶极点也称为**简单极点**
        1. 主部含有无限多项, 则称 $z_0$ 为函数 $f(z)$ 的**本性奇点**
* **可去奇点**
    * $z_0$ 为函数 $f(z)$ 的可去奇点
        $\iff$ 函数 $f(z)$ 在 $z_0$ 点的洛朗级数展开式中不含 $z - z_0$ 的负幂项, 即 $f(z) = C_0 + C_1 (z - z_0) + \cdots + C_n (z - z_0 )^n + \cdots$
        $\iff$ $\displaystyle \lim _{z \to z0} f(z) = C_0, (C_0 为一常复数)$
        $\iff$ 函数 $f(z)$ 在 $z_0$ 某去心邻域内有界
    * $\infin$ 为函数 $f(z)$ 的可去奇点 $\iff$ $\displaystyle \lim _{z \to \infin} f(z) = C_0, (C_0 为一常复数)$
    * 若 $z_0$ 为函数 $f(z)$ 的可去奇点, 则 $f(z)$ 在 $z_0$ 的去心邻域内的洛朗级数就是一个不含负幂项的级数为 $f(z) = C_0 + C_1 (z-z_0) + C_2(z-z_0)^2 + \cdots + C_n (z-z_0)^n + \cdots, \quad 0 < |z-z_0| < \delta$ 显然这个幂级数的和函数 $F(z)$ 在 $|z - z0| < \delta$ 内处处解析, 令 $$\displaystyle f(z_0) = C_0 = \lim _{z \to z0} F (z_0) = \lim _{z \to z0} f(z_0)$$
* **极点**
    * $z_0$ 为函数 $f(z)$ 的极点 $\iff$ $\displaystyle \lim _{z \to z0} f(z) = \infin$
    * $z_0$ 为函数 $f(z)$ 的 $m$ 阶极点 $\iff$ $\displaystyle \lim _{z \to z0} (z-z_0)^m f(z) = C_{-m}, \quad (m 是正整数, C_{-m} 是一个不等于 0 的复常数)$
    * $\infin$ 为函数 $f(z)$ 的极点 $\iff$ $\displaystyle \lim _{z \to \infin} f(z) = \infin$
* **本性奇点**
    * $z_0$ 是 $f(z)$ 的本性奇点 $\iff$ 不存在有限或无穷的极限
    * $\infin$ 为函数 $f(z)$ 的本性奇点 $\iff$ 不存在有限或无穷的极限
* **零点**
    * 若 $f(z)=(z-z_0)^m \varphi(z)$, $\varphi (z)$ 在 $z_0$ 处解析, 且 $\varphi (z_0) \ne 0$, $m$ 为某一正整数, 那么称 $z_0$ 为 $f(z)$ 的 **$m$ 阶零点**
    * 若 $f(z)$ 在 $z_0$ 解析, 那么 $z_0$ 为 $f(z)$ 的 $m$ 阶零点 $\iff$ $\begin{cases} f^{(n)}(z_0) = 0, \ (n=0,1,\cdots, m-1) \\ f^{(m)}(z_0) \ne 0 \end{cases}$
    * 函数的零点与极点有下面的关系: 如果 $z_0$ 是 $f(z)$ 的 $m$ 阶极点, 那么 $z_0$ 就是 $\dfrac{1}{f(z)}$ $m$ 阶零点, 反之亦然

### 11.6.2. 留数
* **定义**:
    * 设 $z_0$ 是解析函数 $f(z)$ 的孤立奇点, 我们把 $f(z)$ 在 $z_0$ 处的洛朗展开式中负一次幂项的系数 $C_l$ 称为 $f(z)$ 在 $z_0$ 处的**留数**. 记作 $\mathrm{Res} \Big[f(z), z_0 \Big]$, 即 $$\mathrm{Res} \Big[f(z), z_0 \Big] = C_{-1}$$ 显然, 留数 $C_{-1}$ 就是积分 $$\dfrac{1}{\mathrm{i} 2\pi} \oint _C f(z) \mathrm{d}z$$ 的值, 其中 $C$ 为解析函数 $f(z)$ 的 $z_0$ 的去心邻域内绕 $z_0$ 的闭曲线
    * 设 $\infin$ 为 $f(z)$ 的一个孤立奇点, 即 $f(z)$ 在圆环域 $R < |z|< +\infin$ 内解析, 则称 $$\displaystyle \dfrac{1}{\mathrm{i} 2\pi} \oint _{C^-} f(z) \mathrm{d} z \ (C: |z| = \rho > R)$$ 为 $f(z)$ 在点 $\infin$ 的**留数**, 记为 $\mathrm{Res} \Big[f(z), \infin \Big]$, 这里 $C^-$ 是指顺时针方向 (这个方向很自然地可以看作是绕无穷远点的正向), 如果 $f(z)$ 在 $R < |z| < +\infin$ 的洛朗展开式为 $\displaystyle f(z) = \sum _{n = -\infin}^{\infin} C_n z^n$, 则有 $\mathrm{Res} \Big[f(z), \infin \Big] = -C_{-1}$
* **定理**:
    1. **留数定理**: 设函数 $f(z)$ 在区域 $D$ 内除有限个孤立奇点 $z_l, z_2,⋯ ,z_n$ 外处处解析, $C$ 是 $D$ 内包围各奇点的一条正向简单闭曲线, 那么 $$\oint _C f(z) \mathrm{d} z = \mathrm{i} 2\pi \sum _{k=1}^{n}\mathrm{Res} \Big[f(z), z_k \Big]$$
    1. 如果 $f(z)$ 在扩充复平面上只有有限个孤立奇点 (包括无穷远点在内), 设为 $z_l, z_2,⋯ ,z_n, \infin$, 则 $f(z)$ 在各点的留数总和为零
    1. **若尔当(Jordan)引理**: 设函数 $g(z)$ 在闭区域 $\theta _1 \leqslant \arg z \leqslant \theta _2, R_0 \leqslant |z| \leqslant + \infin \ (R_0 \geqslant 0, 0 \leqslant \theta _1 \leqslant \theta _2 \leqslant \pi)$ 上连续, 并设 $C_R$ 是该闭区域上的一段以原点为中心, $R \ (R > R_0)$ 为半径的圆弧, 若当 $z$ 在该闭区域上时, $\displaystyle \lim _{z \to \infin} g(z) = 0$, 则对任何 $a>0$, 有 $\displaystyle \lim _{R \to + \infin} \int _{C_R} g(z) \mathrm{e}^{\mathrm{i}az} \mathrm{d}z = 0$
* **留数计算**
    1. 如果 $z_0$ 为 $f(z)$ 的简单极点, 则 $$\displaystyle \mathrm{Res} \Big[f(z), z_0 \Big] = \lim _{z \to z_0} (z-z_0) f(z)$$
    1. 设 $f(z) = \dfrac{P(z)}{Q(z)}$, 其中 $P(z), Q(z)$ 在 $z_0$ 处解析, 如果 $P(z_0) \ne 0$, $z_0$ 为 $Q(z)$ 的一阶零点, 则 $z_0$ 为 $f(z)$ 的一阶极点, 且 $$\displaystyle \mathrm{Res} \Big[f(z), z_0 \Big] = \dfrac{P(z_0)}{Q'(z_0)}$$
    1. 如果 $z_0$ 为 $f(z)$ 的 $m$ 阶极点, 则 $$\displaystyle \mathrm{Res} \Big[f(z), z_0 \Big] = \dfrac{1}{(m-1)!} \lim _{z \to z_0} \dfrac{\mathrm{d}^{m-1}}{\mathrm{d}z^{m-1}}\Big[(z-z_0)^m f(z) \Big]$$
    1. $\mathrm{Res} \Big[f(z), \infin \Big] = - \mathrm{Res} \Big[f(\dfrac{1}{z})\cdot \dfrac{1}{z^2}, 0 \Big]$
* **留数在定积分计算中的应用**
    1. 形如 $\displaystyle \int _0^{2\pi} R(\cos \theta, \sin \theta) \mathrm{d} \theta$ 的积分
        * 令 $z = \mathrm{e}^{\mathrm{i}\theta}$, 则 $\mathrm{d} z = \mathrm{i} \mathrm{e}^{\mathrm{i} \theta} \mathrm{d} \theta, \quad \cos \theta = \dfrac{\mathrm{e}^{\mathrm{i} \theta} + \mathrm{e}^{-\mathrm{i}\theta}}{2} = \dfrac{z^2+1}{2z}, \quad \sin \theta = \dfrac{\mathrm{e}^{\mathrm{i} \theta}-\mathrm{e}^{-\mathrm{i}\theta}}{2 \mathrm{i}} = \dfrac{z^2-1}{\mathrm{i}2z}$
        * 令 $f(z) = R(\dfrac{z^2+1}{2z}, \dfrac{z^2-1}{\mathrm{i} 2 z})$
        </br>
        $\implies$ $\displaystyle \int _0^{2\pi} R(\cos \theta, \sin \theta) \mathrm{d} \theta = \oint _{|z| = 1} R(\dfrac{z^2+1}{2z}, \dfrac{z^2-1}{\mathrm{i} 2 z}) \dfrac{\mathrm{d} z}{\mathrm{i} z} = \oint _{|z| = 1} f(z) \mathrm{d} z = \mathrm{i} 2 \pi \sum _{k=1}^n \mathrm{Res}\Big[f(z), z_k \Big]$
    1. 形如 $\displaystyle \int _{-\infin}^{+\infin} R(x) \mathrm{d} x$ 的积分
        * 令 $R(z) = \dfrac{P(z)}{Q(z)} = \dfrac{z^n + a_1 z^{n-1} + \cdots + a_n}{z^m + b_1 z^{m-1} + \cdots + b_m} \ (m-n \geqslant 2)$
            * $Q(z)$ 比 $P(z)$ 至少高两次
            * $Q(z)$ 在实轴上无零点
            * $R(z)$ 在上半平面 $\mathrm{Im} z > 0$ 内的极点为 $z_k \ (k = 1,2,\cdots,n)$

        $\implies$ $\displaystyle \int _{-\infin}^{+\infin} R(x) \mathrm{d} x = \mathrm{i} 2 \pi \sum _{k=1}^n \mathrm{Res}\Big[R(z), z_k \Big]$
    1. 形如 $\displaystyle \int _{-\infin}^{+\infin} R(x) \mathrm{e}^{\mathrm{i}ax} \mathrm{d} x \ (a >0)$ 的积分
        * 若 $R(x)$ 是真分式 (分母的次数至少比分子的次数高一次),  且在实轴上无奇点, 令 $f(z) = R(z) \mathrm{e}^{\mathrm{i}ax}$, $f(z)$ 在上半平面 $\mathrm{Im} z > 0$ 内的极点为 $z_k \ (k = 1,2,\cdots,n)$

        $\implies$ $\displaystyle \int _{-\infin}^{+\infin} R(x) \mathrm{e}^{\mathrm{i}ax} \mathrm{d} x = \int _{-\infin}^{+\infin} \dfrac{P(x)}{Q(x)}\mathrm{e}^{\mathrm{i}ax} \mathrm{d}x = \mathrm{i} 2 \pi \sum _{k=1}^n \mathrm{Res}\Big[f(z), z_k \Big]$
        * $\begin{cases}\displaystyle \int _{-\infin}^{+\infin} R(x) \cos ax \mathrm{d} x = \mathrm{Re} \Big\{\mathrm{i} 2 \pi \sum _{k=1}^n \mathrm{Res}\Big[f(z), z_k \Big]\Big\} \\ \displaystyle \int _{-\infin}^{+\infin} R(x) \sin ax \mathrm{d} x = \mathrm{Im} \Big\{\mathrm{i} 2 \pi \sum _{k=1}^n \mathrm{Res}\Big[f(z), z_k \Big]\Big\}\end{cases}$
    1. 对于函数在实轴上有奇点的积分
        * 适当选取路径来积分, 使积分路径绕开孤立奇点, 得 $$\displaystyle \int _{-\infin}^{+\infin} f(x) \mathrm{d}x = \mathrm{i} 2 \pi \sum _{k=1}^{n} \mathrm{Res} \Big[f(z), z_k \Big] + \dfrac{1}{2} \sum _{k=1}^{n} \mathrm{Res} \Big[f(z), x_k \Big]$$ 其中 $z_k$ 是上半平面的奇点, $x_k$ 是实轴上的奇点

# 12. 积分变换

## 12.1. 单位冲击函数和卷积
* **单位冲击函数 / 狄拉克函数 / $\delta$ 函数**: $$\delta (t) := \begin{cases} 当 t \ne 0 时, \delta(t) = 0 \\ \\ \displaystyle \int _{-\infin}^{+\infin} \delta (t) \mathrm{d}t = 1 \end{cases}$$
    * **性质**:
        1. **筛选性质**: 设 $f(t)$ 是 $\R$ 上的有界函数, 且在 $t=0$ 处连续 $\implies$ $\displaystyle \int _{-\infin}^{+\infin} \delta(t) f(t) \mathrm{d}t = f(0)$
            * 若 $f(t)$ 在 $t=t_0$ 处连续, 则 $\displaystyle \int _{-\infin}^{+\infin} \delta(t-t_0) f(t) \mathrm{d}t = f(t_0)$
        1. 偶函数: $\delta(t) = \delta(-t)$
        1. 设 $u(t)$ 为**单位阶跃函数**, 即 $u(t) = \begin{cases} 1, & t>0 \\ 0, & t<0 \end{cases}$ $\implies$ $\displaystyle \int _{-\infin}^t \delta (t) \mathrm{d}t = u(t), \ u'(t) = \delta (t)$
* **卷积**: 设 $f_1(t)$ 与 $f_2(t)$ 在 $(-\infin, +\infin)$ 内有定义, 若反常积分 $\displaystyle \int _{-\infin}^{+\infin} f_1(\tau)f_2(t-\tau)\mathrm{d}\tau$ 对任何实数 $t$ 收敛, 则它定义了一个自变量为 $t$ 的函数, 称为 $f_1(t)$ 和 $f_2(t)$ 的**卷积**, 记作 $f_1(t) * f_2(t)$, 即 $$\displaystyle f_1(t) * f_2(t) = \int _{-\infin}^{+\infin} f_1(\tau)f_2(t-\tau)\mathrm{d}\tau$$
    * 满足交换律, 结合律, 分配律

## 12.2. 傅里叶变换
* **傅里叶积分定理**: 如果 $f(t)$ 在 $(-\infin, +\infin)$ 上任一有限区间满足狄利克雷条件, 即
    1. 连续或只有有限个第一类间断点
    1. 只有有限个极值点

    且在 $(-\infin, +\infin)$ 上绝对可积 (即 $\displaystyle \int _{-\infin}^{+\infin} |f(t)| \mathrm{d}t < +\infin$), 则 $$\displaystyle f(t) = \dfrac{1}{2\pi} \int _{-\infin}^{+\infin} \Big[\int _{-\infin}^{+\infin} f(\tau)\mathrm{e}^{-j\omega \tau} \mathrm{d} \tau \Big] \mathrm{e}^{j\omega\tau}\mathrm{d} \omega$$ 成立, 在 $f(t)$ 的间断点处, $$\displaystyle \dfrac{1}{2} \big[f(t^+) + f(t^-) \big] = \dfrac{1}{2\pi} \int _{-\infin}^{+\infin} \Big[\int _{-\infin}^{+\infin} f(\tau)\mathrm{e}^{-j\omega \tau} \mathrm{d} \tau \Big] \mathrm{e}^{j\omega\tau}\mathrm{d} \omega$$
* **傅里叶变换**: $$F(\omega) = \mathscr{F} [f(t)] = \int _{-\infin}^{+\infin} f(t) \mathrm{e}^{-j\omega t} \mathrm{d} t$$
* **傅里叶反变换**: $$f(t) = \mathscr{F}^{-1}[F(\omega)] = \dfrac{1}{2\pi}\int _{-\infin}^{+\infin} F(\omega) \mathrm{e}^{j\omega t} \mathrm{d} \omega$$
* **傅里叶变换的性质**
    1. **时频对偶性**: 设 $f(t) \xrightleftharpoons{FT} F(\omega)$, 则 $$F(jt) \xrightleftharpoons{FT} 2 \pi f(-\omega)$$
        * $f(t) = \begin{cases} 1, &|t| \leqslant \delta \\0, &|t|>\delta \end{cases} \ (\delta > 0)
            \quad \xrightleftharpoons{FT} \quad
            F(\omega) = 2 \delta \dfrac{\sin \delta \omega}{\delta \omega} = 2 \delta \mathrm{Sa}(\delta \omega)$
        * $\dfrac{a}{\pi} \mathrm{Sa}(at)
            \quad \xrightleftharpoons{FT} \quad
            F(\omega) = \begin{cases} 1, & |\omega|< a \\ 0, & |\omega| \geqslant a \end{cases}$
        </br>
        > $\mathrm{Sa}(t) = \dfrac{\sin t}{t}$, 称为抽样函数, $\displaystyle \int _0^{\infin} \dfrac{\sin t}{t} \mathrm{d} t = \dfrac{\pi}{2}$
        * $\delta (t) \quad \xrightleftharpoons{FT} \quad 1$
        * $1 \quad \xrightleftharpoons{FT} \quad 2\pi \delta (\omega)$
        </br>
    1. **线性性质**: 设 $F(\omega) = \mathscr{F} \big[f(t) \big], G(\omega) = \mathscr{F} \big[g(t) \big] , \alpha, \beta$ 为常数, 则 $$\alpha f(t) + \beta g(t) \xrightleftharpoons{FT} \alpha F(\omega) + \beta G(\omega)$$
    1. **位移性质**: 设 $F(\omega) = \mathscr{F} \big[f(t) \big], t_0, \omega _0$ 为常数, 则 $$\begin{aligned} \mathscr{F} \big[f(t-t_0) \big] &= \mathrm{e}^{-j\omega t_0} F(\omega) \\ \mathscr{F}^{-1} \big[F(\omega - \omega _0) \Big] &= \mathrm{e}^{j\omega _0 t} f(t)\end{aligned} $$
        > 时域位移: 相位发生变化, 频域的相位谱变化
        > 频域位移: 频谱搬移, 时域 × 正弦波 ($\cos \omega _0 t = \dfrac{\mathrm{e}^{j \omega _0 t} + \mathrm{e}^{-j \omega _0 t}}{2}$)
    1. **相似性质**: 设 $F(\omega) = \mathscr{F} \big[f(t) \big], \alpha$ 为非零常数, 则 $$\mathscr{F} \big[f(\alpha t) \big] = \dfrac{1}{|\alpha|}F(\dfrac{\omega}{\alpha})$$
        > 时域压缩: 频域扩张
    1. **微分性质**: 设 $F(\omega) = \mathscr{F} \big[f(t) \big]$, 若 $\displaystyle \lim _{|t| \to +\infin} f(t) = 0$, 则 $$\mathscr{F} \big[f'(t) \big] = j\omega F(\omega)$$ 若 $\displaystyle \lim _{|t| \to +\infin} f^{(k)}(t) = 0 \ (k = 1,2,\cdots, n-1)$, 则 $$\mathscr{F} \big[f^{(n)}(t) \big] = (j\omega)^n F(\omega)$$
        频域微分性质: $$F'(\omega) = \mathscr{F} \big[-j t f(t) \big]$$ 一般地, 有 $$F^{(n)}(\omega) = (-j)^n \mathscr{F} \big[t^n f(t) \big]$$
    1. **积分性质**: 设 $F(\omega) = \mathscr{F} \big[f(t) \big]$, $\displaystyle g(t) = \int _{-\infin}^{t} f(t) \mathrm{d} t$, 若 $\displaystyle \lim _{t \to +\infin} g(t) = 0$, 则 $$\mathscr{F} \big[g(t) \big] = \dfrac{1}{j\omega} F(\omega)$$
    1. **帕塞瓦尔(Parseval)等式**: 设 $F(\omega) = \mathscr{F} \big[f(t) \big]$, 则 $$\displaystyle \int _{-\infin}^{+\infin}f^2 (t) \mathrm{d} t = \dfrac{1}{2\pi} \int _{-\infin}^{+\infin} | F(\omega) |^2 \mathrm{d} \omega$$
        > 时域功率积分等于频域功率积分
    1. **卷积**: 设 $F_1(\omega) = \mathscr{F} \big[f_1(t) \big], F_2(\omega) = \mathscr{F} \big[f_2(t) \big]$, 则 $$\begin{aligned} \mathscr{F} \big[f_1(t) * f_2(t) \big] &= F_1(\omega) \cdot F_2(\omega) \\ \mathscr{F} \big[f_1(t) \cdot f_2(t) \big] &= \dfrac{1}{2\pi} F_1(\omega) * F_2(\omega)\end{aligned}$$
        > 时域传递函数是卷积; 频域传递函数是乘积
* 常用的傅里叶变换
    $$\begin{aligned}
        \cos \omega _0 t
                & \quad \xrightleftharpoons{FT} \quad
            \pi[\delta(\omega + \omega _0) + \delta (\omega - \omega _0)]
            \\
        \sin \omega _0 t
                & \quad \xrightleftharpoons{FT} \quad
            j\pi[\delta(\omega + \omega _0) - \delta (\omega - \omega _0)]
            \\
        \dfrac{\sin \omega _0 t}{\pi t}
                & \quad \xrightleftharpoons{FT} \quad
            \begin{cases} 1, & |\omega| \leqslant \omega _0 \\ 0, &|\omega| > \omega _0\end{cases}
            \\
        \dfrac{1}{t} \sin at \ (a > 0)
                & \quad \xrightleftharpoons{FT} \quad
            \begin{cases} \pi & |\omega| \leqslant a \\ 0, & |\omega| > a \end{cases}
            \\
        u(t)
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{1}{j\omega} + \pi \delta(\omega)
            \\
        u(t-c)
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{1}{j\omega} \mathrm{e}^{-j\omega c} + \pi \delta(\omega)
            \\
        u(t) \cdot t
                & \quad \xrightleftharpoons{FT} \quad
            -\dfrac{1}{\omega ^2} + j \pi \delta'(\omega)
            \\
        u(t) \cdot t^n
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{n!}{(j\omega) ^{n+1}} + j^n \pi \delta^{(n)}(\omega)
            \\
        u(t) \sin at
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{a}{a^2 -\omega ^2} + \dfrac{\pi}{2j}[\delta(\omega - \omega _0) - \delta (\omega + \omega _0)]
            \\
        u(t) \cos at
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{j\omega}{a^2 -\omega ^2} + \dfrac{\pi}{2}[\delta(\omega - \omega _0) + \delta (\omega + \omega _0)]
            \\
        u(t) \mathrm{e}^{-\beta t} \ (\beta > 0)
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{1}{\beta + j\omega}
            \\
        u(t)\mathrm{e}^{jat}
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{1}{j(\omega-a)} + \pi \delta(\omega-a)
            \\
        u(t-c)\mathrm{e}^{jat}
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{1}{j(\omega-a)} \mathrm{e}^{-j(\omega-a)c} + \pi \delta(\omega-a)
            \\
        u(t)\mathrm{e}^{jat} t^n
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{n!}{[j(\omega-a)]^{n+1}} + j^n \pi \delta^{(n)}(\omega-a)
            \\
        \mathrm{e}^{a|t|} \ (\mathrm{Re}(a) <0)
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{-2a}{\omega^2 + a^2}
            \\
        \delta(t)
                & \quad \xrightleftharpoons{FT} \quad
            1
            \\
        \delta(t-c)
                & \quad \xrightleftharpoons{FT} \quad
            \mathrm{e}^{-j\omega c}
            \\
        \delta'(t)
                & \quad \xrightleftharpoons{FT} \quad
            j\omega
            \\
        \delta^{(n)}(t)
                & \quad \xrightleftharpoons{FT} \quad
            (j\omega)^n
            \\
        \delta^{(n)}(t-c)
                & \quad \xrightleftharpoons{FT} \quad
            (j\omega)^n \mathrm{e}^{-j\omega c}
            \\
        1
                & \quad \xrightleftharpoons{FT} \quad
            2\pi \delta (\omega)
            \\
        t
                & \quad \xrightleftharpoons{FT} \quad
            2\pi j \delta' (\omega)
            \\
        t^n
                & \quad \xrightleftharpoons{FT} \quad
            2\pi j^n \delta^{(n)} (\omega)
            \\
        \mathrm{e}^{jat}
                & \quad \xrightleftharpoons{FT} \quad
            2\pi \delta(\omega-a)
            \\
        t^n \mathrm{e}^{j\omega t}
                & \quad \xrightleftharpoons{FT} \quad
            2\pi j^n \delta^{(n)} (\omega-a)
            \\
        \mathrm{sgn}(t)
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{2}{j\omega}
            \\
        |t|
                & \quad \xrightleftharpoons{FT} \quad
            -\dfrac{2}{\omega^2}
            \\
        \dfrac{1}{|t|}
                & \quad \xrightleftharpoons{FT} \quad
            \dfrac{\sqrt{2\pi}}{|\omega|}
            \\
    \end{aligned} $$

## 12.3. 拉普拉斯变换
> 傅里叶变换的局限性:
>   1. 函数要满足狄氏条件
>   1. 一般说来还要在 $(-\infin,+\infin)$上绝对可积
>   1. 必须在整个实轴上有定义 (实际工程应用中, $t<0$ 时无意义)
>
> 因此, 傅里叶变换不能适用于有阻尼的系统, 需要拉普拉斯变换

* **拉普拉斯变换**: 设函数 $f(t)$ 是定义在 $[0, +\infin)$ 上的实值函数, 如果对于复参数 $s=\beta + j \omega$, 积分 $$F(s) = \int _0^{+\infin} f(t) \mathrm{e}^{-st} \mathrm{d}t$$ 在复平面 $s$ 的某一域内收敛, 则称 $F(s)$ 为 $f(t)$ 的拉普拉斯变换, 记为 $F(s) = \mathscr{L} \big[f(t) \big]$
    > 拉普拉斯变换就是 $f(t) u(t) \mathrm{e}^{-\beta t}$ 的傅里叶变换
* **拉普拉斯反变换**: $$f(t) = \mathscr{L}^{-1} \big[F(s) \big] = \dfrac{1}{2\pi j} \int _{\beta - j\infin}^{\beta + j\infin} F(s) \mathrm{e}^{st} \mathrm{d} s \ (t > 0)$$
    * **利用留数计算反演积分**: 设 $F(s)$ 除在半平面 $\mathrm{Re} s \leqslant c$ 内有限个孤立奇点 $s_l,s_2, \cdots, s_n$ 外是解析的, 且当 $s \to \infin$ 时, $F(s) \to 0$, 则有 $$f(t) = \dfrac{1}{2\pi j} \int _{\beta - j\infin}^{\beta + j\infin} F(s) \mathrm{e}^{st} \mathrm{d} s  = \sum _{k=1}^{n} \mathrm{Res} \big[F(s) \mathrm{e}^{st}, s_k \big]  \ (t > 0)$$
* **拉普拉斯变换存在定理**: 设函数 $f(t)$ 满足:
    1. 在 $t \geqslant 0$ 的任何有限区间上分段连续
    1. 当 $t \to +\infin$ 时, $f(t)$ 具有有限的增长性, 即存在常数 $M>0$ 及 $c \geqslant 0$, 使得 $|f(t)| \leqslant M \mathrm{e}^{ct}, \ (0 \leqslant t < +\infin)$, 其中 $c$ 称为 $f(t)$ 的增长指数

    则像函数 $F(s)$ 在半平面 $\mathrm{Re} \ s > c$ 上一定存在, 且是解析的
* **拉普拉斯变换的性质**:
    1. **线性性质**: 设 $F(s) = \mathscr{L} \big[f(t) \big], G(s) = \mathscr{L} \big[g(t) \big] , \alpha, \beta$ 为常数, 则 $$\alpha f(t) + \beta g(t) \xrightleftharpoons{LT} \alpha F(s) + \beta G(s)$$
    1. **相似性质**: 设 $F(s) = \mathscr{L} \big[f(t) \big], \alpha$ 为非零常数, 则 $$\mathscr{L} \big[f(\alpha t) \big] = \dfrac{1}{\alpha}F(\dfrac{s}{\alpha})$$
        > 时域压缩: 频域扩张
    1. **延迟性质**: 设 $F(s) = \mathscr{L} \big[f(t) \big]$, 当 $t < 0$ 时, $f(t) = 0$, 则对任一非负实数 $\tau$, $$\mathscr{L} \big[f(t - \tau) \big] = \mathrm{e}^{-st} F(s)$$
    1. **位移性质**: 设 $F(s) = \mathscr{L} \big[f(t) \big]$, 则 $$\mathscr{L} \big[\mathrm{e}^{at} f(t) \big] = F(s-a), \ (a 为一复常数)$$
    1. **时域微分性质**: 设 $F(s) = \mathscr{L} \big[f(t) \big]$, 则 $$\mathscr{L} \big[f'(t) \big] = s F(s) - f(0)$$ 一般地, 有 $$\mathscr{L} \big[f^{(n)}(t) \big] = s^n F(s) - s^{n-1}f(0) - s^{n-2} f'(0) \cdots - f^{(n-1)}(0)$$ 其中, $f^{(k)}(0)$ 应理解为 $\displaystyle \lim _{t \to 0^+} f^{(k)}(t)$
    1. **频域微分性质**: 设 $F(s) = \mathscr{L} \big[f(t) \big]$, 则 $$F'(s) = -\mathscr{L} \big[tf(t) \big]$$ 一般地, 有 $$F^{(n)}(s) = (-1)^n \mathscr{L} \big[t^n f(t) \big]$$
    1. **时域积分性质**: 设 $F(s) = \mathscr{L} \big[f(t) \big]$, 则 $$\displaystyle \mathscr{L} \big[\int _0^t f(t) \mathrm{d} t \big] = \dfrac{1}{s} F(s)$$ 一般地, 有 $$\displaystyle \mathscr{L} \big[\int _0^t \int _0^t \cdots \int _0^t f(t) \mathrm{d} t^n \big] = \dfrac{1}{s^n} F(s)$$
    1. **频域积分性质**: 设 $F(s) = \mathscr{L} \big[f(t) \big]$, 则 $$\displaystyle \int _s^{\infin} F(s) \mathrm{d} s = \mathscr{L} \big[\dfrac{f(t)}{t} \big]$$ 一般地, 有 $$\displaystyle \int _s^{\infin} \int _s^{\infin} \cdots \int _s^{\infin} F(s) \mathrm{d} s^n = \mathscr{L} \big[\dfrac{f(r)}{t^n} \big]$$
    1. **卷积**: 设 $F_1(s) = \mathscr{L} \big[f_1(t) \big], F_2(s) = \mathscr{L} \big[f_2(t) \big]$, 则 $$\begin{aligned} \mathscr{L} \big[f_1(t) * f_2(t) \big] &= F_1(s) \cdot F_2(s) \end{aligned}$$
        > 时域传递函数是卷积; 频域传递函数是乘积

* **周期函数的拉普拉斯变换**: 设 $f(t)$ 是 $[0, \infin)$ 内以 $T$ 为周期的函数, 且 $f(t)$ 在一个周期内逐段光滑, 则 $$\mathscr {L} \big [f(t) \big] = \dfrac{1}{1-\mathrm{e}^{-sT}} \int _0^T f(t) \mathrm{e}^{-st} \mathrm{d} t$$
* 常用的拉普拉斯变换
    $$ \begin{aligned}
        1 / u(t) /\mathrm{sgn}(t)
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{1}{s}
        \\
        \mathrm{e}^{at}
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{1}{s-a}
        \\
        t / t u(t)
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{1}{s^2}
        \\
        t^m / t^m u(t) \ (m > -1)
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{\varGamma(m+1)}{s^{m+1}}
        \\
        t^m \mathrm{e}^{at} \ (m > -1)
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{\varGamma(m+1)}{(s-a)^{m+1}}
        \\
        \delta(t)
                & \quad \xrightleftharpoons{LT} \quad
            1
        \\
        \delta^{(n)}(t)
                & \quad \xrightleftharpoons{LT} \quad
            s^n
        \\
        \sin at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{a}{s^2+a^2}
        \\
        \cos at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{s}{s^2+a^2}
        \\
        \sinh at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{a}{s^2-a^2}
        \\
        \cosh at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{s}{s^2-a^2}
        \\
        t\sin at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{2as}{(s^2+a^2)^2}
        \\
        t\cos at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{s^2-a^2}{(s^2+a^2)^2}
        \\
        t\sinh at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{2as}{(s^2-a^2)^2}
        \\
        t\cosh at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{s^2+a^2}{(s^2-a^2)^2}
        \\
        t^m \sin at \ (m > -1)
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{\varGamma(m+1)}{2j(s^2+a^2)^{m+1}} \cdot [(s+ja)^{m+1} - (s-ja)^{m+1}]
        \\
        t^m \cos at \ (m > -1)
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{\varGamma(m+1)}{2(s^2+a^2)^{m+1}} \cdot [(s+ja)^{m+1} + (s-ja)^{m+1}]
        \\
        \mathrm{e}^{-bt} \sin at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{a}{(s+b)^2+a^2}
        \\
        \mathrm{e}^{-bt} \cos at
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{s+b}{(s+b)^2+a^2}
        \\
        \mathrm{e}^{-bt} \sin (at+c)
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{(s+b)\sin c + a \cos c}{(s+b)^2+a^2}
        \\
        \sin^2 t
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{1}{2} \Big(\dfrac{1}{s}-\dfrac{s}{s^2+4}\Big)
        \\
        \cos^2 t
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{1}{2} \Big(\dfrac{1}{s}+\dfrac{s}{s^2+4}\Big)
        \\
        \sin at \sin bt
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{2abs}{[s^2+(a+b)^2][s^2+(a-b)^2]}
        \\

                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{s}{(s+a)^2}
        \\
        t(1-\dfrac{a}{2} t) \mathrm{e}^{-at}
                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{s}{(s+a)^3}

                & \quad \xrightleftharpoons{LT} \quad
            \dfrac{1}{s(s+a)}
        \\
        \dfrac{1}{t}\sin at
                & \quad \xrightleftharpoons{LT} \quad
            \arctan \dfrac{a}{s}
        \\
    \end{aligned}$$

其中 $\displaystyle \varGamma(z) = \int _0^{+\infin} t^{z-1}\mathrm{e}^{-t}\mathrm{d}t$, 也叫欧拉第二积分，是阶乘函数在实数与复数上扩展的一类函数
